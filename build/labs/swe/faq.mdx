<Accordion title="전체 Open SWE 실행 비용은 얼마인가요?">
  실행당 비용은 작업의 복잡도, 저장소의 크기, 변경해야 하는 파일의 수에 따라 크게 달라집니다.

  대부분의 작업에서 Claude Sonnet 4를 사용할 때 `$0.50` -> `$3.00` 사이의 비용을 예상할 수 있습니다.
  Claude Opus 4/4.1에서 동일한 작업을 실행할 때는 `$1.50` -> `$9.00` 사이의 비용을 예상할 수 있습니다.

  비용에 민감하다면 항상 실행을 모니터링하는 것을 잊지 마세요. 제가 본 가장 비싼 Open SWE 실행은 약 50M Opus 4 토큰으로 `$25.00`의 비용이 발생했습니다.
</Accordion>

<Accordion title="Open SWE는 자동으로 토큰을 캐시하나요?">
  네. Anthropic 모델을 사용할 때 모든 입력 토큰은 Anthropic의 서버에 캐시됩니다.
</Accordion>

<Accordion title="요청을 제출할 때 Open SWE가 이슈를 생성하지 않도록 비활성화할 수 있나요?">
  네. 요청을 제출할 때 Open SWE가 이슈를 생성하지 않도록 비활성화하는 두 가지 방법이 있습니다:

  1. 요청을 제출할 때 메인 채팅 영역의 '눈' 아이콘을 토글합니다.
  2. 설정의 구성 탭에서 'Should Create Issue' 스위치를 토글합니다.

  기본적으로 `true`로 설정되어 있습니다. 구성 탭에서 이 설정을 수정하면 모든 실행이 해당 설정을 기본값으로 사용합니다. 메인 채팅 영역의 '눈' 아이콘을 토글하여 실행별로 이 설정을 재정의할 수 있습니다.

  ![Should Create Issue Toggle](/images/dont_create_issue_eye_screenshot.png)
  ![Global Should Create Issue Toggle](/images/dont_create_issue_global_toggle_screenshot.png)
</Accordion>

<Accordion title="실행이 중간에 실패했습니다. 어떻게 해야 하나요?">
  불편을 드려 죄송합니다! Open SWE는 변경 사항을 자동으로 draft pull request에 커밋합니다. 즉, 모든 진행 상황이 저장되며 마지막 체크포인트부터 실행을 재시작할 수 있습니다.

  실행을 재시작하려면 `Restart from last checkpoint` 버튼을 클릭하세요. 새 스레드가 생성되고 중단된 지점부터 재개됩니다.

  ![Restart Run Screenshot](/images/restart_run_screenshot.png)
</Accordion>

<Accordion title="프로덕션 환경에서 Open SWE를 사용할 수 있나요?">
  네! LangChain에서 한동안 내부적으로 Open SWE를 사용해왔으며 훌륭한 결과를 얻고 있습니다.

  프로덕션 환경에서 사용할 계획이라면 Open SWE를 직접 포크하여 배포하는 것을 권장합니다. 제품을 확인하는 용도로는 [데모 애플리케이션](https://swe.langchain.com)을 사용해도 괜찮습니다.
</Accordion>

<Accordion title="조직의 저장소에 Open SWE를 설치했는데 UI에 표시되지 않습니다. 왜 그런가요?">
  일부 GitHub 조직은 GitHub 앱을 설치하기 위해 관리자 승인이 필요합니다. 조직의 관리자에게 연락하여 설치 요청을 승인받으세요.  
</Accordion>

<Accordion title="Open SWE는 어떤 샌드박스 환경에서 실행되나요?">
  Open SWE의 샌드박스 환경은 [Daytona.io](https://daytona.io)로 구동됩니다.
</Accordion>

<Accordion title="Open SWE에 기여할 수 있나요?">
  네! Open SWE 개선을 도와줄 기여자를 항상 찾고 있습니다. [open issue](https://github.com/langchain-ai/open-swe/issues)를 선택하거나 새로운 기능이나 버그 수정이 포함된 pull request를 제출해 주세요.
</Accordion>
