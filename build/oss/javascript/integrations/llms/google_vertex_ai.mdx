[Google Vertex](https://cloud.google.com/vertex-ai)는 `gemini-1.5-pro`, `gemini-1.5-flash` 등 Google Cloud에서 사용 가능한 모든 foundation 모델을 제공하는 서비스입니다.

이 문서는 LangChain을 사용하여 VertexAI completion 모델(LLM)을 시작하는 데 도움을 드립니다. `VertexAI` 기능 및 구성 옵션에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_google_vertexai.VertexAI.html)를 참조하세요.

## Setup

LangChain.js는 Node.js 환경에서 실행하는지 웹 환경에서 실행하는지에 따라 두 가지 다른 인증 방법을 지원합니다.

VertexAI 모델에 액세스하려면 Google Cloud Platform(GCP) 계정을 생성하고, API 키를 받고, `@langchain/google-vertexai` integration package를 설치해야 합니다.

### Credentials

#### Node.js

관련 프로젝트에 대해 Vertex AI API가 활성화되어 있고 다음 방법 중 하나를 사용하여 Google Cloud에 인증했는지 확인해야 합니다:

- 해당 프로젝트에 권한이 있는 계정으로 로그인되어 있습니다(`gcloud auth application-default login` 사용).
- 프로젝트에 권한이 있는 service account를 사용하는 머신에서 실행 중입니다.
- 프로젝트에 권한이 있는 service account의 credentials를 다운로드하고 `GOOGLE_APPLICATION_CREDENTIALS` environment variable을 이 파일의 경로로 설정했습니다.
  **또는**
- 프로젝트의 API key로 `GOOGLE_API_KEY` environment variable을 설정했습니다.

#### Web

웹 환경(Edge function 등)에서 Vertex AI 모델을 호출하려면 `@langchain/google-vertexai-web` package를 설치해야 합니다.

그런 다음 service account credentials를 `GOOGLE_VERTEX_AI_WEB_CREDENTIALS` environment variable로 직접 추가해야 합니다:

```
GOOGLE_VERTEX_AI_WEB_CREDENTIALS={"type":"service_account","project_id":"YOUR_PROJECT-12345",...}
```

다음과 같이 코드에서 직접 credentials를 전달할 수도 있습니다:

```typescript
import { VertexAI } from "@langchain/google-vertexai";
// Or uncomment this line if you're using the web version:
// import { VertexAI } from "@langchain/google-vertexai-web";

const model = new VertexAI({
  authOptions: {
    credentials: {"type":"service_account","project_id":"YOUR_PROJECT-12345",...},
  },
});
```

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain VertexAI integration은 `@langchain/google-vertexai` package에 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/google-vertexai @langchain/core
```
```bash yarn
yarn add @langchain/google-vertexai @langchain/core
```
```bash pnpm
pnpm add @langchain/google-vertexai @langchain/core
```
</CodeGroup>

또는 웹 환경의 경우:

<CodeGroup>

```bash yarn
yarn add @langchain/google-vertexai-web @langchain/core
```
```bash pnpm
pnpm add @langchain/google-vertexai-web @langchain/core
```
</CodeGroup>

## Instantiation

이제 model object를 인스턴스화하고 chat completion을 생성할 수 있습니다:

```typescript
import { VertexAI } from "@langchain/google-vertexai-web"

const llm = new VertexAI({
  model: "gemini-pro",
  temperature: 0,
  maxRetries: 2,
  // other params...
})
```

## Invocation

```typescript
const inputText = "VertexAI is an AI company that "

const completion = await llm.invoke(inputText)
completion
```

```txt
offers a wide range of cloud computing services and artificial intelligence solutions to businesses and developers worldwide.
```
