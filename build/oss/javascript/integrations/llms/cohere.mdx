<Warning>
**레거시**

Cohere는 LLM용 `generate` endpoint를 deprecated로 표시했습니다. Chat API를 사용하려면 [`ChatCohere`](/oss/javascript/integrations/chat/cohere) integration을 통해 [마이그레이션 가이드](https://docs.cohere.com/docs/migrating-from-cogenerate-to-cochat)를 따르세요.
</Warning>

<Warning>
**현재 Cohere 모델을 text completion 모델로 사용하는 방법을 설명하는 페이지에 있습니다. Cohere에서 사용 가능한 많은 인기 모델은 [chat completion 모델](/oss/javascript/langchain/models)입니다.**

[이 페이지](/oss/javascript/integrations/chat/cohere/)를 찾고 계실 수 있습니다.
</Warning>

이 문서는 LangChain을 사용하여 Cohere completion 모델(LLM)을 시작하는 데 도움을 드립니다. `Cohere` 기능 및 구성 옵션에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_cohere.Cohere.html)를 참조하세요.

## Setup

Cohere 모델에 액세스하려면 Cohere 계정을 만들고 API key를 받은 다음 `@langchain/cohere` integration package를 설치해야 합니다.

### Credentials

[cohere.com](https://cohere.com)으로 이동하여 Cohere에 가입하고 API key를 생성하세요. 완료한 후 `COHERE_API_KEY` environment variable을 설정하세요:

```bash
export COHERE_API_KEY="your-api-key"
```

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain Cohere integration은 `@langchain/cohere` package에 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/cohere @langchain/core
```
```bash yarn
yarn add @langchain/cohere @langchain/core
```
```bash pnpm
pnpm add @langchain/cohere @langchain/core
```
</CodeGroup>

## Instantiation

이제 model object를 인스턴스화하고 chat completion을 생성할 수 있습니다:

```typescript
import { Cohere } from "@langchain/cohere"

const llm = new Cohere({
  model: "command",
  temperature: 0,
  maxTokens: undefined,
  maxRetries: 2,
  // other params...
})
```

### Custom client for Cohere on Azure, Cohere on AWS Bedrock, and Standalone Cohere Instance

커스텀 `CohereClient`를 인스턴스화하여 ChatCohere constructor에 전달할 수 있습니다.

**참고:** 커스텀 client가 제공되면 `COHERE_API_KEY` environment variable과 constructor의 `apiKey` parameter가 모두 무시됩니다.

```typescript
import { Cohere } from "@langchain/cohere";
import { CohereClient } from "cohere-ai";

const client = new CohereClient({
  token: "<your-api-key>",
  environment: "<your-cohere-deployment-url>", //optional
  // other params
});

const llmWithCustomClient = new Cohere({
  client,
  // other params...
});
```

## Invocation

```typescript
const inputText = "Cohere is an AI company that "

const completion = await llm.invoke(inputText)
completion
```

```output
Cohere is a company that provides natural language processing models that help companies improve human-machine interactions. Cohere was founded in 2019 by Aidan Gomez, Ivan Zhang, and Nick Frosst.
```
