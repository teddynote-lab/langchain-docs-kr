[Fireworks AI](https://fireworks.ai/)는 모델을 실행하고 커스터마이징할 수 있는 AI inference 플랫폼입니다. Fireworks에서 제공하는 모든 모델 목록은 [Fireworks 문서](https://fireworks.ai/models)를 참조하세요.

이 가이드는 LangChain을 사용하여 Fireworks completion 모델(LLM)을 시작하는 데 도움을 드립니다. `Fireworks` 기능 및 구성 옵션에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_community_llms_fireworks.Fireworks.html)를 참조하세요.

## Setup

Fireworks 모델에 액세스하려면 Fireworks 계정을 생성하고 API key를 발급받은 후 `@langchain/community` integration package를 설치해야 합니다.

### Credentials

[fireworks.ai](https://fireworks.ai/)로 이동하여 Fireworks에 가입하고 API key를 생성하세요. 완료한 후 `FIREWORKS_API_KEY` environment variable을 설정하세요:

```bash
export FIREWORKS_API_KEY="your-api-key"
```

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain Fireworks integration은 `@langchain/community` package에 포함되어 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## Instantiation

이제 model 객체를 인스턴스화하고 chat completion을 생성할 수 있습니다:

```typescript
import { Fireworks } from "@langchain/community/llms/fireworks"

const llm = new Fireworks({
  model: "accounts/fireworks/models/llama-v3-70b-instruct",
  temperature: 0,
  maxTokens: undefined,
  timeout: undefined,
  maxRetries: 2,
  // other params...
})
```

## Invocation

```typescript
const inputText = "Fireworks is an AI company that "

const completion = await llm.invoke(inputText)
completion
```
