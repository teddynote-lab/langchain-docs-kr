---
title: HuggingFace Transformers
---

`TransformerEmbeddings` class는 [Transformers.js](https://huggingface.co/docs/transformers.js/index) 패키지를 사용하여 주어진 텍스트에 대한 embedding을 생성합니다.

로컬에서 실행되며 브라우저에서도 직접 작동하므로, 내장된 embedding 기능을 갖춘 웹 앱을 만들 수 있습니다.

## Setup

[@huggingface/transformers](https://www.npmjs.com/package/@huggingface/transformers) 패키지를 peer dependency로 설치해야 합니다:

<Tip>
**호환성**

0.3.21 이전 버전의 community를 사용하는 경우, 이전 버전인 `@xenova/transformers` 패키지를 설치하고
아래에서 `"@langchain/community/embeddings/hf_transformers"`로부터 embedding을 import하세요.
</Tip>

```bash npm
npm install @huggingface/transformers
```
<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/javascript/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/community @langchain/core
```

## Example

브라우저 환경에서 사용하는 경우, 메인 스레드 차단을 방지하기 위해 모든 추론 관련 코드를 web worker에 배치하는 것이 좋습니다.

프로젝트 설정 방법에 대한 아이디어는 [이 가이드](https://huggingface.co/docs/transformers.js/tutorials/next)와 Transformers.js 문서의 다른 리소스를 참조하세요.

```typescript
import { HuggingFaceTransformersEmbeddings } from "@langchain/community/embeddings/huggingface_transformers";

const model = new HuggingFaceTransformersEmbeddings({
  model: "Xenova/all-MiniLM-L6-v2",
});

/* Embed queries */
const res = await model.embedQuery(
  "What would be a good company name for a company that makes colorful socks?"
);
console.log({ res });
/* Embed documents */
const documentRes = await model.embedDocuments(["Hello world", "Bye bye"]);
console.log({ documentRes });
```
