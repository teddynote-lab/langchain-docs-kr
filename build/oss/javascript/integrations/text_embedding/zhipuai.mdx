title: ZhipuAI
---

`ZhipuAIEmbeddings` class는 ZhipuAI API를 사용하여 주어진 텍스트에 대한 embedding을 생성합니다.

## Setup

ZhipuAI API key에 가입하고 `ZHIPUAI_API_KEY`라는 이름의 환경 변수로 설정해야 합니다.

https://open.bigmodel.cn

그런 다음 [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) package를 설치해야 합니다:

<Tip>
LangChain package 설치에 대한 일반적인 지침은 [이 섹션](/oss/javascript/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/community @langchain/core jsonwebtoken
```

## Usage

```typescript
import { ZhipuAIEmbeddings } from "@langchain/community/embeddings/zhipuai";

const model = new ZhipuAIEmbeddings({});
const res = await model.embedQuery(
  "What would be a good company name a company that makes colorful socks?"
);
console.log({ res });
```
