## Setup

이 integration은 `@langchain/community` 패키지에 포함되어 있습니다.

```bash
npm install @langchain/community
```

### Credentials

Google Scholar에 액세스하기 위한 적절한 API key가 있는지 확인하세요. 환경 변수에 설정하세요:

```typescript
process.env.GOOGLE_SCHOLAR_API_KEY="your-serp-api-key"
```

최고 수준의 observability를 위해 [LangSmith](https://smith.langchain.com/)를 설정하는 것도 유용합니다:

```typescript
process.env.LANGSMITH_TRACING="true"
process.env.LANGSMITH_API_KEY="your-langchain-api-key"
```

## Instantiation

다음과 같이 `SERPGoogleScholarAPITool` tool의 인스턴스를 import하고 인스턴스화할 수 있습니다:

```python
import { SERPGoogleScholarAPITool } from "@langchain/community/tools/google_scholar";

const tool = new SERPGoogleScholarAPITool({
  apiKey: process.env.SERPAPI_API_KEY,
});
```

## Invocation

### Invoke directly with args

쿼리 인수로 tool을 직접 호출할 수 있습니다:

```python
const results = await tool.invoke({
  query: "neural networks",
  maxResults: 5,
});

console.log(results);
```

### Invoke with ToolCall

모델이 생성한 `ToolCall`로 tool을 호출할 수도 있습니다:

```python
const modelGeneratedToolCall = {
  args: { query: "machine learning" },
  id: "1",
  name: tool.name,
  type: "tool_call",
};
await tool.invoke(modelGeneratedToolCall);
```
