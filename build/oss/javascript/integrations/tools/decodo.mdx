[@decodo/langchain-ts](https://www.npmjs.com/package/@decodo/langchain-ts) 패키지는 개발자가 LangChain 애플리케이션과 함께 Decodo의 Web Scraper API를 사용할 수 있도록 합니다.

Web Scraper API 기능:

- 간편한 웹 데이터 액세스. 웹사이트 및 온라인 소스에서 정보를 간편하게 검색할 수 있습니다.
- 지리적 유연성. 지역 제한과 관계없이 콘텐츠에 액세스할 수 있습니다.
- 안정적인 스크래핑. 탐지 및 차단을 방지하는 고급 기술을 사용합니다.

## Features

@decodo/langchain-ts 플러그인 기능:

- **Web Scraping**: 모든 URL을 스크래핑하고 Markdown 콘텐츠를 검색합니다
- **Google Search**: Google을 검색하고 구조화된 결과를 검색합니다
- **Amazon Search**: Amazon을 검색하고 구조화된 제품 데이터를 검색합니다
- 위치에 민감한 콘텐츠를 위한 **지리적 위치 선택**
- 실제 브라우저가 필요한 대상을 위한 **JavaScript 렌더링**
- LLM과의 토큰 효율적인 연결을 위한 **Markdown 출력**

## Installation

```bash
npm install @decodo/langchain-ts
```

## Quickstart

이 프로젝트의 도구를 사용하려면 [Decodo Advanced Web Scraping API](https://help.decodo.com/docs/web-scraping-api-core-and-advanced-plans) 구독이 필요합니다. 무료 평가판은 [dashboard](https://dashboard.decodo.com/)에서 이용할 수 있습니다.

## Examples

각 도구에 대한 에이전트 예제 모음입니다.

### Universal

```typescript
import dotenv from "dotenv";
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "@langchain/agents";
import { DecodoUniversalTool } from "@decodo/langchain-ts";

dotenv.config();

const main = async () => {
  const username = process.env.SCRAPER_API_USERNAME!;
  const password = process.env.SCRAPER_API_PASSWORD!;

  const decodoUniversalTool = new DecodoUniversalTool({ username, password });

  const model = new ChatOpenAI({
    model: "gpt-4o-mini",
  });

  const agent = createAgent({
    llm: model,
    tools: [decodoUniversalTool],
  });

  const result = await agent.invoke({
    messages: [
      {
        role: "user",
        content:
          "scrape the wikipedia NBA 2025 season page and tell me who won in 2025?",
      },
    ],
  });

  console.log(result.messages[result.messages.length - 1].content);
};

if (require.main === module) {
  main();
}
```

더 많은 예제는 GitHub의 [@decodo/langchain-ts](https://github.com/Decodo/decodo-langchain-ts)를 참조하세요.

## Google Search

```typescript
  const username = process.env.SCRAPER_API_USERNAME!;
```


## Configuration

모든 도구는 `DecodoConfig` 객체를 허용합니다:

```typescript
type DecodoConfig = {
  username: string; // Your Web Advanced product username
  password: string; // Your Web Advanced product password
};
```
