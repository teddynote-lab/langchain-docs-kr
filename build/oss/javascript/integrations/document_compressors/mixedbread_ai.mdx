## 개요

이 가이드는 [Mixedbread AI](https://mixedbread.ai/) reranking API를 통합하고 사용하는 방법을 안내합니다. reranking API를 사용하면 주어진 query를 기반으로 document 목록을 재정렬하여 검색 결과나 순위 목록의 관련성을 향상시킬 수 있습니다.

## 설치

시작하려면 `@langchain/mixedbread-ai` package를 설치하세요:

<Tip>
LangChain package 설치에 대한 일반적인 지침은 [이 섹션](/oss/javascript/langchain/install)을 참조하세요.
</Tip>

```bash
npm install @langchain/mixedbread-ai
```

## 인증

[Mixedbread AI](https://mixedbread.ai/)에 가입하여 API key를 받으세요. 그런 다음 `MXBAI_API_KEY` 환경 변수를 Mixedbread AI API key로 설정하거나 class를 생성할 때 `apiKey` option으로 직접 전달할 수 있습니다.

## Reranking 사용하기

`MixedbreadAIReranker` class는 reranking API에 대한 접근을 제공합니다. 사용 방법은 다음과 같습니다:

1. **Class Import하기**: 먼저 package에서 `MixedbreadAIReranker` class를 import합니다.

```typescript
import { MixedbreadAIReranker } from "@langchain/mixedbread-ai";
```

2. **Class 인스턴스화하기**: API key를 사용하여 `MixedbreadAIReranker`의 instance를 생성합니다.

```typescript
const reranker = new MixedbreadAIReranker({ apiKey: "your-api-key" });
```

3. **Document Rerank하기**: `rerankDocuments` method를 사용하여 query를 기반으로 document를 재정렬합니다.

```typescript
const documents = [
  { pageContent: "To bake bread you need flour" },
  { pageContent: "To bake bread you need yeast" },
  { pageContent: "To eat bread you need nothing but good taste" },
];
const query = "What do you need to bake bread?";
const result = await reranker.compressDocuments(documents, query);
console.log(result);
```
