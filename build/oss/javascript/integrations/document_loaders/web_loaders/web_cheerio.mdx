이 노트북은 [CheerioWebBaseLoader](/oss/javascript/integrations/document_loaders/)를 시작하기 위한 간단한 개요를 제공합니다. CheerioWebBaseLoader의 모든 기능과 구성에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_community_document_loaders_web_cheerio.CheerioWebBaseLoader.html)를 참조하세요.

## Overview

### Integration details

이 예제는 Cheerio를 사용하여 웹페이지에서 데이터를 로드하는 방법을 다룹니다. 각 웹페이지마다 하나의 document가 생성됩니다.

Cheerio는 jQuery와 유사한 구문을 사용하여 HTML 문서를 파싱하고 탐색할 수 있는 빠르고 경량의 라이브러리입니다. Cheerio를 사용하면 브라우저에서 렌더링하지 않고도 웹 페이지에서 데이터를 추출할 수 있습니다.

그러나 Cheerio는 웹 브라우저를 시뮬레이션하지 않으므로 페이지에서 JavaScript 코드를 실행할 수 없습니다. 즉, JavaScript로 렌더링해야 하는 동적 웹 페이지에서는 데이터를 추출할 수 없습니다. 이를 위해서는 대신 [`PlaywrightWebBaseLoader`](/oss/javascript/integrations/document_loaders/web_loaders/web_playwright) 또는 [`PuppeteerWebBaseLoader`](/oss/javascript/integrations/document_loaders/web_loaders/web_puppeteer)를 사용할 수 있습니다.

## Setup

`CheerioWebBaseLoader` document loader에 접근하려면 `cheerio` peer dependency와 함께 `@langchain/community` integration package를 설치해야 합니다.

### Credentials

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain CheerioWebBaseLoader integration은 `@langchain/community` package에 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core cheerio
```
```bash yarn
yarn add @langchain/community @langchain/core cheerio
```
```bash pnpm
pnpm add @langchain/community @langchain/core cheerio
```
</CodeGroup>

## Instantiation

이제 model object를 인스턴스화하고 document를 로드할 수 있습니다:

```typescript
import { CheerioWebBaseLoader } from "@langchain/community/document_loaders/web/cheerio"

const loader = new CheerioWebBaseLoader("https://news.ycombinator.com/item?id=34817881", {
  // optional params: ...
})
```

const docs = await loader.load()

## Additional configurations

`CheerioWebBaseLoader`는 loader를 인스턴스화할 때 추가 구성을 지원합니다. 다음은 `selector` 필드를 전달하여 제공된 HTML class name에서만 콘텐츠를 로드하도록 하는 사용 예제입니다:

```typescript
import { CheerioWebBaseLoader } from "@langchain/community/document_loaders/web/cheerio"

const loaderWithSelector = new CheerioWebBaseLoader("https://news.ycombinator.com/item?id=34817881", {
  selector: "p",
});

const docsWithSelector = await loaderWithSelector.load();
docsWithSelector[0].pageContent;
```
