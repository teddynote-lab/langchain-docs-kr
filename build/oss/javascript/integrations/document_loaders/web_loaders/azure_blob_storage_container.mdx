이 문서는 Azure Blob Storage의 container를 LangChain document로 로드하는 방법을 다룹니다.

## Setup

이 loader를 실행하려면 사용 가능한 URL endpoint에서 Unstructured가 이미 설정되어 사용 준비가 되어 있어야 합니다. 로컬에서 실행되도록 구성할 수도 있습니다.

설정 방법에 대한 자세한 내용은 [여기](/oss/javascript/integrations/document_loaders/file_loaders/unstructured) 문서를 참조하세요.

또한 공식 Azure Storage Blob client library를 설치해야 합니다:

```bash npm
npm install @langchain/community @langchain/core @azure/storage-blob
```

## Usage

Unstructured가 구성되면 Azure Blob Storage Container loader를 사용하여 파일을 로드하고 Document로 변환할 수 있습니다.

```typescript
import { AzureBlobStorageContainerLoader } from "@langchain/community/document_loaders/web/azure_blob_storage_container";

const loader = new AzureBlobStorageContainerLoader({
  azureConfig: {
    connectionString: "",
    container: "container_name",
  },
  unstructuredConfig: {
    apiUrl: "http://localhost:8000/general/v0/general",
    apiKey: "", // this will be soon required
  },
});

const docs = await loader.load();

console.log(docs);
```
