## Description

[Browserbase](https://browserbase.com)는 headless browser를 안정적으로 실행, 관리 및 모니터링하기 위한 개발자 플랫폼입니다.

다음 기능으로 AI 데이터 검색을 강화하세요:

- 복잡한 UI에서 데이터를 추출할 수 있는 안정적인 브라우저를 제공하는 [Serverless Infrastructure](https://docs.browserbase.com/under-the-hood)
- 지문 인식 방지 기술과 자동 captcha 해결 기능이 포함된 [Stealth Mode](https://docs.browserbase.com/features/stealth-mode)
- 네트워크 타임라인과 로그로 Browser Session을 검사할 수 있는 [Session Debugger](https://docs.browserbase.com/features/sessions)
- 자동화를 빠르게 디버그할 수 있는 [Live Debug](https://docs.browserbase.com/guides/session-debug-connection/browser-remote-control)

## Installation

- [browserbase.com](https://browserbase.com)에서 API key와 Project ID를 발급받아 환경 변수(`BROWSERBASE_API_KEY`, `BROWSERBASE_PROJECT_ID`)에 설정하세요.
- [Browserbase SDK](http://github.com/browserbase/js-sdk)를 설치하세요:

```bash npm
npm i @langchain/community @langchain/core @browserbasehq/sdk
```

## Example

다음과 같이 `BrowserbaseLoader`를 활용하여 agent가 웹사이트를 로드할 수 있도록 하세요:

```typescript
import { BrowserbaseLoader } from "@langchain/community/document_loaders/web/browserbase";

const loader = new BrowserbaseLoader(["https://example.com"], {
  textContent: true,
});
const docs = await loader.load();
```

## Arguments

- `urls`: 필수. 로드할 URL 목록입니다.

## Options

- `textContent` 텍스트 콘텐츠만 검색합니다. 기본값은 `false`입니다.
- `sessionId` 선택사항. 기존 Session ID를 제공합니다.
- `proxy` 선택사항. Proxy를 활성화/비활성화합니다.
```
