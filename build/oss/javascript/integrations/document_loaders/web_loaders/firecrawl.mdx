[FireCrawl](https://firecrawl.dev)은 모든 웹사이트를 크롤링하고 LLM에 적합한 데이터로 변환합니다. 접근 가능한 모든 하위 페이지를 크롤링하고 각 페이지에 대한 깔끔한 markdown과 metadata를 제공합니다. sitemap이 필요하지 않습니다.

FireCrawl은 reverse proxy, caching, rate limit, JavaScript로 차단된 콘텐츠와 같은 복잡한 작업을 처리합니다. [mendable.ai](https://mendable.ai) 팀이 개발했습니다.

이 가이드는 전체 웹사이트를 스크랩하고 크롤링하여 LangChain의 `FireCrawlLoader`를 사용하여 로드하는 방법을 보여줍니다.

## 설정

`FireCrawlLoader` document loader에 액세스하려면 `@langchain/community` integration과 `@mendable/firecrawl-js@0.0.36` package를 설치해야 합니다. 그런 다음 **[FireCrawl](https://firecrawl.dev)** 계정을 만들고 API key를 받으세요.

### 자격 증명

시작하려면 가입하고 무료 [FireCrawl API key](https://firecrawl.dev)를 받으세요. FireCrawl은 시작할 수 있도록 300개의 무료 크레딧을 제공하며, 자체 호스팅을 원하는 경우 [오픈 소스](https://github.com/mendableai/firecrawl)입니다.

완료되면 `FIRECRAWL_API_KEY` 환경 변수를 설정하세요:

```bash
export FIRECRAWL_API_KEY="your-api-key"
```

모델 호출에 대한 자동 추적을 원하는 경우 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 설치

## 인스턴스화

다음은 `FireCrawlLoader`를 사용하여 웹 검색 결과를 로드하는 방법의 예시입니다:

Firecrawl은 `scrape`, `crawl`, `map`의 3가지 모드를 제공합니다. `scrape` 모드에서는 제공한 페이지만 스크랩합니다. `crawl` 모드에서는 전체 웹사이트를 크롤링합니다. `map` 모드에서는 웹사이트와 관련된 의미론적 링크를 반환합니다.

`formats` 매개변수(`crawl` 모드의 경우 `scrapeOptions.formats`)는 `"markdown"`, `"html"`, 또는 `"rawHtml"` 중에서 선택할 수 있습니다. 그러나 로드된 Document는 하나의 형식으로만 콘텐츠를 반환하며, 다음 순서로 우선순위를 지정합니다: `markdown`, `html`, 마지막으로 `rawHtml`.

이제 model 객체를 인스턴스화하고 document를 로드할 수 있습니다:

```typescript
import "@mendable/firecrawl-js";
import { FireCrawlLoader } from "@langchain/community/document_loaders/web/firecrawl"

const loader = new FireCrawlLoader({
  url: "https://firecrawl.dev", // The URL to scrape
  apiKey: "...", // Optional, defaults to `FIRECRAWL_API_KEY` in your env.
  mode: "scrape", // The mode to run the crawler in. Can be "scrape" for single urls or "crawl" for all accessible subpages
  params: {
    // optional parameters based on Firecrawl API docs
    // For API documentation, visit https://docs.firecrawl.dev
  },
})
```
