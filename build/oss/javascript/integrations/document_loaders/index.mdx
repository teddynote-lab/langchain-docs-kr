Document loader는 다양한 소스(Slack, Notion, Google Drive 등)에서 데이터를 읽어 LangChain의 [Document](https://v03.api.js.langchain.com/classes/_langchain_core.documents.Document.html) 형식으로 변환하는 **표준 인터페이스**를 제공합니다.
이를 통해 소스에 관계없이 데이터를 일관되게 처리할 수 있습니다.

모든 document loader는 [BaseLoader](https://v03.api.js.langchain.com/classes/_langchain_core.document_loaders_base.BaseDocumentLoader.html) 인터페이스를 구현합니다.

## Interface

각 document loader는 고유한 매개변수를 정의할 수 있지만, 공통 API를 공유합니다:

- `.load()`: 모든 document를 한 번에 로드합니다.
- `.loadAndSplit()`: 모든 document를 한 번에 로드하고 더 작은 document로 분할합니다.

```typescript
import { CSVLoader } from "@langchain/community/document_loaders/fs/csv";

const loader = new CSVLoader(
  ...  // <-- Integration specific parameters here
);
const data = await loader.load();
```

## 카테고리별

LangChain.js는 document loader를 두 가지 방식으로 분류합니다:

- [File loaders](/oss/javascript/integrations/document_loaders/file_loaders/): 로컬 파일 시스템에서 LangChain 형식으로 데이터를 로드합니다.
- [Web loaders](/oss/javascript/integrations/document_loaders/web_loaders/): 원격 소스에서 데이터를 로드합니다.
