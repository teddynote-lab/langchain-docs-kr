## 설정

`UnstructuredLoader` document loader에 액세스하려면 `@langchain/community` integration package를 설치하고, Unstructured 계정을 생성하여 API key를 받아야 합니다.

### Local

Docker를 사용하여 컴퓨터에서 Unstructured를 로컬로 실행할 수 있습니다. 이를 위해서는 Docker가 설치되어 있어야 합니다. Docker 설치 방법은 [여기](https://docs.docker.com/get-docker/)에서 확인할 수 있습니다.

```bash
docker run -p 8000:8000 -d --rm --name unstructured-api downloads.unstructured.io/unstructured-io/unstructured-api:latest --port 8000 --host 0.0.0.0
```

### Credentials

[unstructured.io](https://unstructured.io/api-key-hosted)로 이동하여 Unstructured에 가입하고 API key를 생성하세요. 완료한 후 `UNSTRUCTURED_API_KEY` environment variable을 설정하세요:

```bash
export UNSTRUCTURED_API_KEY="your-api-key"
```

### 설치

LangChain UnstructuredLoader integration은 `@langchain/community` package에 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## 인스턴스화

이제 model object를 인스턴스화하고 document를 로드할 수 있습니다:

```typescript
import { UnstructuredLoader } from "@langchain/community/document_loaders/fs/unstructured"

const loader = new UnstructuredLoader("../../../../../../examples/src/document_loaders/example_data/notion.mdx")
```

## Load

```typescript
const docs = await loader.load()
docs[0]
```

```output
Document {
  pageContent: '# Testing the notion markdownloader',
  metadata: {
    filename: 'notion.mdx',
    languages: [ 'eng' ],
    filetype: 'text/plain',
    category: 'NarrativeText'
  },
  id: undefined
}
```

```typescript
console.log(docs[0].metadata)
```

```output
{
  filename: 'notion.mdx',
  languages: [ 'eng' ],

}
```

## Directories

[`UnstructuredDirectoryLoader`](https://api.js.langchain.com/classes/langchain.document_loaders_fs_unstructured.UnstructuredDirectoryLoader.html)를 사용하여 디렉토리의 모든 파일을 로드할 수도 있습니다. 이는 [`DirectoryLoader`](/oss/javascript/integrations/document_loaders/file_loaders/directory)를 상속합니다:

```typescript
import { UnstructuredDirectoryLoader } from "@langchain/community/document_loaders/fs/unstructured";

const directoryLoader = new UnstructuredDirectoryLoader(
  "../../../../../../examples/src/document_loaders/example_data/",
  {}
);
const directoryDocs = await directoryLoader.load();
console.log("directoryDocs.length: ", directoryDocs.length);
console.log(directoryDocs[0])

```

```output
Unknown file type: Star_Wars_The_Clone_Wars_S06E07_Crisis_at_the_Heart.srt
Unknown file type: test.mp3
```
```output
directoryDocs.length:  247
Document {
  pageContent: 'Bitcoin: A Peer-to-Peer Electronic Cash System',
  metadata: {
    filetype: 'application/pdf',
    languages: [ 'eng' ],
    page_number: 1,
    filename: 'bitcoin.pdf',
    category: 'Title'
  },
  id: undefined
}
```
