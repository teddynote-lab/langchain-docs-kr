이 노트북은 `DirectoryLoader` [document loaders](/oss/javascript/integrations/document_loaders) 시작하기에 대한 간단한 개요를 제공합니다. 모든 `DirectoryLoader` 기능 및 구성에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain.document_loaders_fs_directory.DirectoryLoader.html)를 참조하세요.

이 예제는 여러 파일이 있는 폴더에서 데이터를 로드하는 방법을 다룹니다. 두 번째 인수는 파일 확장자와 loader factory의 매핑입니다. 각 파일은 일치하는 loader로 전달되며, 결과 document들은 함께 연결됩니다.

예제 폴더:

```text
src/document_loaders/example_data/example/
├── example.json
├── example.jsonl
├── example.txt
└── example.csv
```

## Setup

`DirectoryLoader` document loader에 액세스하려면 `langchain` 패키지를 설치해야 합니다.

### Installation

## Instantiation

이제 model 객체를 인스턴스화하고 document를 로드할 수 있습니다:

```typescript
import { DirectoryLoader } from "@langchain/classic/document_loaders/fs/directory";
import {
  JSONLoader,
  JSONLinesLoader,
} from "@langchain/classic/document_loaders/fs/json";
import { TextLoader } from "@langchain/classic/document_loaders/fs/text";
import { CSVLoader } from "@langchain/community/document_loaders/fs/csv";

const loader = new DirectoryLoader(
  "../../../../../../examples/src/document_loaders/example_data",
  {
    ".json": (path) => new JSONLoader(path, "/texts"),
    ".jsonl": (path) => new JSONLinesLoader(path, "/html"),
    ".txt": (path) => new TextLoader(path),
    ".csv": (path) => new CSVLoader(path, "text"),
  }
);
```

const docs = await loader.load()
