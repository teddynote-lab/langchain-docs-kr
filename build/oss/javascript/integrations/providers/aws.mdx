### SageMaker Endpoint

> [Amazon SageMaker](https://aws.amazon.com/sagemaker/)는 완전 관리형 인프라, 도구 및 워크플로우를 통해 머신러닝(ML) 모델을 구축, 학습 및 배포할 수 있는 시스템입니다.

우리는 `SageMaker`를 사용하여 모델을 호스팅하고 이를 `SageMaker Endpoint`로 노출합니다.

[사용 예제](/oss/javascript/integrations/llms/aws_sagemaker)를 참조하세요.

```typescript
import {
  SagemakerEndpoint,
  SageMakerLLMContentHandler,
} from "@langchain/community/llms/sagemaker_endpoint";
```

## Document loaders

### AWS S3 Directory and File

> [Amazon Simple Storage Service (Amazon S3)](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html)는 객체 스토리지 서비스입니다.
> [AWS S3 Directory](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html) >[AWS S3 Buckets](https://docs.aws.amazon.com/AmazonS3/latest/userguide/UsingBucket.html)

[S3FileLoader 사용 예제](/oss/javascript/integrations/document_loaders/web_loaders/s3)를 참조하세요.

```bash npm
npm install @aws-sdk/client-s3
```
```typescript
import { S3Loader } from "@langchain/community/document_loaders/web/s3";
```
