title: Chat model
---

[Chat model](/oss/javascript/langchain/models)은 [message](/oss/javascript/langchain/messages) 시퀀스를 입력으로 사용하고 message를 출력으로 반환하는 language model입니다 <Tooltip tip="이름에 'Chat' 접두사가 포함되지 않거나 'LLM'이 접미사로 포함된 모델은 일반적으로 chat model 인터페이스를 따르지 않고 대신 문자열을 입력으로 받아 문자열을 출력으로 반환하는 인터페이스를 사용하는 이전 모델을 의미합니다.">(일반 텍스트와 대조적으로)</Tooltip>.

## 설치 및 사용

<Tip>
[LangChain 패키지 설치에 대한 일반 지침은 이 섹션을 참조하세요](/oss/javascript/langchain/install).
</Tip>

<AccordionGroup>
<Accordion title="Groq">

설치:

<CodeGroup>
```bash npm
npm i @langchain/groq
```
```bash yarn
yarn add @langchain/groq
```
```bash pnpm
pnpm add @langchain/groq
```
</CodeGroup>

환경 변수 추가:

```bash
GROQ_API_KEY=your-api-key
```

모델 인스턴스화:

```typescript
import { ChatGroq } from "@langchain/groq";

const model = new ChatGroq({
  model: "llama-3.3-70b-versatile",
  temperature: 0
});
```
```javascript
await model.invoke("Hello, world!")
```

</Accordion>
