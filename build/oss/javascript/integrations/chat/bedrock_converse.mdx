[Amazon Bedrock Converse](https://docs.aws.amazon.com/bedrock/latest/APIReference/API_runtime_Converse.html)는 선도적인 AI 스타트업과 Amazon의 Foundation Model(FM)을 API를 통해 사용할 수 있도록 하는 완전 관리형 서비스입니다. 다양한 FM 중에서 사용 사례에 가장 적합한 모델을 선택할 수 있습니다. Bedrock 모델을 위한 통합된 대화형 인터페이스를 제공하지만, 이전 [Bedrock 모델 서비스](/oss/javascript/integrations/chat/bedrock)의 모든 기능과 완전히 동등한 기능을 아직 제공하지는 않습니다.

이 문서는 Amazon Bedrock Converse [chat model](/oss/javascript/langchain/models) 시작하기를 도와드립니다. 모든 `ChatBedrockConverse` 기능 및 구성에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_aws.ChatBedrockConverse.html)를 참조하세요.

## Setup

Bedrock 모델에 액세스하려면 AWS 계정을 생성하고, Bedrock API 서비스를 설정하고, access key ID와 secret key를 받고, `@langchain/community` integration package를 설치해야 합니다.

### Credentials

AWS에 가입하고 자격 증명을 설정하려면 [AWS 문서](https://docs.aws.amazon.com/bedrock/latest/userguide/getting-started.html)를 참조하세요. 또한 [다음 지침](https://docs.aws.amazon.com/bedrock/latest/userguide/model-access.html)에 따라 계정에 대한 모델 액세스를 활성화해야 합니다.

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

## Instantiation

이제 model object를 인스턴스화하고 chat completion을 생성할 수 있습니다.

AWS 인증 방법은 여러 가지가 있습니다 - 아래 예제는 환경 변수에 설정된 access key, secret access key 및 region을 사용합니다:

```typescript
import { ChatBedrockConverse } from "@langchain/aws";

const llm = new ChatBedrockConverse({
  model: "anthropic.claude-3-5-sonnet-20240620-v1:0",
  region: process.env.BEDROCK_AWS_REGION,
  credentials: {
    accessKeyId: process.env.BEDROCK_AWS_ACCESS_KEY_ID!,
    secretAccessKey: process.env.BEDROCK_AWS_SECRET_ACCESS_KEY!,
  },
});
```

## Tool calling

Bedrock 모델의 tool calling은 [다른 모델](/oss/javascript/langchain/tools)과 유사한 방식으로 작동하지만, 모든 Bedrock 모델이 tool calling을 지원하는 것은 아닙니다. 자세한 내용은 [AWS 모델 문서](https://docs.aws.amazon.com/bedrock/latest/APIReference/welcome.html)를 참조하세요.
