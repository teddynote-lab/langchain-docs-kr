## Setup

`ChatFireworks` 모델에 액세스하려면 Fireworks 계정을 만들고 API key를 발급받은 후 `@langchain/community` integration package를 설치해야 합니다.

### Credentials

[Fireworks 웹사이트](https://fireworks.ai/login)로 이동하여 Fireworks에 가입하고 API key를 생성하세요. 완료한 후 `FIREWORKS_API_KEY` environment variable을 설정하세요:

```bash
export FIREWORKS_API_KEY="your-api-key"
```

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain `ChatFireworks` integration은 `@langchain/community` package에 포함되어 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## Instantiation

이제 model 객체를 인스턴스화하고 chat completion을 생성할 수 있습니다:

```typescript
import { ChatFireworks } from "@langchain/community/chat_models/fireworks"

const llm = new ChatFireworks({
    model: "accounts/fireworks/models/llama-v3p1-70b-instruct",
    temperature: 0,
    maxTokens: undefined,
    timeout: undefined,
    maxRetries: 2,
    // other params...
})
```

## Invocation

const aiMsg = await llm.invoke([
