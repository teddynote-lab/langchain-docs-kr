title: TavilySearchAPIRetriever

[Tavily의 Search API](https://tavily.com)는 AI 에이전트(LLM)를 위해 특별히 구축된 검색 엔진으로, 실시간으로 정확하고 사실적인 결과를 빠르게 제공합니다.

## Overview

이 문서는 Tavily Search API [retriever](/oss/javascript/langchain/retrieval) 시작하기를 도와줍니다. 모든 `TavilySearchAPIRetriever` 기능과 설정에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_community_retrievers_tavily_search_api.TavilySearchAPIRetriever.html)를 참조하세요.

### Integration details

| Retriever | Source | Package |
| :--- | :--- | :---: |
[`TavilySearchAPIRetriever`](https://api.js.langchain.com/classes/langchain_community_retrievers_tavily_search_api.TavilySearchAPIRetriever.html) | 웹상의 정보. | [`@langchain/community`](https://npmjs.com/@langchain/community/) |

## Setup

Tavily API 키를 `TAVILY_API_KEY` 환경 변수에 설정하거나 생성자에 `apiKey`로 전달해야 합니다. [웹사이트](https://tavily.com/)에서 가입하여 키를 발급받으세요.

개별 쿼리에서 자동 추적을 원하는 경우, 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API 키를 설정할 수도 있습니다:

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

### Installation

이 retriever는 `@langchain/community` 패키지에 포함되어 있습니다:

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## Instantiation

이제 retriever를 인스턴스화할 수 있습니다:

```typescript
import { TavilySearchAPIRetriever } from "@langchain/community/retrievers/tavily_search_api";

const retriever = new TavilySearchAPIRetriever({
  k: 3,
});
```

허용되는 인자의 전체 목록은 [공식 문서](https://docs.tavily.com/docs/tavily-api/rest_api#parameters)를 참조하세요. `kwargs` 객체를 통해 SDK에 모든 매개변수를 전달할 수 있습니다.

## Usage

```typescript
const query = "what is the current weather in SF?";

await retriever.invoke(query);
```
