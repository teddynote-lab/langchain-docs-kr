---
title: Retriever
---

[Retriever](/oss/javascript/langchain/retrieval)는 비구조화된 query가 주어졌을 때 문서를 반환하는 interface입니다.
Vector store보다 더 일반적인 개념입니다.
Retriever는 문서를 저장할 수 있을 필요는 없으며, 단지 문서를 반환(또는 검색)하기만 하면 됩니다.

Retriever는 문자열 query를 입력으로 받아 Document 목록을 반환합니다.

Retriever 사용 방법에 대한 자세한 내용은 [관련 how-to 가이드](/oss/javascript/langchain/retrieval)를 참조하세요.

모든 [vector store](/oss/javascript/integrations/vectorstores)는 [retriever로 변환](/oss/javascript/langchain/retrieval)될 수 있습니다.
사용 가능한 vector store retriever에 대해서는 vector store [통합 문서](/oss/javascript/integrations/vectorstores/)를 참조하세요.
