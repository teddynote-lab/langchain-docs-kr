title: Momento Vector Index (MVI)

[MVI](https://gomomento.com): 데이터를 위한 가장 생산적이고 사용하기 쉬운 서버리스 vector index입니다. MVI를 시작하려면 계정에 가입하기만 하면 됩니다. 인프라를 처리하거나 서버를 관리하거나 확장에 대해 걱정할 필요가 없습니다. MVI는 필요에 따라 자동으로 확장되는 서비스입니다. Node.js, 브라우저 또는 edge 환경 어디에서든 Momento가 지원합니다.

가입하고 MVI에 액세스하려면 [Momento Console](https://console.gomomento.com)을 방문하세요.

## Setup

1. [Momento Console](https://console.gomomento.com/)에서 API key에 가입하세요.
2. 환경에 맞는 SDK를 설치하세요.

   2.1. **Node.js**의 경우:

   ```bash npm
   npm install @gomomento/sdk
   ```
   2.2. **브라우저 또는 edge 환경**의 경우:

   ```bash npm
   npm install @gomomento/sdk-web
   ```

3. 코드를 실행하기 전에 Momento에 대한 환경 변수를 설정하세요

   3.1 OpenAI

   ```bash
   export OPENAI_API_KEY=YOUR_OPENAI_API_KEY_HERE
   ```

   3.2 Momento

   ```bash
   export MOMENTO_API_KEY=YOUR_MOMENTO_API_KEY_HERE # https://console.gomomento.com
   ```

### `fromTexts`를 사용하여 문서 인덱싱 및 검색

이 예제는 `fromTexts` 메서드를 사용하여 vector store를 인스턴스화하고 문서를 인덱싱하는 방법을 보여줍니다.
index가 존재하지 않으면 생성됩니다. index가 이미 존재하면 문서가 기존 index에 추가됩니다.

`ids`는 선택 사항입니다. 생략하면 Momento가 UUID를 생성합니다.

```typescript
import { MomentoVectorIndex } from "@langchain/community/vectorstores/momento_vector_index";
// For browser/edge, adjust this to import from "@gomomento/sdk-web";
import {
  PreviewVectorIndexClient,
  VectorIndexConfigurations,
  CredentialProvider,
} from "@gomomento/sdk";
import { OpenAIEmbeddings } from "@langchain/openai";
import { sleep } from "@langchain/classic/util/time";

const vectorStore = await MomentoVectorIndex.fromTexts(
  ["hello world", "goodbye world", "salutations world", "farewell world"],
  {},
  new OpenAIEmbeddings(),
  {
    client: new PreviewVectorIndexClient({
      configuration: VectorIndexConfigurations.Laptop.latest(),
      credentialProvider: CredentialProvider.fromEnvironmentVariable({
        environmentVariableName: "MOMENTO_API_KEY",
      }),
    }),
    indexName: "langchain-example-index",
  },
  { ids: ["1", "2", "3", "4"] }
);

// because indexing is async, wait for it to finish to search directly after
await sleep();

const response = await vectorStore.similaritySearch("hello", 2);

console.log(response);

/*
[
  Document { pageContent: 'hello world', metadata: {} },
  Document { pageContent: 'salutations world', metadata: {} }
]
*/
```

### `fromDocuments`를 사용하여 문서 인덱싱 및 검색

위와 유사하게, 이 예제는 `fromDocuments` 메서드를 사용하여 vector store를 인스턴스화하고 문서를 인덱싱하는 방법을 보여줍니다.
index가 존재하지 않으면 생성됩니다. index가 이미 존재하면 문서가 기존 index에 추가됩니다.

`fromDocuments`를 사용하면 다양한 document loader를 인덱싱과 원활하게 연결할 수 있습니다.

```typescript
import { MomentoVectorIndex } from "@langchain/community/vectorstores/momento_vector_index";
// For browser/edge, adjust this to import from "@gomomento/sdk-web";
import {
  PreviewVectorIndexClient,
  VectorIndexConfigurations,
  CredentialProvider,
} from "@gomomento/sdk";
import { OpenAIEmbeddings } from "@langchain/openai";
import { TextLoader } from "@langchain/classic/document_loaders/fs/text";
import { sleep } from "@langchain/classic/util/time";

// Create docs with a loader
const loader = new TextLoader("src/document_loaders/example_data/example.txt");
const docs = await loader.load();

const vectorStore = await MomentoVectorIndex.fromDocuments(
  docs,
  new OpenAIEmbeddings(),
  {
    client: new PreviewVectorIndexClient({
      configuration: VectorIndexConfigurations.Laptop.latest(),
      credentialProvider: CredentialProvider.fromEnvironmentVariable({
        environmentVariableName: "MOMENTO_API_KEY",
      }),
    }),
    indexName: "langchain-example-index",
  }
);

// because indexing is async, wait for it to finish to search directly after
await sleep();

// Search for the most similar document
const response = await vectorStore.similaritySearch("hello", 1);

console.log(response);
/*
[
  Document {
    pageContent: 'Foo\nBar\nBaz\n\n',
    metadata: { source: 'src/document_loaders/example_data/example.txt' }
  }
]
*/
```

### 기존 collection에서 검색

```typescript
import { MomentoVectorIndex } from "@langchain/community/vectorstores/momento_vector_index";
// For browser/edge, adjust this to import from "@gomomento/sdk-web";
import {
  PreviewVectorIndexClient,
  VectorIndexConfigurations,
  CredentialProvider,
} from "@gomomento/sdk";
import { OpenAIEmbeddings } from "@langchain/openai";

const vectorStore = new MomentoVectorIndex(new OpenAIEmbeddings(), {
  client: new PreviewVectorIndexClient({
    configuration: VectorIndexConfigurations.Laptop.latest(),
    credentialProvider: CredentialProvider.fromEnvironmentVariable({
      environmentVariableName: "MOMENTO_API_KEY",
    }),
  }),
  indexName: "langchain-example-index",
});

const response = await vectorStore.similaritySearch("hello", 1);

console.log(response);
/*
[
  Document {
    pageContent: 'Foo\nBar\nBaz\n\n',
    metadata: { source: 'src/document_loaders/example_data/example.txt' }
  }
]
*/
```
