이 페이지는 LangGraph로 개발하는 동안 발생할 수 있는 일반적인 오류를 해결하는 가이드를 포함하고 있습니다.
아래에서 참조되는 오류들은 코드에서 발생할 때 아래 코드 중 하나에 해당하는 `lc_error_code` 속성을 가집니다.

* [GRAPH_RECURSION_LIMIT](/oss/javascript/langgraph/GRAPH_RECURSION_LIMIT)
* [INVALID_CONCURRENT_GRAPH_UPDATE](/oss/javascript/langgraph/INVALID_CONCURRENT_GRAPH_UPDATE)
* [INVALID_GRAPH_NODE_RETURN_VALUE](/oss/javascript/langgraph/INVALID_GRAPH_NODE_RETURN_VALUE)
* [MULTIPLE_SUBGRAPHS](/oss/javascript/langgraph/MULTIPLE_SUBGRAPHS)
* [INVALID_CHAT_HISTORY](/oss/javascript/langgraph/INVALID_CHAT_HISTORY)
