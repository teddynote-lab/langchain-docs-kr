sidebarTitle: 설치

기본 LangGraph 패키지를 설치하려면:



<CodeGroup>
```bash npm
npm install @langchain/langgraph @langchain/core
```

```bash pnpm
pnpm add @langchain/langgraph @langchain/core
```

```bash yarn
yarn add @langchain/langgraph @langchain/core
```

```bash bun
bun add @langchain/langgraph @langchain/core
```
</CodeGroup>


LangGraph를 사용하려면 일반적으로 LLM에 액세스하고 tool을 정의해야 합니다.
이는 원하는 방식으로 수행할 수 있습니다.

이를 수행하는 한 가지 방법(문서에서 사용할 방법)은 [LangChain](/oss/javascript/langchain/overview)을 사용하는 것입니다.

LangChain 설치:



<CodeGroup>
```bash npm
npm install langchain
```

```bash pnpm
pnpm add langchain
```

```bash yarn
yarn add langchain
```

```bash bun
bun add langchain
```
</CodeGroup>


특정 LLM provider 패키지를 사용하려면 별도로 설치해야 합니다.

provider별 설치 지침은 [integrations](/oss/javascript/integrations/providers/overview) 페이지를 참조하세요.
