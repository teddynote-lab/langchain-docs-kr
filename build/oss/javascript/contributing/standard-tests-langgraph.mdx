**Standard test는 통합이 예상대로 작동하는지 확인합니다.**

자신을 위한 custom class를 만들거나 LangGraph integration에 게시할 때, 예상대로 작동하는지 확인하기 위해 standard test를 추가하는 것이 중요합니다. 이 가이드는 각 integration 유형에 standard test를 추가하는 방법을 보여줍니다.
