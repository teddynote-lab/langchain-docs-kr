>[TF-IDF](https://scikit-learn.org/stable/modules/feature_extraction.html#tfidf-term-weighting)는 term-frequency times inverse document-frequency를 의미합니다.

이 노트북은 `scikit-learn` 패키지를 사용하여 내부적으로 [TF-IDF](https://en.wikipedia.org/wiki/Tf%E2%80%93idf)를 사용하는 retriever를 사용하는 방법을 다룹니다.

TF-IDF의 세부 사항에 대한 자세한 내용은 [이 블로그 포스트](https://medium.com/data-science-bootcamp/tf-idf-basics-of-information-retrieval-48de122b2a4c)를 참조하세요.

```python
pip install -qU  scikit-learn
```

```python
from langchain_community.retrievers import TFIDFRetriever
```

## 텍스트로 새 Retriever 생성하기

```python
retriever = TFIDFRetriever.from_texts(["foo", "bar", "world", "hello", "foo bar"])
```

## Document로 새 Retriever 생성하기

이제 생성한 document로 새 retriever를 만들 수 있습니다.

```python
from langchain_core.documents import Document

retriever = TFIDFRetriever.from_documents(
    [
        Document(page_content="foo"),
        Document(page_content="bar"),
        Document(page_content="world"),
        Document(page_content="hello"),
        Document(page_content="foo bar"),
    ]
)
```

## Retriever 사용하기

이제 retriever를 사용할 수 있습니다!

```python
result = retriever.invoke("foo")
```

```python
result
```

```output
[Document(page_content='foo', metadata={}),
 Document(page_content='foo bar', metadata={}),
 Document(page_content='hello', metadata={}),
 Document(page_content='world', metadata={})]
```

## 저장 및 로드

이 retriever를 쉽게 저장하고 로드할 수 있어 로컬 개발에 편리합니다!

```python
retriever.save_local("testing.pkl")
```

```python
retriever_copy = TFIDFRetriever.load_local("testing.pkl")
```

```python
retriever_copy.invoke("foo")
```

```output
[Document(page_content='foo', metadata={}),
 Document(page_content='foo bar', metadata={}),
 Document(page_content='hello', metadata={}),
 Document(page_content='world', metadata={})]
```
