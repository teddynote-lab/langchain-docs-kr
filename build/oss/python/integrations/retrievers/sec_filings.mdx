>[SEC filing](https://www.sec.gov/edgar)은 미국 증권거래위원회(SEC)에 제출되는 재무제표 또는 기타 공식 문서입니다. 상장 기업, 특정 내부자 및 브로커-딜러는 정기적으로 `SEC filing`을 제출해야 합니다. 투자자와 금융 전문가들은 투자 목적으로 평가하는 기업에 대한 정보를 얻기 위해 이러한 filing에 의존합니다.
>
>`SEC filing` 데이터는 [Snowflake Marketplace](https://app.snowflake.com/marketplace/providers/GZTSZAS2KCS/Cybersyn%2C%20Inc)를 통해 [Kay.ai](https://kay.ai)와 [Cybersyn](https://www.cybersyn.com/)에서 제공합니다.

## Setup

먼저 `kay` package를 설치해야 합니다. 또한 API key가 필요합니다: [https://kay.ai](https://kay.ai/)에서 무료로 받을 수 있습니다. API key를 받은 후에는 환경 변수 `KAY_API_KEY`로 설정해야 합니다.

이 예제에서는 `KayAiRetriever`를 사용할 것입니다. 이 retriever가 허용하는 parameter에 대한 자세한 정보는 [kay notebook](/oss/python/integrations/retrievers/kay)을 참조하세요.

```python
# Setup API keys for Kay and OpenAI
from getpass import getpass

KAY_API_KEY = getpass()
OPENAI_API_KEY = getpass()
```

import os

## Example

```python
from langchain.chains import ConversationalRetrievalChain
from langchain_community.retrievers import KayAiRetriever
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-3.5-turbo")
retriever = KayAiRetriever.create(
    dataset_id="company", data_types=["10-K", "10-Q"], num_contexts=6
)
qa = ConversationalRetrievalChain.from_llm(model, retriever=retriever)
```

```python
questions = [
    "What are patterns in Nvidia's spend over the past three quarters?",
    # "What are some recent challenges faced by the renewable energy sector?",
]
chat_history = []

for question in questions:
    result = qa({"question": question, "chat_history": chat_history})
    chat_history.append((question, result["answer"]))
    print(f"-> **Question**: {question} \n")
    print(f"**Answer**: {result['answer']} \n")
```
