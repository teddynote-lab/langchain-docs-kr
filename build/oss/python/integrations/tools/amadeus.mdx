title: Amadeus Toolkit
---

이 노트북은 LangChain을 `Amadeus` 여행 API에 연결하는 방법을 안내합니다.

이 `Amadeus` toolkit은 agent가 여행과 관련된 결정을 내릴 수 있도록 하며, 특히 항공편 검색 및 예약을 지원합니다.

이 toolkit을 사용하려면 Amadeus API key가 필요하며, 자세한 내용은 [Get started Amadeus Self-Service APIs](https://developers.amadeus.com/get-started/get-started-with-self-service-apis-335)에서 확인할 수 있습니다. AMADEUS_CLIENT_ID와 AMADEUS_CLIENT_SECRET를 받으면 아래와 같이 환경 변수로 입력할 수 있습니다.

참고: Amadeus Self-Service API는 [무료 제한 데이터](https://amadeus4dev.github.io/developer-guides/test-data/)가 제공되는 테스트 환경을 제공합니다. 이를 통해 개발자는 프로덕션에 배포하기 전에 애플리케이션을 빌드하고 테스트할 수 있습니다. 실시간 데이터에 액세스하려면 [프로덕션 환경으로 이동](https://amadeus4dev.github.io/developer-guides/API-Keys/moving-to-production/)해야 합니다.

```python
pip install -qU  amadeus > /dev/null
```

```python
pip install -qU langchain-community
```

## 환경 변수 할당

toolkit은 사용자를 인증하기 위해 AMADEUS_CLIENT_ID와 AMADEUS_CLIENT_SECRET 환경 변수를 읽으므로 여기에서 설정해야 합니다.

```python
# Set environmental variables here
import os

os.environ["AMADEUS_CLIENT_ID"] = "CLIENT_ID"
os.environ["AMADEUS_CLIENT_SECRET"] = "CLIENT_SECRET"

# os.environ["AMADEUS_HOSTNAME"] = "production" or "test"
```

## Amadeus Toolkit 생성 및 Tool 가져오기

시작하려면 toolkit을 생성하여 나중에 해당 tool에 액세스할 수 있도록 해야 합니다.

기본적으로 `AmadeusToolkit`은 [`ChatOpenAI`](https://reference.langchain.com/python/integrations/langchain_openai/ChatOpenAI/)를 사용하여 주어진 위치에서 가장 가까운 공항을 식별합니다. 이를 사용하려면 `OPENAI_API_KEY`를 설정하기만 하면 됩니다.

```python
os.environ["OPENAI_API_KEY"] = "YOUR_OPENAI_API_KEY"
```

```python
from langchain_community.agent_toolkits.amadeus.toolkit import AmadeusToolkit

toolkit = AmadeusToolkit()
tools = toolkit.get_tools()
```

또는 `HuggingFaceHub`와 같이 langchain에서 지원하는 모든 LLM을 사용할 수 있습니다.

```python
from langchain_community.llms import HuggingFaceHub

os.environ["HUGGINGFACEHUB_API_TOKEN"] = "YOUR_HF_API_TOKEN"

llm = HuggingFaceHub(
    repo_id="tiiuae/falcon-7b-instruct",
    model_kwargs={"temperature": 0.5, "max_length": 64},
)

toolkit_hf = AmadeusToolkit(llm=llm)
```

## Agent 내에서 Amadeus Toolkit 사용

```python
from langchain_classic import hub
from langchain.agents import AgentExecutor, create_agent
from langchain.agents.output_parsers import ReActJsonSingleInputOutputParser
from langchain.tools.render import render_text_description_and_args
from langchain_openai import ChatOpenAI
```

```python
llm = ChatOpenAI(temperature=0)

prompt = hub.pull("hwchase17/react-json")
agent = create_agent(
    llm,
    tools,
    prompt,
    tools_renderer=render_text_description_and_args,
    output_parser=ReActJsonSingleInputOutputParser(),
)

agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    verbose=True,
)
```

```python
agent_executor.invoke({"input": "What is the name of the airport in Cali, Colombia?"})
```
