title: Mojeek Search
---

다음 노트북은 Mojeek Search를 사용하여 결과를 얻는 방법을 설명합니다. API key를 얻으려면 [Mojeek 웹사이트](https://www.mojeek.com/services/search/web-search-api/)를 방문하세요.

```python
from langchain_community.tools import MojeekSearch
```

```python
api_key = "KEY"  # obtained from Mojeek Website
```

```python
search = MojeekSearch.config(api_key=api_key, search_kwargs={"t": 10})
```

`search_kwargs`에는 [Mojeek 문서](https://www.mojeek.com/support/api/search/request_parameters.html)에서 찾을 수 있는 모든 검색 매개변수를 추가할 수 있습니다.

```python
search.run("mojeek")
```
