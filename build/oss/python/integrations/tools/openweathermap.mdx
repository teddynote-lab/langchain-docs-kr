이 노트북은 날씨 정보를 가져오기 위해 `OpenWeatherMap` component를 사용하는 방법을 다룹니다.

먼저, `OpenWeatherMap API` key에 가입해야 합니다:

1. OpenWeatherMap으로 이동하여 [여기](https://openweathermap.org/api/)에서 API key에 가입하세요
2. pip install pyowm

그런 다음 몇 가지 environment variable을 설정해야 합니다:

1. API KEY를 OPENWEATHERMAP_API_KEY env variable에 저장하세요

## wrapper 사용하기

```python
pip install -qU pyowm
```

```output
Note: you may need to restart the kernel to use updated packages.
```

```python
import os

from langchain_community.utilities import OpenWeatherMapAPIWrapper

os.environ["OPENWEATHERMAP_API_KEY"] = ""

weather = OpenWeatherMapAPIWrapper()
```

```python
weather_data = weather.run("London,GB")
print(weather_data)
```


## tool 사용하기

```python
import os

from langchain.agents import create_agent


os.environ["OPENAI_API_KEY"] = ""
os.environ["OPENWEATHERMAP_API_KEY"] = ""

tools = [weather.run]
agent = create_agent("openai:gpt-4.1-mini", tools)
```

```python
input_message = {
    "role": "user",
    "content": "What's the weather like in London?",
}

for step in agent.stream(
    {"messages": [input_message]},
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```
