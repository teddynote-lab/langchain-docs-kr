title: acreom
---

[acreom](https://acreom.com)은 로컬 markdown 파일에서 작업을 실행하는 개발자 우선 지식 베이스입니다.

아래는 로컬 acreom vault를 LangChain에 로드하는 방법에 대한 예제입니다. acreom의 로컬 vault는 일반 텍스트 .md 파일의 폴더이므로, loader는 디렉토리 경로가 필요합니다.

Vault 파일에는 YAML 헤더로 저장된 일부 메타데이터가 포함될 수 있습니다. `collect_metadata`가 true로 설정된 경우 이러한 값들이 문서의 메타데이터에 추가됩니다.

```python
from langchain_community.document_loaders import AcreomLoader
```

```python
loader = AcreomLoader("<path-to-acreom-vault>", collect_metadata=False)
```

```python
docs = loader.load()
```
