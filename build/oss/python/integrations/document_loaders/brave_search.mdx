>[Brave Search](https://en.wikipedia.org/wiki/Brave_Search)는 Brave Software에서 개발한 검색 엔진입니다.
>
> - `Brave Search`는 자체 웹 인덱스를 사용합니다. 2022년 5월 기준으로 100억 개 이상의 페이지를 포함하고 있으며, 제3자에 의존하지 않고 검색 결과의 92%를 제공하는 데 사용되었습니다. 나머지는 Bing API에서 서버 측으로 또는 (선택 사항으로) Google에서 클라이언트 측으로 검색됩니다. Brave에 따르면, 이 인덱스는 스팸 및 기타 저품질 콘텐츠를 피하기 위해 "의도적으로 Google이나 Bing보다 작게" 유지되었으며, 그 단점으로 "Brave Search는 롱테일 쿼리 복구에서 아직 Google만큼 좋지 않습니다."
>- `Brave Search Premium`: 2023년 4월 기준으로 Brave Search는 광고 없는 웹사이트이지만, 결국 광고를 포함하는 새로운 모델로 전환될 예정이며 프리미엄 사용자는 광고 없는 경험을 얻게 됩니다. IP 주소를 포함한 사용자 데이터는 기본적으로 사용자로부터 수집되지 않습니다. 선택적 데이터 수집을 위해서는 프리미엄 계정이 필요합니다.

## Installation and Setup

Brave Search API에 액세스하려면 [계정을 생성하고 API 키를 받아야](https://api.search.brave.com/app/dashboard) 합니다.

```python
api_key = "..."
```

```python
from langchain_community.document_loaders import BraveSearchLoader
```

## Example

```python
loader = BraveSearchLoader(
    query="obama middle name", api_key=api_key, search_kwargs={"count": 3}
)
docs = loader.load()
len(docs)
```

```output
3
```

```python
[doc.metadata for doc in docs]
```
