title: AWS S3 Directory

>[Amazon Simple Storage Service (Amazon S3)](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html)는 객체 스토리지 서비스입니다

>[AWS S3 Directory](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html)

이 문서는 `AWS S3 Directory` 객체에서 document 객체를 로드하는 방법을 다룹니다.

```python
pip install -qU  boto3
```

```python
from langchain_community.document_loaders import S3DirectoryLoader
```

```python
loader = S3DirectoryLoader("testing-hwc")
```

```python
loader.load()
```

## prefix 지정하기

로드할 파일을 더 세밀하게 제어하기 위해 prefix를 지정할 수도 있습니다.

```python
loader = S3DirectoryLoader("testing-hwc", prefix="fake")
```

```python
loader.load()
```

```output
[Document(page_content='Lorem ipsum dolor sit amet.', lookup_str='', metadata={'source': 's3://testing-hwc/fake.docx'}, lookup_index=0)]
```

## AWS Boto3 client 설정하기

S3DirectoryLoader를 생성할 때 named argument를 전달하여 AWS [Boto3](https://boto3.amazonaws.com/v1/documentation/api/latest/index.html) client를 설정할 수 있습니다.
이는 예를 들어 AWS 자격 증명을 환경 변수로 설정할 수 없는 경우에 유용합니다.
설정할 수 있는 [parameter 목록](https://boto3.amazonaws.com/v1/documentation/api/latest/reference/core/session.html#boto3.session.Session)을 참조하세요.

```python
loader = S3DirectoryLoader(
    "testing-hwc", aws_access_key_id="xxxx", aws_secret_access_key="yyyy"
)
```

```python
loader.load()
```
