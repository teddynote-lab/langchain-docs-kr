>[SurrealDB](https://surrealdb.com/)는 웹, 모바일, 서버리스, Jamstack, 백엔드 및 전통적인 애플리케이션을 포함한 현대적인 애플리케이션을 위해 설계된 엔드투엔드 클라우드 네이티브 데이터베이스입니다. SurrealDB를 사용하면 데이터베이스와 API 인프라를 단순화하고, 개발 시간을 단축하며, 안전하고 성능이 뛰어난 앱을 빠르고 비용 효율적으로 구축할 수 있습니다.
>
>**SurrealDB의 주요 기능:**
>
>* **개발 시간 단축:** SurrealDB는 대부분의 서버 측 컴포넌트의 필요성을 제거하여 데이터베이스와 API 스택을 단순화하므로, 안전하고 성능이 뛰어난 앱을 더 빠르고 저렴하게 구축할 수 있습니다.
>* **실시간 협업 API 백엔드 서비스:** SurrealDB는 데이터베이스와 API 백엔드 서비스 모두로 작동하여 실시간 협업을 가능하게 합니다.
>* **다양한 쿼리 언어 지원:** SurrealDB는 클라이언트 디바이스에서의 SQL 쿼리, GraphQL, ACID 트랜잭션, WebSocket 연결, 구조화 및 비구조화 데이터, 그래프 쿼리, 전문 인덱싱 및 지리공간 쿼리를 지원합니다.
>* **세밀한 접근 제어:** SurrealDB는 행 수준 권한 기반 접근 제어를 제공하여 데이터 접근을 정밀하게 관리할 수 있습니다.
>
>[기능](https://surrealdb.com/features), 최신 [릴리스](https://surrealdb.com/releases) 및 [문서](https://surrealdb.com/docs)를 확인하세요.

이 노트북은 `SurrealDBLoader`와 관련된 기능을 사용하는 방법을 보여줍니다.

## 개요

SurrealDB Document Loader는 SurrealDB 데이터베이스에서 LangChain Document 목록을 반환합니다.

Document Loader는 다음과 같은 선택적 매개변수를 받습니다:

* `dburl`: websocket endpoint에 대한 연결 문자열. 기본값: `ws://localhost:8000/rpc`
* `ns`: namespace의 이름. 기본값: `langchain`
* `db`: database의 이름. 기본값: `database`
* `table`: table의 이름. 기본값: `documents`
* `db_user`: 필요한 경우 SurrealDB 자격 증명: db username.
* `db_pass`: 필요한 경우 SurrealDB 자격 증명: db password.
* `filter_criteria`: table에서 결과를 필터링하기 위한 `WHERE` 절을 구성하는 dictionary.

출력 [`Document`](https://reference.langchain.com/python/langchain_core/documents/#langchain_core.documents.base.Document)는 다음과 같은 형태를 가집니다:

```
Document(
    page_content=<json encoded string containing the result document>,
    metadata={
        'id': <document id>,
        'ns': <namespace name>,
        'db': <database_name>,
        'table': <table name>,
        ... <additional fields from metadata property of the document>
    }
)
```

## 설정

surrealdb와 langchain을 설치하려면 아래 셀의 주석을 해제하세요.

```python
# pip install -qU  surrealdb langchain langchain-community
```

```python
# add this import for running in jupyter notebook
import nest_asyncio

nest_asyncio.apply()
```

```python
import json

from langchain_community.document_loaders.surrealdb import SurrealDBLoader
```

```python
loader = SurrealDBLoader(
    dburl="ws://localhost:8000/rpc",
    ns="langchain",
    db="database",
    table="documents",
    db_user="root",
    db_pass="root",
    filter_criteria={},
)
docs = loader.load()
len(docs)
```

```output
42
```

```python
doc = docs[-1]
doc.metadata
```

```output

 'ns': 'langchain',

 'table': 'documents'}
```
