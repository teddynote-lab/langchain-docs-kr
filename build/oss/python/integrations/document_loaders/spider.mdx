[Spider](https://spider.cloud/)는 LLM에 바로 사용 가능한 데이터를 반환하는 [가장 빠르고](https://github.com/spider-rs/spider/blob/main/benches/BENCHMARKS.md) 가장 저렴한 크롤러이자 스크래퍼입니다.

## Setup

```python
pip install spider-client
```

## Usage

Spider를 사용하려면 [spider.cloud](https://spider.cloud/)에서 API key가 필요합니다.

```python
from langchain_community.document_loaders import SpiderLoader

loader = SpiderLoader(
    api_key="YOUR_API_KEY",
    url="https://spider.cloud",
    mode="scrape",  # if no API key is provided it looks for SPIDER_API_KEY in env
)

data = loader.load()
print(data)
```

## Modes

- `scrape`: 단일 URL을 스크래핑하는 기본 모드
- `crawl`: 제공된 도메인 URL의 모든 하위 페이지를 크롤링

## Crawler options

`params` parameter는 loader에 전달할 수 있는 dictionary입니다. 사용 가능한 모든 parameter를 확인하려면 [Spider 문서](https://spider.cloud/docs/api)를 참조하세요
