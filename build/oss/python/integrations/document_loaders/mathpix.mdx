## Setup

### Credentials

Mathpix에 가입하고 [API key를 생성](https://mathpix.com/docs/ocr/creating-an-api-key)하여 환경 변수에 `MATHPIX_API_KEY`를 설정하세요

```python
import getpass
import os

if "MATHPIX_API_KEY" not in os.environ:
    os.environ["MATHPIX_API_KEY"] = getpass.getpass("Enter your Mathpix API key: ")
```

모델 호출의 자동 추적을 활성화하려면 [LangSmith](https://docs.smith.langchain.com/) API key를 설정하세요:

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### Installation

**langchain-community**를 설치합니다.

```python
pip install -qU langchain-community
```

## Initialization

이제 loader를 초기화할 준비가 되었습니다:

```python
from langchain_community.document_loaders import MathpixPDFLoader

file_path = "./example_data/layout-parser-paper.pdf"
loader = MathpixPDFLoader(file_path)
```

## Load

```python
docs = loader.load()
docs[0]
```

```python
print(docs[0].metadata)
```

## Lazy Load

```python
page = []
for doc in loader.lazy_load():
    page.append(doc)
    if len(page) >= 10:
        # do some paged operation, e.g.
        # index.upsert(page)

        page = []
```
