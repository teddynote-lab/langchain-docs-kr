title: Hugging Face의 BGE

>[HuggingFace의 BGE 모델](https://huggingface.co/BAAI/bge-large-en-v1.5)은 [최고의 오픈소스 embedding 모델](https://huggingface.co/spaces/mteb/leaderboard) 중 하나입니다.
>BGE 모델은 [Beijing Academy of Artificial Intelligence (BAAI)](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence)에서 만들었습니다. `BAAI`는 AI 연구 및 개발에 종사하는 민간 비영리 조직입니다.

이 노트북은 `Hugging Face`를 통해 `BGE Embeddings`를 사용하는 방법을 보여줍니다

```python
pip install -qU  sentence_transformers
```

```python
from langchain_community.embeddings import HuggingFaceBgeEmbeddings

model_name = "BAAI/bge-small-en"
model_kwargs = {"device": "cpu"}
encode_kwargs = {"normalize_embeddings": True}
hf = HuggingFaceBgeEmbeddings(
    model_name=model_name, model_kwargs=model_kwargs, encode_kwargs=encode_kwargs
)
```

`model_name="BAAI/bge-m3"`의 경우 `query_instruction=""`를 전달해야 한다는 점에 유의하세요. [FAQ BGE M3](https://huggingface.co/BAAI/bge-m3#faq)를 참조하세요.

```python
embedding = hf.embed_query("hi this is harrison")
len(embedding)
```

```output
384
```
