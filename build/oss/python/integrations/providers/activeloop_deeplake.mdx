>[Activeloop Deep Lake](https://docs.activeloop.ai/)는 Deep Learning 애플리케이션을 위한 data lake로, vector store로 사용할 수 있습니다.

## 왜 Deep Lake인가?

- 단순한 (multi-modal) vector store 이상입니다. 나중에 dataset을 사용하여 자체 LLM model을 fine-tune할 수 있습니다.
- embedding뿐만 아니라 자동 버전 관리와 함께 원본 데이터도 저장합니다.
- 진정한 serverless입니다. 별도의 서비스가 필요하지 않으며 주요 클라우드 제공업체(`AWS S3`, `GCS` 등)와 함께 사용할 수 있습니다.
