>[Ascend](https://https://www.hiascend.com/)는 Huawei에서 제공하는 Natural Process Unit입니다

이 페이지에서는 LangChain에서 ascend NPU를 사용하는 방법을 다룹니다.

### 설치

torch-npu를 사용하여 설치하세요:

<CodeGroup>
```bash pip
pip install torch-npu
```

```bash uv
uv add torch-npu
```
</CodeGroup>

아래 명시된 대로 설치 지침을 따르세요:
* [여기](https://www.hiascend.com/document/detail/zh/canncommercial/700/quickstart/quickstart/quickstart_18_0002.html)에 표시된 대로 CANN을 설치하세요.

### Embedding Models

[사용 예제](/oss/python/integrations/text_embedding/ascend)를 참조하세요.

```python
from langchain_community.embeddings import AscendEmbeddings
```
