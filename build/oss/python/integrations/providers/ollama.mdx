[Ollama](https://ollama.com/)와의 모든 LangChain 통합입니다.

Ollama를 사용하면 오픈 소스 모델(예: [`gpt-oss`](https://ollama.com/library/gpt-oss))을 로컬에서 실행할 수 있습니다.

지원되는 모델 및 변형의 전체 목록은 [Ollama 모델 라이브러리](https://ollama.ai/library)를 참조하세요.
