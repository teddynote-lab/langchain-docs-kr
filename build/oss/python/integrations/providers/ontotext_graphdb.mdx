title: Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/)는 RDF 및 SPARQL을 준수하는 그래프 데이터베이스이자 지식 발견 도구입니다.

## Dependencies

다음 명령어로 [rdflib](https://github.com/RDFLib/rdflib) 패키지를 설치하세요

<CodeGroup>
```bash pip
pip install rdflib==7.0.0
```

```bash uv
uv add "rdflib==7.0.0"
```
</CodeGroup>

## Graph QA Chain

GraphDB Database를 채팅 모델과 연결하여 데이터에 대한 인사이트를 얻으세요.

노트북 예제는 [여기](/oss/python/integrations/graphs/ontotext)에서 확인하세요.

```python
from langchain_community.graphs import OntotextGraphDBGraph
from langchain.chains import OntotextGraphDBQAChain
```
