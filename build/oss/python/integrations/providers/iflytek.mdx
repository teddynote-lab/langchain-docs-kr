>[iFlytek](https://www.iflytek.com)은 1999년에 설립된 중국의 정보 기술 회사입니다.
> 음성 인식 소프트웨어와 교육, 통신, 음악, 스마트 장난감 산업을 아우르는
> 음성 기반 인터넷/모바일 제품을 개발합니다.


## 설치 및 설정

- [iFlyTek SparkLLM API Console](https://console.xfyun.cn/services/bm3)에서 `SparkLLM` app_id, api_key, api_secret을 발급받으세요 (자세한 내용은 [iFlyTek SparkLLM Intro](https://xinghuo.xfyun.cn/sparkapi) 참조).
- Python 패키지를 설치하세요 (embedding model에는 필요하지 않음):

<CodeGroup>
```bash pip
pip install websocket-client
```

```bash uv
uv add websocket-client
```
</CodeGroup>
