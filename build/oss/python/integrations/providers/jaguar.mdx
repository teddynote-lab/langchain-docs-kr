이 페이지는 LangChain 내에서 Jaguar vector database를 사용하는 방법을 설명합니다.
세 개의 섹션으로 구성되어 있습니다: 소개, 설치 및 설정, 그리고 Jaguar API.


## 소개

Jaguar vector database는 다음과 같은 특징을 가지고 있습니다:

1. 분산 vector database입니다
2. JaguarDB의 "ZeroMove" 기능은 즉각적인 수평 확장성을 제공합니다
3. 멀티모달: embeddings, 텍스트, 이미지, 비디오, PDF, 오디오, 시계열 데이터, 지리공간 데이터 지원
4. All-masters: 병렬 읽기 및 쓰기를 모두 허용합니다
5. 이상 탐지 기능
6. RAG 지원: LLM을 독점 데이터 및 실시간 데이터와 결합합니다
7. 공유 metadata: 여러 vector index 간 metadata 공유
8. 거리 메트릭: Euclidean, Cosine, InnerProduct, Manhatten, Chebyshev, Hamming, Jeccard, Minkowski

[Jaguar 확장 가능한 vector database 개요](http://www.jaguardb.com)

JaguarDB는 docker container에서 실행하거나, 소프트웨어를 다운로드하여 클라우드 또는 온프레미스에서 실행할 수 있습니다.

## 설치 및 설정

- 하나 또는 여러 호스트에 JaguarDB 설치
- 하나의 호스트에 Jaguar HTTP Gateway server 설치
- JaguarDB HTTP Client package 설치

자세한 단계는 [Jaguar Documents](http://www.jaguardb.com/support.html)에 설명되어 있습니다

클라이언트 프로그램의 환경 변수:

        export OPENAI_API_KEY="......"
        export JAGUAR_API_KEY="......"


## Jaguar API

LangChain과 함께 Jaguar client class를 Python에서 import하여 사용할 수 있습니다:

```python
from langchain_community.vectorstores.jaguar import Jaguar
```

Jaguar class에서 지원하는 API 함수는 다음과 같습니다:

- `add_texts`
- `add_documents`
- `from_texts`
- `from_documents`
- `similarity_search`
- `is_anomalous`
- `create`
- `delete`
- `clear`
- `drop`
- `login`
- `logout`
