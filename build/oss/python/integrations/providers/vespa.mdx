>[Vespa](https://vespa.ai/)는 완전한 기능을 갖춘 검색 엔진이자 vector database입니다.
> 동일한 쿼리에서 vector search (ANN), lexical search, 그리고 구조화된 데이터 검색을 모두 지원합니다.

## Installation and Setup


<CodeGroup>
```bash pip
pip install pyvespa
```

```bash uv
uv add pyvespa
```
</CodeGroup>
