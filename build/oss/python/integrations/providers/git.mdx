>[Git](https://en.wikipedia.org/wiki/Git)은 컴퓨터 파일의 변경 사항을 추적하는 분산 버전 관리 시스템으로, 일반적으로 소프트웨어 개발 중 프로그래머들이 소스 코드를 공동으로 개발할 때 작업을 조율하는 데 사용됩니다.

## Installation and Setup

먼저 `GitPython` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install GitPython
```

```bash uv
uv add GitPython
```
</CodeGroup>
