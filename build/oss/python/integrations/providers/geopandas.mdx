>[GeoPandas](https://geopandas.org/)는 Python에서 지리공간 데이터 작업을 더 쉽게 만들기 위한 오픈 소스 프로젝트입니다. `GeoPandas`는 `pandas`에서 사용하는 데이터 타입을 확장하여 기하학적 타입에 대한 공간 연산을 가능하게 합니다.
> 기하학적 연산은 `shapely`에 의해 수행됩니다.


## Installation and Setup

여러 Python 패키지를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install -U sodapy pandas geopandas
```

```bash uv
uv add sodapy pandas geopandas
```
</CodeGroup>
