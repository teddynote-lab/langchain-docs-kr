> [Meilisearch](https://meilisearch.com)는 오픈소스이며, 매우 빠르고, 높은 관련성을 제공하는 검색 엔진입니다.
> 개발자가 빠른 검색 경험을 구축할 수 있도록 훌륭한 기본 설정을 제공합니다.
>
> [Meilisearch를 자체 호스팅](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation)하거나 [Meilisearch Cloud](https://www.meilisearch.com/pricing)에서 실행할 수 있습니다.
>
>`Meilisearch v1.3`은 vector search를 지원합니다.

## Installation and Setup

자세한 설정 지침은 [사용 예제](/oss/python/integrations/vectorstores/meilisearch)를 참조하세요.


`meilisearch` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install meilisearch
```

```bash uv
uv add meilisearch
```
</CodeGroup>

## Vector Store

[사용 예제](/oss/python/integrations/vectorstores/meilisearch)를 참조하세요.

```python
from langchain_community.vectorstores import Meilisearch
```
