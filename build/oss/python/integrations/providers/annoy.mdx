> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
> 는 주어진 쿼리 포인트에 가까운 공간의 포인트를 검색하기 위한 Python 바인딩이 포함된 C++ 라이브러리입니다.
> 또한 메모리에 매핑되는 대용량 읽기 전용 파일 기반 데이터 구조를 생성하여
> 여러 프로세스가 동일한 데이터를 공유할 수 있습니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install annoy
```

```bash uv
uv add annoy
```
</CodeGroup>
