---
title: ScraperAPI
---

[ScraperAPI](https://www.scraperapi.com/)는 프록시, 브라우저 또는 CAPTCHA 처리에 대한 걱정 없이 웹 스크래핑 API를 통해 모든 공개 웹사이트에서 데이터를 수집할 수 있게 해줍니다. [langchain-scraperapi](https://github.com/scraperapi/langchain-scraperapi)는 이 서비스를 래핑하여 AI 에이전트가 웹을 탐색하고 데이터를 스크래핑하는 것을 쉽게 만들어줍니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-scraperapi
```

```bash uv
uv add langchain-scraperapi
```
</CodeGroup>

[ScraperAPI](https://www.scraperapi.com/)에서 API key를 받아 환경 변수로 설정하세요:

```python
import os

os.environ["SCRAPERAPI_API_KEY"] = "your-api-key"
```

## Tools

이 패키지는 모든 웹사이트를 스크래핑하고, 구조화된 Google 검색 결과를 가져오고, 구조화된 Amazon 검색 결과를 각각 가져오는 3가지 도구를 제공합니다.

각 도구에 대한 자세한 문서를 참조하세요:
- [ScraperAPITool](/oss/python/integrations/tools/scraperapi) - 모든 웹사이트 탐색 및 스크래핑
- ScraperAPIGoogleSearchTool - 구조화된 Google Search SERP 데이터 가져오기
- ScraperAPIAmazonSearchTool - 구조화된 Amazon 제품 검색 데이터 가져오기

더 자세한 안내는 [공식 저장소](https://github.com/scraperapi/langchain-scraperapi)를 참조하세요.
