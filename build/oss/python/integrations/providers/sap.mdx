>[SAP SE(Wikipedia)](https://www.sap.com/about/company.html)는 독일의 다국적
> 소프트웨어 회사입니다. 비즈니스 운영 및 고객 관계를 관리하기 위한 엔터프라이즈 소프트웨어를 개발합니다.
> 이 회사는 세계 최고의 `enterprise resource planning (ERP)` 소프트웨어 공급업체입니다.

## Installation and Setup

`langchain-hana` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install langchain-hana
```

```bash uv
uv add langchain-hana
```
</CodeGroup>
