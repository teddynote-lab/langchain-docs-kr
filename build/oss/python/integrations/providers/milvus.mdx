>[Milvus](https://milvus.io/docs/overview.md)는 딥 뉴럴 네트워크 및 기타 머신 러닝(ML) 모델에서 생성된
> 대규모 embedding vector를 저장, 인덱싱 및 관리하는 데이터베이스입니다.


## Installation and Setup

Python SDK 설치:

<CodeGroup>
```bash pip
pip install langchain-milvus
```

```bash uv
uv add langchain-milvus
```
</CodeGroup>

## Vector Store

[사용 예제](/oss/python/integrations/vectorstores/milvus)를 참조하세요.

이 vectorstore를 import하려면:
```python
from langchain_milvus import Milvus
```
