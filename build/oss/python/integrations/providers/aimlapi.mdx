>[AI/ML API](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration)는 엔터프라이즈급 가동 시간과 처리량으로 300개 이상의 호스팅된 foundation model(DeepSeek, Gemini, GPT 등)에 액세스할 수 있는 단일 API를 제공합니다.

## Installation 및 setup

- AI/ML API integration package를 설치합니다.

  ```bash
  pip install langchain-aimlapi
  ```

- [aimlapi.com](https://aimlapi.com/app/?utm_source=langchain&utm_medium=github&utm_campaign=integration)에서 계정을 생성하고 API key를 생성합니다.
- `AIMLAPI_API_KEY` environment variable을 설정하여 인증합니다.

```python
import os

os.environ["AIMLAPI_API_KEY"] = "aimlapi_..."
```
