>[PubMed®](https://pubmed.ncbi.nlm.nih.gov/)는 `The National Center for Biotechnology Information, National Library of Medicine`에서 제공하며,
> `MEDLINE`, 생명과학 저널 및 온라인 도서의 생물의학 문헌에 대한 3,500만 개 이상의 인용을 포함합니다.
> 인용에는 `PubMed Central` 및 출판사 웹사이트의 전문 콘텐츠 링크가 포함될 수 있습니다.

## Setup
python 패키지를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install xmltodict
```

```bash uv
uv add xmltodict
```
</CodeGroup>
