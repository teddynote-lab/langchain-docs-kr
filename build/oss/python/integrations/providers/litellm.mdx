title: LiteLLM

>[LiteLLM](https://github.com/BerriAI/litellm)은 Anthropic, Azure, Huggingface, Replicate 등을 호출하는 것을 간소화하는 라이브러리입니다.

## Installation and setup

<CodeGroup>
```bash pip
pip install langchain-litellm
```

```bash uv
uv add langchain-litellm
```
</CodeGroup>

## Chat Models
```python
from langchain_litellm import ChatLiteLLM
```
```python
from langchain_litellm import ChatLiteLLMRouter
```
자세한 내용은 [여기](/oss/python/integrations/chat/litellm) 가이드를 참조하세요.

## API reference
모든 `ChatLiteLLM` 및 `ChatLiteLLMRouter` 기능과 구성에 대한 자세한 문서는 API reference를 참조하세요: https://github.com/Akshay-Dongare/langchain-litellm
