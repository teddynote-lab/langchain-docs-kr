>[Breebs](https://www.breebs.com/)는 개방형 협업 지식 플랫폼입니다.
>누구나 Google Drive 폴더에 저장된 PDF를 기반으로 지식 캡슐인 `Breeb`을 만들 수 있습니다.
>`Breeb`은 모든 LLM/chatbot이 전문성을 향상시키고, 환각(hallucination)을 줄이며, 출처에 대한 접근을 제공하는 데 사용할 수 있습니다.
>내부적으로 `Breebs`는 여러 `Retrieval Augmented Generation (RAG)` 모델을 구현하여
> 각 반복마다 유용한 컨텍스트를 원활하게 제공합니다.

```python
from langchain.retrievers import BreebsRetriever
```
