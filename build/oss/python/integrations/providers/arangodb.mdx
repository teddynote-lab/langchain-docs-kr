>[ArangoDB](https://github.com/arangodb/arangodb)는 연결된 데이터에서 더 빠르게 가치를 창출하는 확장 가능한 graph database 시스템입니다. 단일 query language를 통해 native graph, 통합 검색 엔진, JSON 지원을 제공합니다. ArangoDB는 온프레미스, 클라우드 등 어디서나 실행됩니다.

## Installation and Setup

[ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) package를 다음과 같이 설치하세요

<CodeGroup>
```bash pip
pip install python-arango
```

```bash uv
uv add python-arango
```
</CodeGroup>

## Graph QA Chain

`ArangoDB` Database를 chat model과 연결하여 데이터에 대한 인사이트를 얻으세요.

노트북 예제는 [여기](/oss/python/integrations/graphs/arangodb)를 참조하세요.

```python
from arango import ArangoClient

from langchain_community.graphs import ArangoGraph
from langchain.chains import ArangoGraphQAChain
```
