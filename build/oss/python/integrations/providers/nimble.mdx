[Nimble](https://www.linkedin.com/company/nimbledata)은 최초의 비즈니스 외부 데이터 플랫폼으로, 수상 경력에 빛나는 AI 기반 데이터 구조화 기술을 통해 데이터 의사결정을 그 어느 때보다 쉽게 만들어줍니다. Nimble은 비즈니스 사용자를 공개 웹 지식과 연결합니다.

우리는 미션 크리티컬한 실시간 외부 데이터를 통해 비즈니스가 고급 비즈니스 인텔리전스, 가격 비교 및 영업과 마케팅을 위한 기타 공개 데이터를 활용할 수 있도록 지원합니다. 우리는 데이터를 즉각적인 비즈니스 가치로 전환합니다.

Nimble에 대해 더 자세히 알고 싶으시다면 [nimbleway.com](https://www.nimbleway.com/)을 방문해주세요.

## Retrievers

### NimbleSearchRetriever

개발자가 웹 어디에서나 온라인 정보를 검색하고, 액세스하고, 검색할 수 있는 RAG 애플리케이션과 AI Agent를 구축할 수 있도록 지원합니다.

`langchain-nimble` python package를 설치해야 합니다.

```python
pip install -U langchain-nimble
```

[사용 예제](/oss/python/integrations/retrievers/nimble/)를 참조하세요.

```python
from langchain_nimble import NimbeSearchRetriever
```

인증이 필요하므로 [문서의 Setup 섹션](/oss/python/integrations/retrievers/nimble/#setup)을 참조하시기 바랍니다.
