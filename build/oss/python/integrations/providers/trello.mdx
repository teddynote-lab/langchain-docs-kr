>[Trello](https://www.atlassian.com/software/trello)는 개인과 팀이 작업과 프로젝트를 구성하고 추적할 수 있도록 하는 웹 기반 프로젝트 관리 및 협업 도구입니다. 사용자가 작업과 활동을 나타내는 list와 card를 생성할 수 있는 "board"라는 시각적 인터페이스를 제공합니다.
>TrelloLoader를 사용하면 `Trello` board에서 card를 로드할 수 있습니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install py-trello beautifulsoup4
```

```bash uv
uv add py-trello beautifulsoup4
```
</CodeGroup>
