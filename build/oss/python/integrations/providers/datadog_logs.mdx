>[Datadog](https://www.datadoghq.com/)는 클라우드 규모 애플리케이션을 위한 모니터링 및 분석 플랫폼입니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install datadog_api_client
```

```bash uv
uv add datadog_api_client
```
</CodeGroup>

Datadog API key와 APP key로 loader를 초기화해야 하며, 원하는 로그를 추출하기 위한 쿼리를 설정해야 합니다.
