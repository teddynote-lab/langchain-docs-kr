>[01.AI](https://www.lingyiwanwu.com/en)는 Dr. Kai-Fu Lee가 설립한 AI 2.0의 최전선에 있는 글로벌 기업입니다. 6B부터 수천억 개의 parameter에 이르는 Yi 시리즈를 포함한 최첨단 large language model을 제공합니다. 01.AI는 또한 multimodal model, 개방형 API platform, 그리고 Yi-34B/9B/6B 및 Yi-VL과 같은 오픈소스 옵션을 제공합니다.

## Installation and Setup

중국 사이트 [여기](https://platform.lingyiwanwu.com/apikeys) 또는 글로벌 사이트 [여기](https://platform.01.ai/apikeys)에서 등록하고 API key를 받으세요.

## LLMs

[사용 예제](/oss/python/integrations/llms/yi)를 참조하세요.

```python
from langchain_community.llms import YiLLM
```

## Chat models

[사용 예제](/oss/python/integrations/chat/yi)를 참조하세요.

```python
from langchain_community.chat_models import ChatYi
```
