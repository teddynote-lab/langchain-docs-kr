title: Gradient

>[Gradient](https://gradient.ai/)는 간단한 web API를 통해 LLM을 fine tune하고 completion을 얻을 수 있게 해줍니다.

## Installation and Setup
- Python SDK 설치:
<CodeGroup>
```bash pip
pip install gradientai
```

```bash uv
uv add gradientai
```
</CodeGroup>
[Gradient access token과 workspace](https://gradient.ai/)를 받아서 환경 변수로 설정하세요 (`Gradient_ACCESS_TOKEN`) 및 (`GRADIENT_WORKSPACE_ID`)

## LLM

Gradient LLM wrapper가 존재하며, 다음과 같이 접근할 수 있습니다
[사용 예제](/oss/python/integrations/llms/gradient)를 참조하세요.

```python
from langchain_community.llms import GradientLLM
```

## Text Embedding Model

Gradient Embedding model이 존재하며, 다음과 같이 접근할 수 있습니다
```python
from langchain_community.embeddings import GradientEmbeddings
```
더 자세한 안내는 [이 notebook](/oss/python/integrations/text_embedding/gradient)을 참조하세요
