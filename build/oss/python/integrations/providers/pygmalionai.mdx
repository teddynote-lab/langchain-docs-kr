>[PygmalionAI](https://pygmalion.chat/)는 
> [Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine)을 위한 추론 엔드포인트를 제공하여
> 오픈소스 모델을 지원하는 회사입니다.


## Installation and Setup


<CodeGroup>
```bash pip
pip install aphrodite-engine
```

```bash uv
uv add aphrodite-engine
```
</CodeGroup>
