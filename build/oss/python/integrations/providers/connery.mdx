>[Connery SDK](https://github.com/connery-io/connery-sdk)는 플러그인과 액션 개발을 위해 설계된
> SDK와 CLI를 모두 포함하는 NPM 패키지입니다.
>
>CLI는 개발 프로세스에서 많은 작업을 자동화합니다. SDK는
> 플러그인과 액션을 정의하고 메타데이터에서 생성된 표준화된 REST API를 갖춘
> 플러그인 서버로 패키징하기 위한 JavaScript API를 제공합니다.
> 플러그인 서버는 인증, 입력 검증 및 로깅을 처리합니다.
> 따라서 액션의 로직에만 집중할 수 있습니다.
>
> [Connery SDK 문서](https://sdk.connery.io/docs/use-cases/)에서 사용 사례와 예제를 확인하세요
