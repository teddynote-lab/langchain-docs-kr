>[AwaDB](https://github.com/awa-ai/awadb)는 LLM Application에서 사용되는 embedding vector의 검색 및 저장을 위한 AI Native database입니다.

## Installation and Setup

```bash
pip install awadb
```

## Vector store

```python
from langchain_community.vectorstores import AwaDB
```

[사용 예시](/oss/python/integrations/vectorstores/awadb)를 참조하세요.

## Embedding models

```python
from langchain_community.embeddings import AwaEmbeddings
```

[사용 예시](/oss/python/integrations/text_embedding/awadb)를 참조하세요.
