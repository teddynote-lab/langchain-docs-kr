[Netmind AI](https://www.netmind.ai/)는 더 빠르고, 더 스마트하며, 더 저렴하게 AI를 구축할 수 있도록 지원합니다.
글로벌 GPU 네트워크를 통해 학습, 미세 조정, 추론 실행 및 확장을 수행하세요—올인원 AI 엔진입니다.

이 예제는 LangChain을 사용하여 Netmind AI 모델과 상호작용하는 방법을 다룹니다.

## 설치 및 설정

```bash
pip install langchain-netmind
```

Netmind API 키를 발급받아 환경 변수(`NETMIND_API_KEY`)로 설정하세요.
[www.netmind.ai/](https://www.netmind.ai/)로 이동하여 Netmind에 가입하고 API 키를 생성하세요.
