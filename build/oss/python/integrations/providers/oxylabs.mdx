[Oxylabs](https://oxylabs.io/)는 전 세계 기업들이 데이터 기반 인사이트를 활용할 수 있도록 지원하는 시장 선도적인 웹 인텔리전스 수집 플랫폼으로, 최고 수준의 비즈니스, 윤리 및 규정 준수 표준을 기반으로 운영됩니다.

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/)는
LLM이 Oxylabs Web Scraper API와 상호작용할 수 있도록 하는 도구를 구현합니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-oxylabs
```

```bash uv
uv add langchain-oxylabs
```
</CodeGroup>
