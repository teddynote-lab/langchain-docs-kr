> [Xata](https://xata.io)는 `PostgreSQL` 기반의 serverless 데이터 플랫폼입니다.
> 데이터베이스와 상호작용하기 위한 Python SDK와 데이터 관리를 위한 UI를 제공합니다.
> `Xata`는 네이티브 vector 타입을 가지고 있으며, 이는 모든 테이블에 추가할 수 있고
> 유사도 검색을 지원합니다. LangChain은 vector를 `Xata`에 직접 삽입하고,
> 주어진 vector의 최근접 이웃을 쿼리하므로, 모든 LangChain Embeddings 통합을
> `Xata`와 함께 사용할 수 있습니다.


## Installation and Setup


`xata` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install xata==1.0.0a7
```

```bash uv
uv add "xata==1.0.0a7"
```
</CodeGroup>

## Vector Store

[사용 예제](/oss/python/integrations/vectorstores/xata)를 참조하세요.

```python
from langchain_community.vectorstores import XataVectorStore
```
