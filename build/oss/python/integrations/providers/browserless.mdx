>[Browserless](https://www.browserless.io/docs/start)는 클라우드에서 headless Chrome 인스턴스를 실행할 수 있는 서비스입니다. 자체 인프라를 관리할 필요 없이 브라우저 기반 자동화를 대규모로 실행할 수 있는 훌륭한 방법입니다.

## Installation and Setup

API key는 [여기](https://www.browserless.io/pricing/)에서 받을 수 있습니다.
