title: Featherless AI
---

[Featherless AI](https://featherless.ai/)는 4300개 이상의 오픈소스 모델에 대한 액세스를 제공하는 서버리스 AI 추론 플랫폼입니다. 우리의 목표는 모든 AI 모델을 서버리스 추론에 사용할 수 있도록 만드는 것입니다. 지속적으로 확장되는 오픈 웨이트 모델 라이브러리에 대한 API를 통한 추론을 제공합니다.

# Installation and Setup
`pip install langchain-featherless-ai`
1. [Featherless](https://featherless.ai/register)에서 계정을 등록하세요
2. 플랜을 구독하고 [API Keys](https://featherless.ai/account/api-keys)에서 API key를 받으세요
3. API key를 환경 변수(`FEATHERLESSAI_API_KEY`)로 설정하세요

# Model catalog
모든 모델의 개요를 보려면 모델 카탈로그를 방문하세요: https://featherless.ai/models
