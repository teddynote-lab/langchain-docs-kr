title: Scrapeless
---

[Scrapeless](https://scrapeless.com)는 광범위한 매개변수 사용자 정의 및 다중 형식 내보내기 지원을 통해 유연하고 기능이 풍부한 데이터 수집 서비스를 제공합니다.

## 설치 및 설정

<CodeGroup>
```bash pip
pip install langchain-scrapeless
```

```bash uv
uv add langchain-scrapeless
```
</CodeGroup>

Scrapeless API key를 설정해야 합니다:

```python
import os
os.environ["SCRAPELESS_API_KEY"] = "your-api-key"
```

## Tools

Scrapeless integration은 여러 tool을 제공합니다:

- [ScrapelessDeepSerpGoogleSearchTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 모든 결과 유형에 걸쳐 Google SERP 데이터의 포괄적인 추출을 가능하게 합니다.
- [ScrapelessDeepSerpGoogleTrendsTool](/oss/python/integrations/tools/scrapeless_scraping_api) - 시간 경과에 따른 인기도, 지역별 관심도 및 관련 검색을 포함하여 Google에서 키워드 트렌드 데이터를 검색합니다.
- [ScrapelessUniversalScrapingTool](/oss/python/integrations/tools/scrapeless_universal_scraping) - 일반적으로 봇을 차단하는 JS-Render 웹사이트에서 데이터에 액세스하고 추출합니다.
- [ScrapelessCrawlerCrawlTool](/oss/python/integrations/tools/scrapeless_crawl) - 웹사이트와 연결된 페이지를 크롤링하여 포괄적인 데이터를 추출합니다.
- [ScrapelessCrawlerScrapeTool](/oss/python/integrations/tools/scrapeless_crawl) - 단일 웹페이지에서 정보를 추출합니다.
