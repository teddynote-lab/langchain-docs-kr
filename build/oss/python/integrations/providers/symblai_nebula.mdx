이 페이지는 LangChain 내에서 [Symbl.ai](https://symbl.ai/)의 LLM인 [Nebula](https://symbl.ai/nebula) 생태계를 사용하는 방법을 다룹니다.
설치 및 설정, 그리고 특정 Nebula wrapper에 대한 참조, 이렇게 두 부분으로 나뉩니다.

## 설치 및 설정

- [Nebula API Key](https://info.symbl.ai/Nebula_Private_Beta.html)를 발급받고 환경 변수 `NEBULA_API_KEY`로 설정하세요
- 자세한 내용은 [Nebula 문서](https://docs.symbl.ai/docs/nebula-llm)를 참조하세요.

### LLM

Nebula LLM wrapper가 존재하며, 다음과 같이 접근할 수 있습니다
```python
from langchain_community.llms import Nebula
llm = Nebula()
```
