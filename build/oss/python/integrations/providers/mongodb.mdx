>[MongoDB](https://www.mongodb.com/)는 동적 스키마를 가진 JSON과 유사한 문서를 지원하는 NoSQL, 문서 지향 데이터베이스입니다.

**참고:**
- [MongoDB Atlas 페이지](/oss/python/integrations/providers/mongodb_atlas)에서 다른 `MongoDB` 통합을 확인하세요.

## Installation and Setup

Python package를 설치합니다:

<CodeGroup>
```bash pip
pip install langchain-mongodb
```

```bash uv
uv add langchain-mongodb
```
</CodeGroup>
