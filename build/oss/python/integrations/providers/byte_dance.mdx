## Document loaders

>[Lark Suite](https://www.larksuite.com/)는 `ByteDance`에서 개발한 기업용 협업 플랫폼입니다.

### Lark Suite for Document

[사용 예시](/oss/python/integrations/document_loaders/larksuite/#load-from-document)를 참조하세요.

```python
from langchain_community.document_loaders.larksuite import LarkSuiteDocLoader
```

### Lark Suite for Wiki

[사용 예시](/oss/python/integrations/document_loaders/larksuite/#load-from-wiki)를 참조하세요.

```python
from langchain_community.document_loaders.larksuite import LarkSuiteWikiLoader
```
