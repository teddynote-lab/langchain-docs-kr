>[Fauna](https://fauna.com/)는 지역, 클라우드 또는 전 세계에 걸쳐 확장 가능한 관계형 ACID 호환 데이터베이스의 강력함과 문서의 유연성을 결합한 분산 문서-관계형 데이터베이스입니다.


## Installation and Setup

secret key를 가져와야 합니다.
자세한 내용은 [가이드](https://docs.fauna.com/fauna/current/learn/security_model/)를 참조하세요.

`fauna` package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install -U fauna
```

```bash uv
uv add fauna
```
</CodeGroup>
