>[ElevenLabs](https://elevenlabs.io/about)는 모든 언어와 음성으로 콘텐츠를 보편적으로 접근 가능하게 만드는 것을 목표로 하는 음성 AI 연구 및 배포 회사입니다.
>
>`ElevenLabs`는 가장 현실적이고 다재다능하며 맥락을 인식하는 AI 오디오를 생성하여, 29개 언어로 수백 개의 신규 및 기존 음성으로 음성을 생성할 수 있는 기능을 제공합니다.

## Installation and Setup

먼저 ElevenLabs 계정을 설정해야 합니다. [여기 안내](https://docs.elevenlabs.io/welcome/introduction)를 따라 진행할 수 있습니다.

Python package를 설치하세요:

<CodeGroup>
```bash pip
pip install elevenlabs
```

```bash uv
uv add elevenlabs
```
</CodeGroup>
