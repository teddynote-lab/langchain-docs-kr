>[Perplexity](https://www.perplexity.ai/pro)는 무제한 Pro Search, 업그레이드된 AI 모델, 무제한 파일 업로드,
> 이미지 생성 및 API 크레딧을 제공하는 가장 강력한 인터넷 검색 방법입니다.
>
> [사용 가능한 모델 목록](https://docs.perplexity.ai/docs/model-cards)을 확인할 수 있습니다.

## Installation and Setup

Perplexity x LangChain integration package를 설치하세요:

<CodeGroup>
```bash pip
pip install langchain-perplexity
```

```bash uv
uv add langchain-perplexity
```
</CodeGroup>

[여기](https://docs.perplexity.ai/docs/getting-started)에서 API key를 받으세요.

## Chat models

다양한 사용 예제는 [여기](/oss/python/integrations/chat/perplexity)에서 확인하세요.

```python
from langchain_perplexity import ChatPerplexity
```
