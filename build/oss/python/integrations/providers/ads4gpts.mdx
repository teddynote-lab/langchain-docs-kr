> [ADS4GPTs](https://www.ads4gpts.com/)는 AI-Native 인터넷의 오픈 수익화 백본을 구축하고 있습니다. UX와 프라이버시를 우선으로 하는 접근 방식으로 AI 애플리케이션이 광고를 통해 수익을 창출할 수 있도록 지원합니다.

## Installation and Setup

### pip 사용하기
PyPI에서 직접 패키지를 설치할 수 있습니다:

<CodeGroup>
```bash pip
pip install ads4gpts-langchain
```

```bash uv
uv add ads4gpts-langchain
```
</CodeGroup>

### 소스에서 설치
또는 소스에서 설치할 수 있습니다:

<CodeGroup>
```bash pip
git clone https://github.com/ADS4GPTs/ads4gpts.git
cd ads4gpts/libs/python-sdk/ads4gpts-langchain
pip install .
```

```bash uv
git clone https://github.com/ADS4GPTs/ads4gpts.git
cd ads4gpts/libs/python-sdk/ads4gpts-langchain
uv add .
```
</CodeGroup>

## Prerequisites

- Python 3.11+
- ADS4GPTs API Key ([API Key 받기](https://www.ads4gpts.com))

## Environment Variables
API 인증을 위해 다음 환경 변수를 설정하세요:

```bash
export ADS4GPTS_API_KEY='your-ads4gpts-api-key'
```

또는 클래스를 초기화할 때 API key를 직접 전달하거나 `.env` 파일에 저장할 수 있습니다.

## Tools

ADS4GPTs는 수익화를 위한 두 가지 주요 도구를 제공합니다:

### Ads4gptsInlineSponsoredResponseTool
이 도구는 AI 애플리케이션의 출력 내에 원활하게 통합될 수 있는 네이티브 스폰서 응답을 가져옵니다.

```python
from ads4gpts_langchain import Ads4gptsInlineSponsoredResponseTool
```

### Ads4gptsSuggestedPromptTool
사용자 참여를 향상시키고 수익화 기회를 제공하기 위해 스폰서 프롬프트 제안을 생성합니다.

```python
from ads4gpts_langchain import Ads4gptsSuggestedPromptTool
```
### Ads4gptsInlineConversationalTool
채팅 인터페이스와 대화 내에서 자연스럽게 어울리는 대화형 스폰서 콘텐츠를 제공합니다.

```python
from ads4gpts_langchain import Ads4gptsInlineConversationalTool
```

### Ads4gptsInlineBannerTool
AI 애플리케이션의 응답 내에 표시될 수 있는 인라인 배너 광고를 제공합니다.

```python
from ads4gpts_langchain import Ads4gptsInlineBannerTool
```

### Ads4gptsSuggestedBannerTool
사용자에게 추천 콘텐츠로 제시될 수 있는 배너 광고 제안을 생성합니다.

```python
from ads4gpts_langchain import Ads4gptsSuggestedBannerTool
```

## Toolkit

`Ads4gptsToolkit`은 LangChain 애플리케이션에서 편리하게 액세스할 수 있도록 이러한 도구들을 결합합니다.

```python
from ads4gpts_langchain import Ads4gptsToolkit
```
