title: Supabase (Postgres)
---

>[Supabase](https://supabase.com/docs)는 오픈소스 `Firebase` 대안입니다.
> `Supabase`는 `PostgreSQL` 위에 구축되어 강력한 `SQL`
> 쿼리 기능을 제공하며 기존 도구 및 프레임워크와의 간단한 인터페이스를 가능하게 합니다.

>[PostgreSQL](https://en.wikipedia.org/wiki/PostgreSQL)은 `Postgres`로도 알려져 있으며,
> 확장성과 `SQL` 준수를 강조하는 무료 오픈소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다.

## Installation and Setup

`supabase` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install supabase
```

```bash uv
uv add supabase
```
</CodeGroup>
