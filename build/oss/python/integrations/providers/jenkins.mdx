[Jenkins](https://www.jenkins.io/)는 소프트웨어 팀이 개발 워크플로우를 간소화할 수 있도록 지원하는 오픈 소스 자동화 플랫폼입니다. CI/CD 파이프라인을 통해 애플리케이션의 빌드, 테스트 및 배포를 자동화하는 도구로 DevOps 커뮤니티에서 널리 채택되고 있습니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-jenkins
```

```bash uv
uv add langchain-jenkins
```
</CodeGroup>
