>[Cube](https://cube.dev/)는 데이터 앱을 구축하기 위한 Semantic Layer입니다. 데이터 엔지니어와 애플리케이션 개발자가 최신 데이터 저장소에서 데이터에 액세스하고,
> 일관된 정의로 구성하며, 모든 애플리케이션에 제공할 수 있도록 지원합니다.

## Installation and Setup

Cube 인스턴스의 API key와 URL을 가져와야 합니다.
[이 지침](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path)을 참조하세요.
