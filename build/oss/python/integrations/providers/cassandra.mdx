> [Apache Cassandra®](https://cassandra.apache.org/)는 NoSQL, row-oriented, 높은 확장성과 가용성을 제공하는 데이터베이스입니다.
> 버전 5.0부터 [vector search 기능](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html)을 제공합니다.

이 페이지에 설명된 통합은 `Cassandra`뿐만 아니라 다른 CQL 호환 데이터베이스,
즉 `Cassandra Query Language` 프로토콜을 사용하는 데이터베이스에서도 사용할 수 있습니다.


## Installation and Setup

다음 Python package를 설치하세요:

<CodeGroup>
```bash pip
pip install "cassio>=0.1.6"
```

```bash uv
uv add "cassio>=0.1.6"
```
</CodeGroup>

from langchain_community.vectorstores import Cassandra

## LLM Cache

```python
from langchain.globals import set_llm_cache
from langchain_community.cache import CassandraCache
set_llm_cache(CassandraCache())
```

## Semantic LLM Cache

```python
from langchain.globals import set_llm_cache
from langchain_community.cache import CassandraSemanticCache
set_llm_cache(CassandraSemanticCache(
    embedding=my_embedding,
    table_name="my_store",
))
```

## Toolkit

`Cassandra Database toolkit`은 AI 엔지니어가 agent를
Cassandra 데이터와 효율적으로 통합할 수 있도록 지원합니다.

```python
from langchain_community.agent_toolkits.cassandra_database.toolkit import (
    CassandraDatabaseToolkit,
)
```

자세한 내용은 [예제 노트북](/oss/python/integrations/tools/cassandra_database)에서 확인하세요.


Cassandra Database 개별 tool:

### Get Schema

Apache Cassandra 데이터베이스에서 keyspace의 schema를 가져오는 tool입니다.

```python
from langchain_community.tools import GetSchemaCassandraDatabaseTool
```

### Get Table Data

Apache Cassandra 데이터베이스의 table에서 데이터를 가져오는 tool입니다.

```python
from langchain_community.tools import GetTableDataCassandraDatabaseTool
```

### Query

제공된 CQL로 Apache Cassandra 데이터베이스를 쿼리하는 tool입니다.

```python
from langchain_community.tools import QueryCassandraDatabaseTool
```
