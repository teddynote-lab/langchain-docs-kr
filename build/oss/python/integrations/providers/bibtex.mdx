>[BibTeX](https://www.ctan.org/pkg/bibtex)는 `LaTeX` 조판 시스템과 함께 일반적으로 사용되는 파일 형식이자 참고문헌 관리 시스템입니다. 학술 및 연구 문서의 서지 정보를 구성하고 저장하는 방법으로 사용됩니다.

## Installation and Setup

`bibtexparser`와 `pymupdf` 패키지를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install bibtexparser pymupdf
```

```bash uv
uv add bibtexparser pymupdf
```
</CodeGroup>

