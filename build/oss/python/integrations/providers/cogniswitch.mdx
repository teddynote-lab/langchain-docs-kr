>[CogniSwitch](https://www.cogniswitch.ai/aboutus)는 API 기반 데이터 플랫폼으로,
> 엔티티, 개념 및 그들의 관계를 추출하여 기업 데이터를 향상시키고
> 이 데이터를 다차원 형식으로 변환하여
> 이러한 향상된 정보를 수용할 수 있는 데이터베이스에 저장합니다. 우리의 경우 데이터는
> 지식 그래프에 저장됩니다. 이렇게 향상된 데이터는 이제 LLM 및
> 기타 GenAI 애플리케이션에서 사용할 준비가 되어 있으며, 데이터가 소비 가능하고 컨텍스트를 유지할 수 있도록 보장합니다.
> 이를 통해 환각(hallucination)을 제거하고 정확성을 제공합니다.

## Toolkit

[설치 지침 및 사용 예제](/oss/python/integrations/tools/cogniswitch)를 참조하세요.

## Tools

### CogniswitchKnowledgeRequest

>CogniSwitch 서비스를 사용하여 질문에 답변하는 도구입니다.

```python
from langchain_community.tools.cogniswitch.tool import CogniswitchKnowledgeRequest
```

### CogniswitchKnowledgeSourceFile

>CogniSwitch 서비스를 사용하여 파일에서 데이터를 저장하는 도구입니다.

```python
from langchain_community.tools.cogniswitch.tool import CogniswitchKnowledgeSourceFile
```

### CogniswitchKnowledgeSourceURL

>CogniSwitch 서비스를 사용하여 URL에서 데이터를 저장하는 도구입니다.

```python
from langchain_community.tools.cogniswitch.tool import CogniswitchKnowledgeSourceURL
```

### CogniswitchKnowledgeStatus

>CogniSwitch 서비스를 사용하여 업로드된 문서 또는 URL의 상태를 가져오는 도구입니다.

