>[Label Studio](https://labelstud.io/guide/get_started)는 대규모 언어 모델(LLM)을 미세 조정하기 위한 데이터 레이블링에 있어 LangChain에 유연성을 제공하는 오픈 소스 데이터 레이블링 플랫폼입니다. 또한 커스텀 학습 데이터 준비와 인간 피드백을 통한 응답 수집 및 평가를 가능하게 합니다.

## Installation and Setup

설치 옵션은 [Label Studio 설치 가이드](https://labelstud.io/guide/install)를 참조하세요.

`label-studio`와 `label-studio-sdk-python` Python 패키지를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install label-studio label-studio-sdk
```

```bash uv
uv add label-studio label-studio-sdk
```
</CodeGroup>


## Callbacks

[사용 예제](/oss/python/integrations/callbacks/labelstudio)를 참조하세요.

```python
from langchain.callbacks import LabelStudioCallbackHandler
```
