## Overview

`LocalFileStore`는 선택한 폴더에 모든 것을 저장하는 `ByteStore`의 영구 구현입니다. 단일 머신을 사용하고 파일이 추가되거나 삭제되는 것을 허용할 수 있는 경우 유용합니다.

### Installation

LangChain `LocalFileStore` integration은 `langchain` 패키지에 포함되어 있습니다:

```python
pip install -qU langchain-classic
```

## Instantiation

이제 byte store를 인스턴스화할 수 있습니다:

```python
from pathlib import Path

from langchain_classic.storage import LocalFileStore

root_path = Path.cwd() / "data"  # can also be a path set by a string

kv_store = LocalFileStore(root_path)
```

## Usage

`mset` method를 사용하여 다음과 같이 key 아래에 데이터를 설정할 수 있습니다:

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```output
[b'value1', b'value2']
```

`data` 폴더에서 생성된 파일을 확인할 수 있습니다:

```python
!ls {root_path}
```

```output
key1 key2
```

그리고 `mdelete` method를 사용하여 데이터를 삭제할 수 있습니다:

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```output
[None, None]
```
