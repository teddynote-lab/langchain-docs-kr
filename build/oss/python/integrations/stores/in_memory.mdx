## Overview

`InMemoryByteStore`는 Python dictionary에 모든 것을 저장하는 `ByteStore`의 비영구적 구현입니다. 이는 데모 및 Python 프로세스의 수명을 넘어서는 영속성이 필요하지 않은 경우를 위한 것입니다.

### Installation

LangChain `InMemoryByteStore` integration은 `langchain-core` package에 포함되어 있습니다:

```python
pip install -qU langchain-core
```

## Instantiation

이제 byte store를 인스턴스화할 수 있습니다:

```python
from langchain_core.stores import InMemoryByteStore

kv_store = InMemoryByteStore()
```

## Usage

`mset` method를 사용하여 다음과 같이 key 아래에 데이터를 설정할 수 있습니다:

```python
kv_store.mset(
    [
        ["key1", b"value1"],
        ["key2", b"value2"],
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```output
[b'value1', b'value2']
```

그리고 `mdelete` method를 사용하여 데이터를 삭제할 수 있습니다:

```python
kv_store.mdelete(
    [
        "key1",
        "key2",
    ]
)

kv_store.mget(
    [
        "key1",
        "key2",
    ]
)
```

```output
[None, None]
```
