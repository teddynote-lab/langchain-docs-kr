**LangGraph v1은 에이전트 런타임의 안정성에 중점을 둔 릴리스입니다.** 핵심 graph API와 실행 모델은 변경되지 않은 채로 유지하면서, 타입 안전성, 문서, 개발자 편의성을 개선했습니다.

[LangChain v1](/oss/python/releases/langchain-v1)과 함께 작동하도록 설계되었으며 (LangChain의 `create_agent`는 LangGraph 기반으로 구축됨), 높은 수준에서 시작하여 필요할 때 세밀한 제어로 전환할 수 있습니다.

<CardGroup cols={1}>
    <Card title="안정적인 핵심 API" icon="diagram-project">
        Graph 기본 요소(state, node, edge)와 실행/런타임 모델은 변경되지 않아 업그레이드가 간단합니다.
    </Card>
    <Card title="기본적으로 제공되는 안정성" icon="database">
        체크포인팅, 영속성, 스트리밍, human-in-the-loop을 통한 내구성 있는 실행이 계속해서 일급 기능으로 제공됩니다.
    </Card>
    <Card title="LangChain v1과 원활한 통합" icon="link">
        LangChain의 `create_agent`는 LangGraph에서 실행됩니다. 빠른 시작을 위해 LangChain을 사용하고, 커스텀 오케스트레이션을 위해 LangGraph로 전환하세요.
    </Card>
</CardGroup>

업그레이드하려면,

<CodeGroup>
```bash pip
pip install -U langgraph
```
```bash uv
uv add langgraph
```
</CodeGroup>

## `create_react_agent`의 지원 중단

LangGraph [`create_react_agent`](https://reference.langchain.com/python/langgraph/prebuilt/#langgraph.prebuilt.chat_agent_executor.create_react_agent) 사전 구축 기능은 LangChain의 [`create_agent`](https://reference.langchain.com/python/langchain/agents/#langchain.agents.create_agent)를 위해 지원 중단되었습니다. 더 간단한 인터페이스를 제공하며, 미들웨어 도입을 통해 더 큰 커스터마이징 가능성을 제공합니다.

* 새로운 [`create_agent`](https://reference.langchain.com/python/langchain/agents/#langchain.agents.create_agent) API에 대한 정보는 [LangChain v1 릴리스 노트](/oss/python/releases/langchain-v1#create-agent)를 참조하세요.
* [`create_react_agent`](https://reference.langchain.com/python/langgraph/prebuilt/#langgraph.prebuilt.chat_agent_executor.create_react_agent)에서 [`create_agent`](https://reference.langchain.com/python/langchain/agents/#langchain.agents.create_agent)로 마이그레이션하는 방법은 [LangChain v1 마이그레이션 가이드](/oss/python/migrate/langchain-v1#create-agent)를 참조하세요.
