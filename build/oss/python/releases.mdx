## 최신 릴리스

현재 안정 버전은 v1.x입니다. 릴리스 주기에 대한 자세한 내용은 [버전 관리 정책](/oss/python/versioning)을 참조하세요.

자세한 릴리스 노트와 마이그레이션 가이드가 포함된 릴리스 페이지 링크는 사이드바를 참조하세요.
