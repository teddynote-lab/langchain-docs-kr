우리는 문서를 최대한 접근 가능하게 만들고자 합니다. AI 어시스턴트, 코드 에디터, Model Context Protocol (MCP)과 같은 직접 통합을 통해 이러한 문서를 프로그래밍 방식으로 사용할 수 있는 여러 방법을 제공합니다.

## MCP server 사용하기

우리 문서에는 AI 애플리케이션이 최신 문서를 실시간으로 쿼리할 수 있는 내장 **Model Context Protocol (MCP) server**가 포함되어 있습니다.

LangChain docs MCP server는 다음 주소에서 사용할 수 있습니다:

```txt
https://docs.langchain.com/mcp
```

연결되면 LangChain, LangGraph, LangSmith에 대해 AI 어시스턴트에게 질문할 수 있으며, 정확하고 최신의 답변을 제공하기 위해 우리 문서를 검색합니다.

### Claude Code와 연결하기

Claude Code를 사용하는 경우, 터미널에서 다음 명령을 실행하여 현재 프로젝트에 server를 추가하세요:

```bash
claude mcp add --transport http docs-langchain https://docs.langchain.com/mcp
```

<Note>
    **프로젝트(로컬) 범위**

    위 명령은 현재 프로젝트/작업 디렉토리에만 MCP server를 추가합니다. MCP server를 전역으로 추가하고 모든 프로젝트에서 액세스하려면 명령에 `--scope user`를 추가하여 user scope를 추가하세요:

    ```bash
    claude mcp add --transport http docs-langchain --scope user https://docs.langchain.com/mcp
    ```
</Note>

### Claude Desktop과 연결하기

1. Claude Desktop을 엽니다
2. Settings > Connectors로 이동합니다
3. MCP server URL을 추가합니다: `https://docs.langchain.com/mcp`

### Cursor 또는 VS Code와 연결하기

MCP settings configuration file에 다음을 추가하세요:

```json
{
  "mcpServers": {
    "docs-langchain": {
      "url": "https://docs.langchain.com/mcp"
    }
  }
}
```
