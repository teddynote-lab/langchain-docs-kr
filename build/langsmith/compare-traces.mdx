---
title: Trace 비교하기
sidebarTitle: Trace 비교하기
---

trace를 비교하려면 trace 뷰의 오른쪽 상단에 있는 **Compare** 버튼을 클릭하세요.

![](/langsmith/images/compare-button.png)

trace run 테이블이 표시됩니다. 원본 trace와 비교할 trace를 선택하세요.

![](/langsmith/images/select-trace.png)

선택한 두 trace가 나란히 비교되는 뷰로 패널이 열립니다.

![](/langsmith/images/compare-trace.png)

비교를 중지하려면 패널을 닫거나 패널 오른쪽 상단의 **Stop comparing**을 클릭하세요.
