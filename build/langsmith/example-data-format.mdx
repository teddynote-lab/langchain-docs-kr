LangSmith는 dataset에 예제를 다음과 같이 저장합니다:

| Field Name          | Type     | Description                                                                                                       |
| ------------------- | -------- | ----------------------------------------------------------------------------------------------------------------- |
| **id**              | UUID     | 예제의 고유 식별자입니다.                                                                                |
| **name**            | string   | 예제의 이름입니다.                                                                                          |
| **created\_at**     | datetime | 이 예제가 생성된 시간입니다.                                                                                 |
| **modified\_at**    | datetime | 이 예제가 마지막으로 수정된 시간입니다.                                                                           |
| **inputs**          | object   | 예제의 입력값 맵입니다.                                                                                  |
| **outputs**         | object   | 실행에 의해 생성된 출력값의 맵 또는 집합입니다.                                                                     |
| **dataset\_id**     | UUID     | 예제가 속한 dataset입니다.                                                                                |
| **source\_run\_id** | UUID     | 이 예제가 LangSmith [`Run`](/langsmith/run-data-format)에서 생성된 경우, 해당 run의 ID입니다. |
| **metadata**        | object   | 예제에 저장할 수 있는 추가적인 사용자 또는 SDK 정의 정보의 맵입니다.                            |

평가에서 예제가 어떻게 사용되는지 자세히 알아보려면 [LLM 애플리케이션 평가하기](/langsmith/evaluate-llm-application)에 대한 how-to 가이드를 읽어보세요.
