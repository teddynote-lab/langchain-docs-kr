왼쪽 사이드바의 섹션 또는 애플리케이션 홈페이지에서 편집하려는 prompt를 클릭하세요.

## 메타데이터 업데이트

prompt 메타데이터(설명, 사용 사례 등)를 업데이트하려면 "Edit" 연필 아이콘을 클릭하세요.

![](/langsmith/images/metadata-edit-button.png)

저장하면 prompt 메타데이터가 업데이트됩니다.

![](/langsmith/images/update-prompt-form.png)

## Prompt 콘텐츠 업데이트

prompt 콘텐츠 자체를 업데이트하려면 prompt playground에 들어가야 합니다. "Edit in playground"를 클릭하세요. 이제 prompt를 변경하고 다양한 입력으로 테스트할 수 있습니다. prompt가 만족스러우면 "Commit"을 클릭하여 저장하세요.

![](/langsmith/images/edit-in-playground.png)
![](/langsmith/images/prompt-playground-edit-commit.png)

## Prompt 버전 관리

prompt에 commit을 추가하면 새로운 버전의 prompt가 생성됩니다. prompt 보기에서 "Commits" 탭을 클릭하여 모든 이전 버전을 확인할 수 있습니다.

![](/langsmith/images/prompt-commits-tab.png)
