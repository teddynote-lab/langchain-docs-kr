일부 애플리케이션은 PDF나 이미지에 대한 질문에 답변할 수 있는 챗봇과 같이 멀티모달 콘텐츠를 기반으로 합니다. 이러한 경우, prompt에 멀티모달 콘텐츠를 포함하고 콘텐츠에 대한 질문에 답변하는 모델의 능력을 테스트하고 싶을 것입니다.

LangSmith Playground는 prompt에 멀티모달 콘텐츠를 통합하는 두 가지 방법을 지원합니다:

1. Inline 콘텐츠: 정적 파일(이미지, PDF, 오디오)을 prompt에 직접 임베드합니다. 이는 prompt의 모든 사용에서 동일한 멀티모달 콘텐츠를 일관되게 포함하려는 경우에 이상적입니다. 예를 들어, 모델의 응답을 기반으로 하는 참조 이미지를 포함할 수 있습니다.

2. Template variable: 매번 다른 콘텐츠로 채울 수 있는 첨부 파일용 동적 placeholder를 생성합니다. 이 접근 방식은 더 많은 유연성을 제공하여 다음을 가능하게 합니다:

   * 모델이 다양한 입력을 처리하는 방식 테스트
   * 다양한 콘텐츠와 함께 작동하는 재사용 가능한 prompt 생성

<Note>
모든 모델이 멀티모달 콘텐츠를 지원하는 것은 아닙니다. playground에서 멀티모달 기능을 사용하기 전에 선택한 모델이 사용하려는 파일 유형을 지원하는지 확인하세요.
</Note>

## Inline 콘텐츠

멀티모달 콘텐츠를 추가하려는 메시지에서 파일 아이콘을 클릭합니다. `Upload content` 탭에서 파일을 업로드하고 prompt에 inline으로 포함할 수 있습니다.

![](/langsmith/images/upload-inline-multimodal-content.png)

## Template variable

멀티모달 콘텐츠를 추가하려는 메시지에서 파일 아이콘을 클릭합니다. `Template variables` 탭에서 특정 첨부 파일 유형에 대한 template variable을 생성할 수 있습니다. 현재 이미지, PDF, 오디오 파일(.wav, .mp3)만 지원됩니다.

![](/langsmith/images/template-variable-multimodal-content.png)

## Template variable 채우기

Template variable을 추가한 후, 화면 오른쪽의 패널을 사용하여 콘텐츠를 제공할 수 있습니다. `+` 버튼을 클릭하여 template variable을 채우는 데 사용할 콘텐츠를 업로드하거나 선택하기만 하면 됩니다.

![](/langsmith/images/manual-prompt-multimodal.png)

## Evaluation 실행

prompt를 수동으로 테스트한 후, [evaluation을 실행](/langsmith/evaluate-with-attachments?mode=ui)하여 골든 데이터셋 예제에서 prompt가 어떻게 수행되는지 확인할 수 있습니다.
