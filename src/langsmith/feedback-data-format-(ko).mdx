**Feedback**은 특정 trace 또는 중간 run(span)에 대한 평가 기준과 점수를 저장하는 LangSmith의 방식입니다. Feedback은 다음과 같은 다양한 방법으로 생성될 수 있습니다:

1. LLM 애플리케이션에서 [trace와 함께 전송](/langsmith/attach-user-feedback)
2. 앱에서 사용자가 [인라인](/langsmith/annotate-traces-inline)으로 또는 [annotation queue](/langsmith/annotation-queues)에서 생성
3. [오프라인 평가](/langsmith/evaluate-llm-application) 중 자동 evaluator에 의해 생성
4. [온라인 evaluator](/langsmith/online-evaluations)에 의해 생성

Feedback은 다음 필드를 포함하는 간단한 형식으로 저장됩니다:

<FeedbackDataFields/>
