이 페이지는 Agent Builder를 사용하기 전에 추가해야 하는 workspace secret 목록입니다. LangSmith workspace 설정의 Secrets에서 이를 추가하세요. 값은 workspace 범위로 유지하고 프롬프트나 코드에 자격 증명을 넣지 마세요.

## workspace secret 추가 방법

[LangSmith UI](https://smith.langchain.com)에서 Anthropic API key가 [workspace secret](/langsmith/administration-overview#workspace-secrets)으로 설정되어 있는지 확인하세요.

1. <Icon icon="gear" /> **Settings**로 이동한 다음 **Secrets** 탭으로 이동하세요.
1. **Add secret**을 선택하고 `ANTHROPIC_API_KEY`를 입력한 후 API key를 **Value**로 입력하세요.
1. **Save secret**을 선택하세요.

<Note> LangSmith UI에서 workspace secret을 추가할 때 secret key가 모델 제공자가 예상하는 환경 변수 이름과 일치하는지 확인하세요.</Note>

## 필수 모델 key

- `ANTHROPIC_API_KEY`: Agent Builder 모델에 필요합니다. agent graph는 추론을 위해 workspace secret에서 이 key를 로드합니다.

## 선택적 도구 key

활성화하는 도구에 대한 key를 추가하세요. 이는 런타임에 workspace secret에서 읽어옵니다.

- `EXA_API_KEY`: Exa 검색 도구(일반 웹 및 LinkedIn 프로필 검색)에 필요합니다.
- `TAVILY_API_KEY`: Tavily 웹 검색에 필요합니다.
- `TWITTER_API_KEY` 및 `TWITTER_API_KEY_SECRET`: Twitter/X 읽기 작업(app‑only bearer)에 필요합니다. 게시/미디어 업로드는 활성화되지 않습니다.
