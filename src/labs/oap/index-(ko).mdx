## Open Agent Platform이란?

Open Agent Platform은 LangGraph 에이전트를 생성, 관리 및 상호작용하기 위한 현대적인 웹 기반 인터페이스를 제공합니다. 단순성을 염두에 두고 설계되어 기술 전문 지식이 없는 사용자도 접근할 수 있으면서, 개발자를 위한 고급 기능도 제공합니다.

## 주요 기능

- **Agent Management**: 직관적인 인터페이스를 통해 에이전트를 구축, 구성 및 상호작용
- **RAG Integration**: [LangConnect](https://github.com/langchain-ai/langconnect)를 사용한 Retrieval Augmented Generation에 대한 일급 지원
- **MCP Tools**: MCP 서버를 통해 에이전트를 외부 도구에 연결
- **Agent Supervision**: Agent Supervisor를 통해 여러 에이전트가 함께 작동하도록 조율
- **Authentication**: 내장된 인증 및 접근 제어
