LLM을 사용하여 loop에서 tool을 호출하는 것은 가장 간단한 형태의 agent입니다. 그러나 이러한 아키텍처는 "얕은" agent를 생성하여 더 길고 복잡한 작업에 대해 계획하고 실행하는 데 실패할 수 있습니다. "Deep Research", "Manus", "Claude Code"와 같은 애플리케이션은 네 가지 요소의 조합을 구현하여 이러한 제한을 극복했습니다: **planning tool**, **sub agents**, **file system**에 대한 액세스, 그리고 **상세한 prompt**.

Deep Agents는 이러한 구성 요소를 범용적인 방식으로 구현하여 **Python**과 **JavaScript/TypeScript** 모두에서 애플리케이션을 위한 정교한 agent를 쉽게 만들 수 있도록 합니다.

## 주요 기능

- **Planning Tool**: 복잡한 작업을 분해하고 장기 계획을 가능하게 하는 내장 계획 기능.
- **Persistent state**: 대화 기록을 넘어선 tool 호출 간의 메모리.
- **Sub-agents**: context 격리를 통한 특정 작업을 위한 전문화된 agent.
- **Virtual File System**: 충돌 없이 persistent state를 위한 mock file system.
- **Detailed System Prompt**: Claude Code에서 크게 영감을 받은 prompt로, 검증된 패턴을 활용합니다.

Deep Agents는 Claude Code와 같은 정교한 agent를 효과적으로 만드는 아키텍처 패턴을 구현합니다.
