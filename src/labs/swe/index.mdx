Open SWE는 [LangGraph](https://docs.langchain.com/oss/javascript/langgraph/overview)로 구축된 오픈 소스 클라우드 기반 코딩 에이전트입니다. 전체 repository에 걸쳐 코드 변경 사항을 자율적으로 이해하고, 계획하고, 실행하도록 설계되었습니다.

## 작동 방식

Open SWE는 세 가지 특화된 LangGraph 에이전트를 통해 작동합니다:

- **Manager Graph**: 사용자 상호작용을 조율하고 다른 graph들 간의 조정을 담당합니다
- **Planner Graph**: 요구사항을 분석하고 상세한 실행 계획을 생성합니다
- **Programmer Graph**: 승인된 계획을 기반으로 코드 변경을 실행합니다

이 에이전트는 웹 인터페이스를 통해 사용하거나 GitHub webhook을 통해 자동으로 트리거할 수 있어, 대화형 개발과 자동화된 워크플로우 모두에 유연하게 활용할 수 있습니다.
