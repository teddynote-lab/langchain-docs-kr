## Technology Stack

Open SWE는 확장성과 개발자 경험을 위해 설계된 최신 기술로 구축되었습니다:

### Core Technologies

- **TypeScript** - 전체 codebase에 걸친 엄격한 type safety
- **Yarn** - workspace 지원이 있는 package manager (v3.5.1)
- **Turbo** - Monorepo build orchestration 및 task running

### Agent Infrastructure

- **LangGraph** - 세 가지 특화된 graph를 가진 multi-agent orchestration framework:
  - 사용자 상호작용 orchestration을 위한 Manager graph
  - 실행 계획 생성을 위한 Planner graph
  - 코드 변경 실행을 위한 Programmer graph
- **Daytona** - 안전한 코드 실행을 위한 sandboxed development environment

### Web Application

- **Next.js** - App Router를 사용하는 React framework
- **Shadcn UI** - Radix UI primitive 기반으로 구축된 component library
- **Tailwind CSS** - Utility-first CSS framework

### Documentation

- **Mintlify** - MDX 지원이 있는 documentation platform

## Monorepo Structure

<Note>
  Open SWE는 세 개의 주요 application과 공통 utility 및 type을 위한 공유 package가 있는 Yarn workspace monorepo를 사용합니다.
</Note>

- **`apps/open-swe`** - LangGraph agent application
- **`apps/web`** - Next.js web interface
- **`apps/docs`** - Mintlify documentation site
- **`packages/shared`** - 공유 utility, type 및 constant

monorepo는 Turbo에 의해 orchestration되며, package 간 build dependency 및 병렬 task 실행을 처리합니다.
