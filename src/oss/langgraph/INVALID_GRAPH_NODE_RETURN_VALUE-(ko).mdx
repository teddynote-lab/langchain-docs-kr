---
title: INVALID_GRAPH_NODE_RETURN_VALUE
---



:::python
LangGraph [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.state.StateGraph)가
node로부터 dict가 아닌 반환 타입을 받았습니다. 다음은 예시입니다:

```python
class State(TypedDict):
    some_key: str

def bad_node(state: State):
    # Should return a dict with a value for "some_key", not a list
    return ["whoops"]

builder = StateGraph(State)
builder.add_node(bad_node)
...

graph = builder.compile()
```

위 graph를 호출하면 다음과 같은 오류가 발생합니다:

```python
graph.invoke({ "some_key": "someval" });
```

```
InvalidUpdateError: Expected dict, got ['whoops']
For troubleshooting, visit: https://python.langchain.com/docs/troubleshooting/errors/INVALID_GRAPH_NODE_RETURN_VALUE
```

graph의 node는 state에 정의된 하나 이상의 key를 포함하는 dict를 반환해야 합니다.
:::

:::js
LangGraph [`StateGraph`](https://langchain-ai.github.io/langgraph/reference/graphs/#langgraph.graph.state.StateGraph)가
node로부터 object가 아닌 반환 타입을 받았습니다. 다음은 예시입니다:

```typescript
import * as z from "zod";
import { StateGraph } from "@langchain/langgraph";

const State = z.object({
  someKey: z.string(),
});

const badNode = (state: z.infer<typeof State>) => {
  // Should return an object with a value for "someKey", not an array
  return ["whoops"];
};

const builder = new StateGraph(State).addNode("badNode", badNode);
// ...

const graph = builder.compile();
```

위 graph를 호출하면 다음과 같은 오류가 발생합니다:

```typescript
await graph.invoke({ someKey: "someval" });
```

```
InvalidUpdateError: Expected object, got ['whoops']
For troubleshooting, visit: https://langchain-ai.github.io/langgraphjs/troubleshooting/errors/INVALID_GRAPH_NODE_RETURN_VALUE
```

graph의 node는 state에 정의된 하나 이상의 key를 포함하는 object를 반환해야 합니다.
:::

## Troubleshooting

다음 사항이 이 오류를 해결하는 데 도움이 될 수 있습니다:

:::python
* node에 복잡한 로직이 있는 경우, 모든 코드 경로가 정의된 state에 적합한 dict를 반환하는지 확인하세요.
:::

:::js
* node에 복잡한 로직이 있는 경우, 모든 코드 경로가 정의된 state에 적합한 object를 반환하는지 확인하세요.
:::
