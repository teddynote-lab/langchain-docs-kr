---
title: MULTIPLE_SUBGRAPHS
---



단일 LangGraph node 내에서 각 subgraph에 대해 checkpointing이 활성화된 상태로 subgraph를 여러 번 호출하고 있습니다.

이는 현재 subgraph의 checkpoint namespacing 작동 방식에 대한 내부 제한으로 인해 허용되지 않습니다.

## Troubleshooting

다음 방법이 이 오류를 해결하는 데 도움이 될 수 있습니다:

:::python
* subgraph에서 interrupt/resume이 필요하지 않은 경우, 다음과 같이 컴파일할 때 `checkpointer=False`를 전달하세요: `.compile(checkpointer=False)`
:::

:::js
* subgraph에서 interrupt/resume이 필요하지 않은 경우, 다음과 같이 컴파일할 때 `checkpointer: false`를 전달하세요: `.compile({ checkpointer: false })`
:::

* 동일한 node에서 graph를 여러 번 명령적으로 호출하지 말고, 대신 [`Send`](/oss/langgraph/graph-api#send) API를 사용하세요.
