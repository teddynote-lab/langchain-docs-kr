MHTML은 이메일뿐만 아니라 보관된 웹페이지에도 사용됩니다. MHTML은 때때로 MHT라고도 불리며, MIME HTML의 약자로 전체 웹페이지가 하나의 파일로 보관됩니다. 웹페이지를 MHTML 형식으로 저장하면, 이 파일 확장자에는 HTML 코드, 이미지, 오디오 파일, 플래시 애니메이션 등이 포함됩니다.

```python
from langchain_community.document_loaders import MHTMLLoader
```

```python
# Create a new loader object for the MHTML file
loader = MHTMLLoader(
    file_path="../../../../../../tests/integration_tests/examples/example.mht"
)

# Load the document from the file
documents = loader.load()

# Print the documents to see the results
for doc in documents:
    print(doc)
```
