>[AZLyrics](https://www.azlyrics.com/)는 매일 성장하는 대규모의 합법적인 가사 모음집입니다.

이 문서는 AZLyrics 웹페이지를 다운스트림에서 사용할 수 있는 document 형식으로 로드하는 방법을 다룹니다.

```python
from langchain_community.document_loaders import AZLyricsLoader
```

```python
loader = AZLyricsLoader("https://www.azlyrics.com/lyrics/mileycyrus/flowers.html")
```

```python
data = loader.load()
```
