>[Messenger](https://en.wikipedia.org/wiki/Messenger_(software))는 `Meta Platforms`에서 개발한 미국의 독점 인스턴트 메시징 앱 및 플랫폼입니다. 원래 2008년에 `Facebook Chat`으로 개발되었으며, 회사는 2010년에 메시징 서비스를 개편했습니다.

이 노트북은 [Facebook Chats](https://www.facebook.com/business/help/1646890868956360)에서 데이터를 로드하여 LangChain에 수집할 수 있는 형식으로 변환하는 방법을 다룹니다.

```python
# pip install pandas
```

```python
from langchain_community.document_loaders import FacebookChatLoader
```

```python
loader = FacebookChatLoader("example_data/facebook_chat.json")
```

```python
loader.load()
```
