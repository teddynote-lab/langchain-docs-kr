[Socrata](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6)는 도시 공공 데이터를 위한 API를 제공합니다.

[SF crime](https://data.sfgov.org/Public-Safety/Police-Department-Incident-Reports-Historical-2003/tmnf-yvry)과 같은 데이터셋의 경우, 오른쪽 상단의 `API` 탭을 확인하세요.

해당 탭에서 `dataset identifier`를 확인할 수 있습니다.

dataset identifier를 사용하여 특정 city_id(`data.sfgov.org`)에 대한 특정 테이블을 가져올 수 있습니다 -

예: [SF 311 data](https://dev.socrata.com/foundry/data.sfgov.org/vw6y-z8j6)의 경우 `vw6y-z8j6`

예: [SF Police data](https://dev.socrata.com/foundry/data.sfgov.org/tmnf-yvry)의 경우 `tmnf-yvry`

```python
pip install -qU  sodapy
```

```python
from langchain_community.document_loaders import OpenCityDataLoader
```

```python
dataset = "vw6y-z8j6"  # 311 data
dataset = "tmnf-yvry"  # crime data
loader = OpenCityDataLoader(city_id="data.sfgov.org", dataset_id=dataset, limit=2000)
```

```python
docs = loader.load()
```
