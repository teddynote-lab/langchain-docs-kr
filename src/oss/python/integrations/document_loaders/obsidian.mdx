>[Obsidian](https://obsidian.md/)은 로컬 폴더의 일반 텍스트 파일 위에서 작동하는 강력하고 확장 가능한 지식 베이스입니다.

이 노트북은 `Obsidian` 데이터베이스에서 문서를 로드하는 방법을 다룹니다.

`Obsidian`은 Markdown 파일 폴더로 디스크에 저장되므로, 로더는 이 디렉토리 경로만 필요합니다.

`Obsidian` 파일은 때때로 파일 상단에 YAML 블록 형태의 [metadata](https://help.obsidian.md/Editing+and+formatting/Metadata)를 포함하기도 합니다. 이러한 값들은 문서의 metadata에 추가됩니다. (`ObsidianLoader`에 `collect_metadata=False` 인자를 전달하여 이 동작을 비활성화할 수도 있습니다.)

```python
from langchain_community.document_loaders import ObsidianLoader
```

```python
loader = ObsidianLoader("<path-to-obsidian>")
```

```python
docs = loader.load()
```
