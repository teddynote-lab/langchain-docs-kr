>[Jupyter Notebook](https://en.wikipedia.org/wiki/Project_Jupyter#Applications) (이전 명칭 `IPython Notebook`)은 notebook 문서를 생성하기 위한 웹 기반 대화형 계산 환경입니다.

이 notebook은 `Jupyter notebook (.ipynb)`에서 LangChain에 적합한 형식으로 데이터를 로드하는 방법을 다룹니다.

```python
from langchain_community.document_loaders import NotebookLoader
```

```python
loader = NotebookLoader(
    "example_data/notebook.ipynb",
    include_outputs=True,
    max_output_length=20,
    remove_newline=True,
)
```

`NotebookLoader.load()`는 `.ipynb` notebook 파일을 @[`Document`] 객체로 로드합니다.

**Parameters**:

* `include_outputs` (bool): 결과 document에 cell output을 포함할지 여부 (기본값은 False).
* `max_output_length` (int): 각 cell output에서 포함할 최대 문자 수 (기본값은 10).
* `remove_newline` (bool): cell source와 output에서 줄바꿈 문자를 제거할지 여부 (기본값은 False).
* `traceback` (bool): 전체 traceback을 포함할지 여부 (기본값은 False).

```python
loader.load()
```
