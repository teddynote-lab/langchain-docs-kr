### As a ToolCall

모델이 생성한 `ToolCall`로 tool을 호출할 수도 있으며, 이 경우 @[`ToolMessage`]가 반환됩니다:

```python ToolCall icon="toolbox"
# This is usually generated by a model, but we'll create a tool call directly for demo purposes.
model_generated_tool_call = {
    "args": {...},  # TODO: FILL IN
    "id": "1",
    "name": tool.name,
    "type": "tool_call",
}
tool.invoke(model_generated_tool_call)
```

### Within an agent

agent에서 tool을 사용할 수 있습니다. 이를 위해서는 tool-calling 기능이 있는 모델이 필요합니다.

```python Agent with tool icon="robot"
# pip install -qU "langchain[anthropic]" to call the model
from langchain.agents import create_agent

tools = [tool]
agent = create_agent(
    model="anthropic:claude-sonnet-4-5",
    tools=tools,
)

# Run the agent
agent.invoke(
    {"messages": [{"role": "user", "content": "INSERT QUERY"}]}
)
```
