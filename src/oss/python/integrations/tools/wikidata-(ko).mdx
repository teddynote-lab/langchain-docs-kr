>[Wikidata](https://wikidata.org/)는 인간과 기계 모두가 읽고 편집할 수 있는 무료 오픈 지식 베이스입니다. Wikidata는 세계에서 가장 큰 오픈 지식 베이스 중 하나입니다.

먼저 `wikibase-rest-api-client`와 `mediawikiapi` python package를 설치해야 합니다.

```python
pip install -qU wikibase-rest-api-client mediawikiapi
```

```python
from langchain_community.tools.wikidata.tool import WikidataAPIWrapper, WikidataQueryRun

wikidata = WikidataQueryRun(api_wrapper=WikidataAPIWrapper())

print(wikidata.run("Alan Turing"))
```
