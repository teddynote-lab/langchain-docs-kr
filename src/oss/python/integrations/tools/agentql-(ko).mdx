[AgentQL](https://www.agentql.com/) tool은 [AgentQL query](https://docs.agentql.com/agentql-query) 또는 자연어 프롬프트를 사용하여 모든 웹 페이지에서 웹 상호작용 및 구조화된 데이터 추출을 제공합니다. AgentQL은 여러 언어와 웹 페이지에서 시간이 지나도 변경에 영향받지 않고 사용할 수 있습니다.

## Overview

AgentQL은 다음 세 가지 tool을 제공합니다:

- **`ExtractWebDataTool`**은 [AgentQL query](https://docs.agentql.com/agentql-query/query-intro) 또는 데이터에 대한 자연어 설명을 사용하여 URL이 주어진 웹 페이지에서 구조화된 데이터를 JSON으로 추출합니다.

다음 두 tool은 `AgentQLBrowserToolkit`으로 번들되어 있으며 `Playwright` browser 또는 Chrome DevTools Protocol (CDP)을 통한 원격 browser instance와 함께 사용해야 합니다:

- **`ExtractWebDataBrowserTool`**은 [AgentQL query](https://docs.agentql.com/agentql-query/query-intro) 또는 자연어 설명을 사용하여 browser의 활성 웹 페이지에서 구조화된 데이터를 JSON으로 추출합니다.

- **`GetWebElementBrowserTool`**은 자연어 설명을 사용하여 browser의 활성 웹 페이지에서 웹 element를 찾고 추가 상호작용을 위해 CSS selector를 반환합니다.

## Instantiation

### `ExtractWebDataTool`

다음 매개변수로 `ExtractWebDataTool`을 인스턴스화할 수 있습니다:

- `api_key`: [dev.agentql.com](https://dev.agentql.com)에서 받은 AgentQL API key. **`선택사항`.**
- `timeout`: 요청이 timeout되기 전에 대기할 시간(초). 데이터 추출이 timeout되면 증가시키세요. **기본값은 `900`.**
- `is_stealth_mode_enabled`: 실험적인 봇 회피 전략을 활성화할지 여부. 이 기능은 모든 웹사이트에서 항상 작동하지 않을 수 있습니다. 이 모드를 활성화하면 데이터 추출이 더 오래 걸릴 수 있습니다. **기본값은 `False`.**
- `wait_for`: 데이터를 추출하기 전에 페이지가 로드될 때까지 대기할 시간(초). **기본값은 `0`.**
- `is_scroll_to_bottom_enabled`: 데이터를 추출하기 전에 페이지 하단으로 스크롤할지 여부. **기본값은 `False`.**
- `mode`: `"standard"`는 심층 데이터 분석을 사용하고, `"fast"`는 일부 분석 깊이를 속도와 교환하며 대부분의 사용 사례에 적합합니다. [이 가이드에서 모드에 대해 자세히 알아보세요.](https://docs.agentql.com/accuracy/standard-mode) **기본값은 `"fast"`.**
- `is_screenshot_enabled`: 데이터를 추출하기 전에 스크린샷을 찍을지 여부. Base64 문자열로 'metadata'에 반환됩니다. **기본값은 `False`.**

`ExtractWebDataTool`은 AgentQL의 REST API로 구현되어 있으며, [API Reference docs](https://docs.agentql.com/rest-api/api-reference)에서 매개변수에 대한 자세한 내용을 확인할 수 있습니다.

```python
from langchain_agentql.tools import ExtractWebDataTool

extract_web_data_tool = ExtractWebDataTool()
```

### `ExtractWebDataBrowserTool`

**ExtractWebDataBrowserTool**을 인스턴스화하려면 tool을 browser instance와 연결해야 합니다.

다음 매개변수를 설정할 수 있습니다:

- `timeout`: 요청이 timeout되기 전에 대기할 시간(초). 데이터 추출이 timeout되면 증가시키세요. **기본값은 `900`.**
- `wait_for_network_idle`: 실행하기 전에 네트워크가 완전히 유휴 상태에 도달할 때까지 대기할지 여부. **기본값은 `True`.**
- `include_hidden`: 페이지에서 시각적으로 숨겨진 element를 고려할지 여부. **기본값은 `True`.**
- `mode`: `"standard"`는 심층 데이터 분석을 사용하고, `"fast"`는 일부 분석 깊이를 속도와 교환하며 대부분의 사용 사례에 적합합니다. [이 가이드에서 모드에 대해 자세히 알아보세요.](https://docs.agentql.com/accuracy/standard-mode) **기본값은 `"fast"`.**

`ExtractWebDataBrowserTool`은 AgentQL의 SDK로 구현되어 있습니다. 매개변수와 함수에 대한 자세한 내용은 AgentQL의 [API References](https://docs.agentql.com/python-sdk/api-references/agentql-page#querydata)에서 확인할 수 있습니다.

```python
from langchain_agentql.tools import ExtractWebDataBrowserTool
from langchain_agentql.utils import create_async_playwright_browser

async_browser = await create_async_playwright_browser()

extract_web_data_browser_tool = ExtractWebDataBrowserTool(async_browser=async_browser)
```

## Invocation

### `ExtractWebDataTool`

이 tool은 내부적으로 AgentQL의 REST API를 사용하여 공개적으로 사용 가능한 웹 페이지의 URL을 AgentQL의 endpoint로 전송합니다. 이는 비공개 페이지나 로그인된 세션에서는 작동하지 않습니다. 이러한 사용 사례에는 `ExtractWebDataBrowserTool`을 사용하세요.

- `url`: 데이터를 추출하려는 웹 페이지의 URL.
- `query`: 실행할 AgentQL query. 정확하게 구조화된 데이터를 추출하려면 AgentQL query를 사용하세요. [문서에서 AgentQL query 작성 방법에 대해 자세히 알아보거나](https://docs.agentql.com/agentql-query) [AgentQL Playground](https://dev.agentql.com/playground)에서 테스트해보세요.
- `prompt`: 페이지에서 추출할 데이터에 대한 자연어 설명. AgentQL은 프롬프트에서 데이터 구조를 추론합니다. 특정 구조를 정의하지 않고 자유 형식 언어로 정의된 데이터를 추출하려면 `prompt`를 사용하세요.

**참고:** AgentQL을 사용하려면 `query` 또는 `prompt` 중 하나를 정의해야 합니다.

## Chaining

AgentQL tool을 [tool-calling model](/oss/langchain/tools/)에 먼저 바인딩한 다음 호출하여 chain에서 사용할 수 있습니다:

### Instantiate LLM

```python
import os

os.environ["OPENAI_API_KEY"] = "YOUR_OPENAI_API_KEY"
```

```python
from langchain.chat_models import init_chat_model

model = init_chat_model(model="gpt-4o", model_provider="openai")
```

### Execute Tool Chain

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnableConfig, chain

prompt = ChatPromptTemplate(
    [
        ("system", "You are a helpful assistant in extracting data from website."),
        ("human", "{user_input}"),
        ("placeholder", "{messages}"),
    ]
)

# specifying tool_choice will force the model to call this tool.
model_with_tools = model.bind_tools(
    [extract_web_data_tool], tool_choice="extract_web_data_with_rest_api"
)

model_chain = prompt | model_with_tools


@chain
def tool_chain(user_input: str, config: RunnableConfig):
    input_ = {"user_input": user_input}
    ai_msg = model_chain.invoke(input_, config=config)
    tool_msgs = extract_web_data_tool.batch(ai_msg.tool_calls, config=config)
    return {"messages": tool_msgs}


tool_chain.invoke(
    "Extract data from https://www.agentql.com/blog using the following agentql query: { posts[] { title url date author } }"
)
```
