[Tools](/oss/langchain/tools)는 모델이 호출하도록 설계된 유틸리티입니다. 이들의 입력은 모델이 생성하기 쉽도록, 출력은 모델에 다시 전달하기 쉽도록 설계되어 있습니다.

Toolkit은 함께 사용하도록 설계된 tool의 모음입니다.
