`Hugging Face Spaces`에는 수천 개의 `Gradio` 앱이 있습니다. 이 라이브러리는 이러한 앱들을 LLM의 손끝에 가져다 놓습니다 🦾

구체적으로, `gradio-tools`는 `Gradio` 앱을 대규모 언어 모델(LLM) 기반 agent가 작업을 완료하는 데 활용할 수 있는 tool로 변환하는 Python 라이브러리입니다. 예를 들어, LLM은 `Gradio` tool을 사용하여 온라인에서 찾은 음성 녹음을 전사하고 요약할 수 있습니다. 또는 다른 `Gradio` tool을 사용하여 Google Drive의 문서에 OCR을 적용하고 이에 대한 질문에 답변할 수 있습니다.

사전 구축된 tool이 아닌 space를 사용하고 싶다면 자신만의 tool을 만드는 것은 매우 쉽습니다. 이를 수행하는 방법에 대한 정보는 gradio-tools 문서의 해당 섹션을 참조하세요. 모든 기여를 환영합니다!

```python
pip install -qU  gradio_tools langchain-community
```

## agent 내에서 사용하기

```python
from gradio_tools.tools import (
    ImageCaptioningTool,
    StableDiffusionPromptGeneratorTool,
    StableDiffusionTool,
    TextToVideoTool,
)
from langchain.agents import initialize_agent
from langchain.memory import ConversationBufferMemory
from langchain_openai import OpenAI

llm = OpenAI(temperature=0)
memory = ConversationBufferMemory(memory_key="chat_history")
tools = [
    StableDiffusionTool().langchain,
    ImageCaptioningTool().langchain,
    StableDiffusionPromptGeneratorTool().langchain,
    TextToVideoTool().langchain,
]


agent = initialize_agent(
    tools, llm, memory=memory, agent="conversational-react-description", verbose=True
)
output = agent.run(
    input=(
        "Please create a photo of a dog riding a skateboard "
        "but improve my prompt prior to using an image generator."
        "Please caption the generated image and create a video for it using the improved prompt."
    )
)
```
