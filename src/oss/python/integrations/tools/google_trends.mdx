이 노트북은 Google Trends Tool을 사용하여 트렌드 정보를 가져오는 방법을 다룹니다.

먼저, [serpapi.com/users/sign_up](https://serpapi.com/users/sign_up)에서 `SerpApi key`에 가입해야 합니다.

그런 다음 다음 명령어로 `google-search-results`를 설치해야 합니다:

`pip install google-search-results`

그런 다음 환경 변수 `SERPAPI_API_KEY`를 `SerpApi key`로 설정해야 합니다

[또는 wrapper에 argument로 key를 전달할 수 있습니다 `serp_api_key="your secret key"`]

## Tool 사용하기

```python
pip install -qU  google-search-results langchain_community
```

```python
import os

from langchain_community.tools.google_trends import GoogleTrendsQueryRun
from langchain_community.utilities.google_trends import GoogleTrendsAPIWrapper

os.environ["SERPAPI_API_KEY"] = ""
tool = GoogleTrendsQueryRun(api_wrapper=GoogleTrendsAPIWrapper())
```

```python
tool.run("Water")
```
