**Dappier: 동적이고 실시간 데이터 모델로 AI를 강화하세요**

Dappier는 개발자들에게 뉴스, 엔터테인먼트, 금융, 시장 데이터, 날씨 등 다양한 실시간 데이터 모델에 즉시 액세스할 수 있는 최첨단 플랫폼을 제공합니다. 사전 학습된 데이터 모델을 통해 AI 애플리케이션을 강화하여 정확하고 최신의 응답을 제공하고 부정확성을 최소화할 수 있습니다.

Dappier 데이터 모델은 세계 최고의 브랜드에서 제공하는 신뢰할 수 있고 최신의 콘텐츠로 차세대 LLM 앱을 구축하는 데 도움을 줍니다. 간단한 API를 통해 실행 가능한 독점 데이터로 GPT 앱이나 AI 워크플로우를 향상시키고 창의성을 발휘하세요. 신뢰할 수 있는 소스의 독점 데이터로 AI를 보강하는 것은 질문에 관계없이 환각을 줄이고 사실적이며 최신의 응답을 보장하는 가장 좋은 방법입니다.

개발자를 위한, 개발자에 의한
개발자를 염두에 두고 설계된 Dappier는 데이터 통합부터 수익화까지의 여정을 단순화하여 AI 모델을 배포하고 수익을 창출할 수 있는 명확하고 직관적인 경로를 제공합니다. **[dappier.com/](https://dappier.com/)**에서 새로운 인터넷을 위한 수익화 인프라의 미래를 경험하세요.

이 예제는 LangChain을 사용하여 Dappier AI 모델과 상호작용하는 방법을 다룹니다

-----------------------------------------------------------------------------------

Dappier AI 데이터 모델 중 하나를 사용하려면 API key가 필요합니다. Dappier Platform ([platform.dappier.com/](https://platform.dappier.com/))을 방문하여 로그인하고 프로필에서 API key를 생성하세요.

API reference에서 자세한 내용을 확인할 수 있습니다: [docs.dappier.com/introduction](https://docs.dappier.com/introduction)

Dappier Chat Model을 사용하려면 클래스를 초기화할 때 dappier_api_key라는 parameter를 통해 key를 직접 전달하거나
환경 변수로 설정할 수 있습니다.

```bash
export DAPPIER_API_KEY="..."
```

```python
from langchain_community.chat_models.dappier import ChatDappierAI
from langchain.messages import HumanMessage
```

```python
chat = ChatDappierAI(
    dappier_endpoint="https://api.dappier.com/app/datamodelconversation",
    dappier_model="dm_01hpsxyfm2fwdt2zet9cg6fdxt",
    dappier_api_key="...",
)
```

```python
messages = [HumanMessage(content="Who won the super bowl in 2024?")]
chat.invoke(messages)
```

```output
AIMessage(content='Hey there! The Kansas City Chiefs won Super Bowl LVIII in 2024. They beat the San Francisco 49ers in overtime with a final score of 25-22. It was quite the game! 🏈')
```

```python
await chat.ainvoke(messages)
```

```output
AIMessage(content='The Kansas City Chiefs won Super Bowl LVIII in 2024! 🏈')
```
