[Goodfire](https://www.goodfire.ai/)는 AI 안전성과 해석 가능성에 중점을 둔 연구소입니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-goodfire
```

```bash uv
uv add langchain-goodfire
```
</CodeGroup>
