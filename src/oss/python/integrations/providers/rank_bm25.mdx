[rank_bm25](https://github.com/dorianbrown/rank_bm25)는 문서를 쿼리하고 가장 관련성 높은 문서를 반환하도록 설계된 오픈소스 알고리즘 모음으로, 검색 엔진을 만드는 데 일반적으로 사용됩니다.

사용 가능한 알고리즘은 [프로젝트 페이지](https://github.com/dorianbrown/rank_bm25)를 참조하세요.


## Installation and Setup

먼저 `rank_bm25` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install rank_bm25
```

```bash uv
uv add rank_bm25
```
</CodeGroup>
