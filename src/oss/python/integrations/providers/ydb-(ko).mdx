YDB와 관련된 모든 기능입니다.

> [YDB](https://ydb.tech/)는 고가용성과 확장성을 강력한 일관성 및 ACID 트랜잭션과 결합한 다목적 오픈 소스 분산 SQL 데이터베이스입니다.
> 트랜잭션(OLTP), 분석(OLAP), 스트리밍 워크로드를 동시에 수용합니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-ydb
```

```bash uv
uv add langchain-ydb
```
</CodeGroup>

## Vector Store

YDB vector store를 import하려면:

```python
from langchain_ydb.vectorstores import YDB
```

YDB vector store에 대한 자세한 안내는 [이 노트북](/oss/integrations/vectorstores/ydb)을 참조하세요.
