>[DuckDB](https://duckdb.org/)는 in-process SQL OLAP 데이터베이스 관리 시스템입니다.

## Installation and Setup

먼저, `duckdb` python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install duckdb
```

```bash uv
uv add duckdb
```
</CodeGroup>
