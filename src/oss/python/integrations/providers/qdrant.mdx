>[Qdrant](https://qdrant.tech/documentation/) (읽기: quadrant)는 벡터 유사도 검색 엔진입니다.
> 벡터와 추가 payload를 포함하는 point를 저장, 검색 및 관리할 수 있는 편리한 API를 갖춘 프로덕션 준비 서비스를 제공합니다. `Qdrant`는 확장된 필터링 지원에 최적화되어 있습니다.


## Installation and Setup

Python partner package를 설치합니다:

<CodeGroup>
```bash pip
pip install langchain-qdrant
```

```bash uv
uv add langchaindrant
```
</CodeGroup>

## Embedding models

### FastEmbedSparse

```python
from langchain_qdrant import FastEmbedSparse
```

### SparseEmbeddings

```python
from langchain_qdrant import SparseEmbeddings
```

## Vector Store

`Qdrant` index를 감싸는 wrapper가 존재하여, 의미론적 검색이나 예제 선택을 위한 vectorstore로 사용할 수 있습니다.

이 vectorstore를 import하려면:
```python
from langchain_qdrant import QdrantVectorStore
```

Qdrant wrapper에 대한 자세한 안내는 [이 notebook](/oss/integrations/vectorstores/qdrant)을 참조하세요
