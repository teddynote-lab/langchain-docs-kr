>[UpTrain](https://uptrain.ai/)은 Generative AI 애플리케이션을 평가하고
>개선하기 위한 오픈소스 통합 플랫폼입니다. 20개 이상의 사전 구성된 평가
>(언어, 코드, embedding 사용 사례 포함)에 대한 등급을 제공하고, 실패
>사례에 대한 근본 원인 분석을 수행하며 이를 해결하는 방법에 대한 인사이트를 제공합니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install uptrain
```

```bash uv
uv add uptrain
```
</CodeGroup>
