>[YouTube](https://www.youtube.com/)는 Google의 온라인 동영상 공유 및 소셜 미디어 플랫폼입니다.
> `YouTube` transcript와 동영상 정보를 다운로드합니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install youtube-transcript-api
pip install pytube
```

```bash uv
uv add youtube-transcript-api
uv add pytube
```
</CodeGroup>
[사용 예시](/oss/integrations/document_loaders/youtube_transcript)를 참조하세요.
