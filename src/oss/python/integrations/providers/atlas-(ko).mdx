>[Nomic Atlas](https://docs.nomic.ai/index.html)는 소규모 및 인터넷 규모의 비정형 데이터셋과 상호작용하기 위한 플랫폼입니다.


## Installation and Setup

- `pip install nomic` 명령어로 Python 패키지를 설치하세요
- `Nomic`은 langchains poetry extras에도 포함되어 있습니다: `poetry install -E all`


## VectorStore

[사용 예제](/oss/integrations/vectorstores/atlas)를 참조하세요.

```python
from langchain_community.vectorstores import AtlasDB
```
