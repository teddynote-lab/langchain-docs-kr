[SingleStore](https://singlestore.com/)는 [클라우드](https://www.singlestore.com/cloud/) 및 온프레미스 환경 모두에서 뛰어난 성능을 발휘하도록 설계된 고성능 분산 SQL 데이터베이스입니다. 다양한 기능 세트, 원활한 배포 옵션, 탁월한 성능을 제공합니다.

이 integration은 SingleStore의 기능을 활용하기 위해 다음 component들을 제공합니다:

- **`SingleStoreLoader`**: SingleStore 데이터베이스 테이블에서 직접 document를 로드합니다.
- **`SingleStoreSemanticCache`**: embedding의 효율적인 저장 및 검색을 위한 semantic cache로 SingleStore를 사용합니다.
- **`SingleStoreChatMessageHistory`**: SingleStore에 채팅 메시지 기록을 저장하고 검색합니다.
- **`SingleStoreVectorStore`**: document embedding을 저장하고 빠른 vector 및 full-text 검색을 수행합니다.

이러한 component들은 효율적인 document 저장, embedding 관리, 고급 검색 기능을 지원하며, full-text와 vector 기반 검색을 결합하여 빠르고 정확한 쿼리를 제공합니다.

```python
from langchain_singlestore import (
    SingleStoreChatMessageHistory,
    SingleStoreLoader,
    SingleStoreSemanticCache,
    SingleStoreVectorStore,
)
```
