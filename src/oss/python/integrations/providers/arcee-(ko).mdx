>[Arcee](https://www.arcee.ai/about/about-us)는 우리가 SLM(Small, specialized, secure, and scalable language models)이라고 명명한
> 소형, 특화, 보안, 확장 가능한 언어 모델의 개발과 발전을 가능하게 합니다.
> SLM Adaptation System과 원활하고 안전한 통합을 제공함으로써,
> `Arcee`는 기업이 도메인에 적응된 언어 모델의 잠재력을 최대한 활용하여
> 운영에서 혁신적인 변화를 주도할 수 있도록 지원합니다.


## Installation and Setup

`Arcee API` key를 발급받으세요.


## LLMs

[사용 예시](/oss/integrations/llms/arcee)를 참조하세요.

```python
from langchain_community.llms import Arcee
```

[사용 예시](/oss/integrations/retrievers/arcee)를 참조하세요.
