>[KoNLPY](https://konlpy.org/)는 한국어 자연어 처리(NLP)를 위한 Python 패키지입니다.


## Installation and Setup

`konlpy` python 패키지를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install konlpy
```

```bash uv
uv add konlpy
```
</CodeGroup>

from langchain_text_splitters import KonlpyTextSplitter
