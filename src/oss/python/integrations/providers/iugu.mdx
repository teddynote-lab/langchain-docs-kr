>[Iugu](https://www.iugu.com/)는 브라질의 서비스 및 SaaS(Software as a Service) 
> 기업입니다. 전자상거래 웹사이트와 모바일 애플리케이션을 위한 결제 처리 소프트웨어 및 
> application programming interface를 제공합니다.


## Installation and Setup

`Iugu API`는 access token이 필요하며, 이는 `Iugu` 대시보드 내에서 찾을 수 있습니다.


## Document Loader

[사용 예시](/oss/integrations/document_loaders/iugu)를 참조하세요.

from langchain_community.document_loaders import IuguLoader
