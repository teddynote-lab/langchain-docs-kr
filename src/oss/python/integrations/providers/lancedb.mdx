이 페이지는 LangChain 내에서 [LanceDB](https://github.com/lancedb/lancedb)를 사용하는 방법을 다룹니다.
설치 및 설정, 그리고 특정 LanceDB wrapper에 대한 참조, 이렇게 두 부분으로 나뉩니다.

## 설치 및 설정

- `pip install lancedb` 명령어로 Python SDK를 설치하세요

## Wrappers

### VectorStore

LanceDB 데이터베이스를 감싸는 wrapper가 존재하며, 이를 통해 의미론적 검색이나 예제 선택을 위한 vectorstore로 사용할 수 있습니다.

이 vectorstore를 import하려면:

```python
from langchain_community.vectorstores import LanceDB
```

LanceDB wrapper에 대한 더 자세한 안내는 [이 노트북](/oss/integrations/vectorstores/lancedb)을 참조하세요
