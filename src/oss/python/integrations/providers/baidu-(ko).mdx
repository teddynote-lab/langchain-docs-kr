>[Baidu Cloud](https://cloud.baidu.com/)는 베이징에 본사를 둔 `Baidu, Inc.`에서 제공하는 클라우드 서비스입니다.
> 클라우드 스토리지 서비스, 클라이언트 소프트웨어,
> 파일 관리, 리소스 공유 및 Third Party Integration을 제공합니다.


## Installation and Setup

[여기](https://cloud.baidu.com/product/wenxinworkshop)에서 `Qianfan` `AK` 및 `SK` 키를 등록하고 받으세요.

## LLMs

### Baidu Qianfan

[사용 예제](/oss/integrations/llms/baidu_qianfan_endpoint)를 참조하세요.

```python
from langchain_community.llms import QianfanLLMEndpoint
```

## Chat models

### Qianfan Chat Endpoint
