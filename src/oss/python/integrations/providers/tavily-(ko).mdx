[Tavily](https://tavily.com)는 AI agent를 위해 특별히 설계된 검색 엔진입니다.
Tavily는 search와 extract API를 모두 제공하며, AI 개발자들은 실시간 온라인 정보를
애플리케이션에 손쉽게 통합할 수 있습니다. Tavily의 주요 목표는 신뢰할 수 있는 출처로부터
사실적이고 신뢰할 수 있는 정보를 제공하여 AI가 생성한 콘텐츠와 추론의 정확성과
신뢰성을 향상시키는 것입니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-tavily
```

```bash uv
uv add langchain-tavily
```
</CodeGroup>

## Tools

사용 가능한 tool에 대한 자세한 내용은 [tavily_search](/oss/integrations/tools/tavily_search)와 [tavily_extract](/oss/integrations/tools/tavily_extract)를 참조하세요.
