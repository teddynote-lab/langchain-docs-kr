---
title: HTML을 텍스트로 변환
---

>[html2text](https://github.com/Alir3z4/html2text/)는 `HTML` 페이지를 깔끔하고 읽기 쉬운 일반 `ASCII` 텍스트로 변환하는 Python 패키지입니다.

ASCII는 또한 유효한 `Markdown` (텍스트를 HTML로 변환하는 형식)이기도 합니다.

## 설치 및 설정

<CodeGroup>
```bash pip
pip install html2text
```

```bash uv
uv add html2text
```
</CodeGroup>

## Document Transformer

[사용 예제](/oss/integrations/document_transformers/html2text)를 참조하세요.
