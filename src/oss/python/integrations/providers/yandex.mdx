Yandex Cloud와 관련된 모든 기능

>[Yandex Cloud](https://cloud.yandex.com/en/)는 퍼블릭 클라우드 플랫폼입니다.

## Installation and Setup

Yandex Cloud SDK는 PyPI에서 pip를 통해 설치할 수 있습니다:

<CodeGroup>
```bash pip
pip install yandexcloud
```

```bash uv
uv add yandexcloud
```
</CodeGroup>

## LLMs

### YandexGPT

## Parser

### YandexSTTParser

오디오 파일을 전사하고 파싱합니다.

`YandexSTTParser`는 `OpenAIWhisperParser`와 유사합니다.
[OpenAIWhisperParser 사용 예제](/oss/integrations/document_loaders/youtube_audio)를 참조하세요.

```python
from langchain_community.document_loaders import YandexSTTParser
```
