>[Confident AI](https://confident-ai.com)는 `DeepEval`의 개발사입니다.
>
>[DeepEval](https://github.com/confident-ai/deepeval)은 LLM을 위한 단위 테스트 패키지입니다.
> `DeepEval`을 사용하면 단위 테스트와 통합 테스트를 모두 활용하여 더 빠른 반복을 통해 강력한 언어 모델을 구축할 수 있습니다. `DeepEval`은 합성 데이터 생성부터 테스트까지 반복 과정의 각 단계를 지원합니다.

## Installation and Setup

[DeepEval API credentials](https://app.confident-ai.com)를 받아야 합니다.

`DeepEval` Python 패키지를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install deepeval
```

```bash uv
uv add deepeval
```
</CodeGroup>
