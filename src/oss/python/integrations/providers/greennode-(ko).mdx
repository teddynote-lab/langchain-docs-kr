>**GreenNode**는 글로벌 AI 솔루션 제공업체이자 **NVIDIA Preferred Partner**로서, 미국, MENA, APAC 지역의 기업들에게 인프라부터 애플리케이션까지 풀스택 AI 역량을 제공합니다.
>**세계적 수준의 인프라**(LEED Gold, TIA‑942, Uptime Tier III)를 기반으로 운영되는 **GreenNode**는 기업, 스타트업, 연구자들에게 포괄적인 AI 서비스 제품군을 제공합니다:
>
>- [강력한 AI 인프라:](https://greennode.ai/) APAC 지역 최초의 하이퍼스케일 AI 클러스터 중 하나로, NVIDIA H100 GPU로 구동되는 GreenNode의 인프라는 고처리량 머신러닝 및 딥러닝 워크로드에 최적화되어 있습니다.
>- [GreenNode AI Platform:](https://greennode.ai/product/ai-platform) 기술 팀을 위해 설계된 GreenNode의 셀프서비스 AI 플랫폼은 최적화된 compute instance로 사전 구성된 Jupyter notebook 환경의 빠른 배포를 가능하게 합니다. 이 포털에서 개발자는 최소한의 설정 시간으로 ML 학습, fine-tuning, hyperparameter 최적화, inference 워크플로우를 시작할 수 있습니다. 플랫폼은 100개 이상의 엄선된 오픈소스 모델에 대한 액세스를 포함하며 일반적인 MLOps 도구 및 스토리지 프레임워크와의 통합을 지원합니다.
>- [GreenNode Serverless AI:](https://greennode.ai/product/model-as-a-service) GreenNode Serverless AI는 text gen, code gen, text to speech, speech to text, embedding 및 reranking 모델과 같은 도메인에 걸쳐 사전 학습된 프로덕션 준비 모델 라이브러리를 제공합니다. 이 서비스는 모델 인프라를 관리하지 않고 AI 솔루션을 프로토타입하거나 배포하려는 팀에게 이상적입니다.
>- [AI Applications:](https://vngcloud.vn/en/solution) 지능형 데이터 관리 및 문서 처리(IDP)부터 스마트 비디오 분석까지—GreenNode는 대규모 실제 AI 사용 사례를 지원합니다.
>다음 LLM 워크플로우를 구축하거나, AI 연구를 확장하거나, 엔터프라이즈급 애플리케이션을 배포하든, **GreenNode**는 여러분의 여정을 가속화할 도구와 인프라를 제공합니다.

## Installation and Setup

GreenNode integration은 pip를 통해 설치할 수 있습니다:

```python
pip install -qU langchain-greennode
```

```output
Note: you may need to restart the kernel to use updated packages.
```

### API Key

GreenNode Serverless AI를 사용하려면 [GreenNode Serverless AI](https://aiplatform.console.greennode.ai/api-keys)에서 얻을 수 있는 API key가 필요합니다. API key는 초기화 매개변수 `api_key`로 전달하거나 환경 변수 `GREENNODE_API_KEY`로 설정할 수 있습니다.

```python
import getpass
import os

if not os.getenv("GREENNODE_API_KEY"):
    os.environ["GREENNODE_API_KEY"] = getpass.getpass("Enter your GreenNode API key: ")
```

## Chat models

```python
from langchain_greennode import ChatGreenNode

chat = ChatGreenNode(
    model="deepseek-ai/DeepSeek-R1-Distill-Qwen-32B",  # Choose from available models
    temperature=0.6,
    top_p=0.95,
)
```

GreenNode [Chat Model](https://python.langchain.com/docs/integrations/chat/greennode/) 사용법

## Embedding models

```python
from langchain_greennode import GreenNodeEmbeddings

# Initialize embeddings
embeddings = GreenNodeEmbeddings(
    model="BAAI/bge-m3"  # Choose from available models
)
```

GreenNode [Embedding Model](https://python.langchain.com/docs/integrations/text_embedding/greennode) 사용법

## Rerank

```python
from langchain_greennode import GreenNodeRerank

# Initialize reranker
rerank = GreenNodeRerank(
    model="BAAI/bge-reranker-v2-m3",  # Choose from available models
    top_n=-1,
)
```

GreenNode [Rerank Model](https://python.langchain.com/docs/integrations/retrievers/greennode-reranker) 사용법
