>[The Apache Software Foundation (Wikipedia)](https://en.wikipedia.org/wiki/The_Apache_Software_Foundation)
> 은 개발자들의 분산형 오픈 소스 커뮤니티입니다. 이들이 생산하는 소프트웨어는
> 자유 및 오픈 소스 소프트웨어(FOSS)를 위한 허용적 오픈 소스 라이선스인
> Apache License의 조건에 따라 배포됩니다. Apache 프로젝트는
> 협업적이고 합의 기반의 개발 프로세스와 개방적이고 실용적인 소프트웨어 라이선스를
> 특징으로 하며, 이는 소프트웨어를 무료로 받은 개발자들이 비자유 조건으로
> 재배포할 수 있도록 허용한다는 것을 의미합니다. 각 프로젝트는 해당 프로젝트에
> 적극적으로 기여하는 기술 전문가들로 구성된 자체 선정 팀에 의해 관리됩니다.

## Apache AGE

>[Apache AGE](https://age.apache.org/)는 그래프 데이터베이스 기능을 제공하는
> `PostgreSQL` 확장입니다. `AGE`는 `A Graph Extension`의 약자이며,
> 멀티모달 데이터베이스인 Bitnine의 `PostgreSQL 10` 포크인 `AgensGraph`에서
> 영감을 받았습니다. 이 프로젝트의 목표는 관계형 및 그래프 모델 데이터를 모두
> 처리할 수 있는 단일 스토리지를 만들어 사용자가 표준 ANSI SQL과 그래프 쿼리
> 언어인 `openCypher`를 함께 사용할 수 있도록 하는 것입니다.
> `Apache AGE`가 저장하는 데이터 요소는 노드, 노드를 연결하는 엣지, 그리고
> 노드와 엣지의 속성입니다.

[Apache AGE와의 통합](/oss/integrations/graphs/apache_age)에 대해 자세히 알아보세요.

## Apache Cassandra

>[Apache Cassandra](https://cassandra.apache.org/)는 NoSQL, 행 지향,
> 높은 확장성과 가용성을 가진 데이터베이스입니다. 버전 5.0부터
> 데이터베이스는 벡터 검색 기능을 제공합니다.

[Apache Cassandra와의 통합](/oss/integrations/providers/cassandra/)에 대해 자세히 알아보세요.

## Apache Doris

>[Apache Doris](https://doris.apache.org/)는 실시간 분석을 위한 현대적인
> 데이터 웨어하우스입니다. 대규모 실시간 데이터에 대한 초고속 분석을 제공합니다.
>
>일반적으로 `Apache Doris`는 OLAP으로 분류되며, 분석 DBMS를 위한 벤치마크인
> ClickBench에서 뛰어난 성능을 보여주었습니다. 초고속 벡터화된 실행 엔진을
> 가지고 있어 빠른 vectordb로도 사용할 수 있습니다.

[Apache Doris와의 통합](/oss/integrations/providers/apache_doris/)에 대해 자세히 알아보세요.

## Apache Kafka

>[Apache Kafka](https://github.com/apache/kafka)는 레코드 스트림을 게시하고
> 구독하는 데 사용되는 분산 메시징 시스템입니다.


## Apache Spark

>[Apache Spark](https://spark.apache.org/)는 대규모 데이터 처리를 위한
> 통합 분석 엔진입니다. Scala, Java, Python, R에서 고수준 API를 제공하며,
> 데이터 분석을 위한 일반 계산 그래프를 지원하는 최적화된 엔진을 제공합니다.
> 또한 SQL 및 DataFrame을 위한 `Spark SQL`, pandas 워크로드를 위한
> `pandas API on Spark`, 머신러닝을 위한 `MLlib`, 그래프 처리를 위한
> `GraphX`, 스트림 처리를 위한 `Structured Streaming`을 포함한 풍부한
> 고수준 도구 세트를 지원합니다.

[Apache Spark와의 통합](/oss/integrations/providers/spark)에 대해 자세히 알아보세요.
