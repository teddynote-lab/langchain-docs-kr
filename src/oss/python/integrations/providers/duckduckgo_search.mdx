>[DuckDuckGo Search](https://github.com/deedy5/duckduckgo_search)는 `DuckDuckGo.com` 검색 엔진을 사용하여 단어, 문서, 이미지, 비디오, 뉴스, 지도 및 텍스트 번역을 검색하는 패키지입니다. 파일과 이미지를 로컬 하드 드라이브에 다운로드합니다.

## Installation and Setup

python 패키지를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install duckduckgo-search
```

```bash uv
uv add duckduckgo-search
```
</CodeGroup>

## Tools

[사용 예제](/oss/integrations/tools/ddg)를 참조하세요.

사용 가능한 두 가지 tool이 있습니다:

```python
from langchain_community.tools import DuckDuckGoSearchRun
```
