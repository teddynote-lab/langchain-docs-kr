이 페이지는 LangChain 내에서 [MariaDB](https://github.com/mariadb/) 생태계를 사용하는 방법을 다룹니다.
설치 및 설정, 그리고 특정 PGVector wrapper에 대한 참조, 이렇게 두 부분으로 나뉩니다.

## Installation
- c/c connector 설치

Debian, Ubuntu의 경우
```bash
sudo apt install libmariadb3 libmariadb-dev
```

CentOS, RHEL, Rocky Linux의 경우
```bash
sudo yum install MariaDB-shared MariaDB-devel
```

- `pip install mariadb`로 Python connector package를 설치합니다


## Setup
1. 첫 번째 단계는 MariaDB 11.7.1 이상이 설치되어 있어야 합니다.

    docker image가 시작하기 가장 쉬운 방법입니다.

## Wrappers

### VectorStore

MariaDB vector database를 감싸는 wrapper가 존재하며, 이를 통해 semantic search나 example selection을 위한 vectorstore로 사용할 수 있습니다.

이 vectorstore를 import하려면:
```python
from langchain_mariadb import MariaDBStore
```
