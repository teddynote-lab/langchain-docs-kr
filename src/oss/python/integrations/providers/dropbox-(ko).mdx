>[Dropbox](https://en.wikipedia.org/wiki/Dropbox)는 기존 파일, 클라우드 콘텐츠, 웹 바로가기를 한 곳에 모아주는 파일 호스팅 서비스입니다.


## Installation and Setup

자세한 [설치 가이드](/oss/integrations/document_loaders/dropbox#prerequisites)를 참조하세요.

<CodeGroup>
```bash pip
pip install -U dropbox
```

```bash uv
uv add dropbox
```
</CodeGroup>

[사용 예제](/oss/integrations/document_loaders/dropbox)를 참조하세요.
