>[E2B](https://e2b.dev/)는 AI가 생성한 코드 실행을 위한 오픈소스 보안 샌드박스를 제공합니다. 자세한 내용은 [여기](https://github.com/e2b-dev)를 참조하세요.

## 설치 및 설정

python 패키지를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install e2b_code_interpreter
```

```bash uv
uv add e2b_code_interpreter
```
</CodeGroup>
