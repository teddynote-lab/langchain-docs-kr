>[Confluence](https://www.atlassian.com/software/confluence)는 프로젝트 관련 자료를 모두 저장하고 정리하는 위키 협업 플랫폼입니다. `Confluence`는 주로 콘텐츠 관리 활동을 처리하는 지식 베이스입니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install atlassian-python-api
```

```bash uv
uv add atlassian-python-api
```
</CodeGroup>

`username/api_key` 또는 `Oauth2 login`을 설정해야 합니다.
[지침](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)을 참조하세요.
