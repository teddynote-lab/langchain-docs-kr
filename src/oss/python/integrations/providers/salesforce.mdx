[Salesforce](https://www.salesforce.com/)는 고객 관계 관리(CRM) 솔루션과 영업, 고객 서비스, 마케팅 자동화 및 분석에 중점을 둔 엔터프라이즈 애플리케이션 제품군을 제공하는 클라우드 기반 소프트웨어 회사입니다.

[langchain-salesforce](https://pypi.org/project/langchain-salesforce/)는 LLM이 Salesforce 데이터와 상호작용할 수 있도록 하는 도구를 구현합니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-salesforce
```

```bash uv
uv add langchain-salesforce
```
</CodeGroup>
