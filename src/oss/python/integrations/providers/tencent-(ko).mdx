>[Tencent Holdings Ltd. (Wikipedia)](https://en.wikipedia.org/wiki/Tencent) (중국어: 腾讯; 병음: Téngxùn)
> 는 선전에 본사를 둔 중국의 다국적 기술 대기업이자 지주 회사입니다. `Tencent`는 매출 기준으로
> 세계에서 가장 높은 수익을 올리는 멀티미디어 기업 중 하나입니다. 또한 지분 투자를 기준으로
> 비디오 게임 산업에서 세계 최대 기업입니다.


## Chat model

>[Tencent의 하이브리드 모델 API](https://cloud.tencent.com/document/product/1729) (`Hunyuan API`)
> 는 대화 커뮤니케이션, 콘텐츠 생성, 분석 및 이해를 구현하며, 지능형 고객 서비스, 지능형 마케팅,
> 역할 연기, 광고, 카피라이팅, 제품 설명, 스크립트 제작, 이력서 생성, 기사 작성, 코드 생성,
> 데이터 분석 및 콘텐츠 분석과 같은 다양한 시나리오에 광범위하게 사용될 수 있습니다.


자세한 내용은 [이 노트북](/oss/integrations/chat/tencent_hunyuan)을 참조하세요

```python
from langchain_community.chat_models import ChatHunyuan
```


## Document loaders

### Tencent COS

>[Tencent Cloud Object Storage (COS)](https://www.tencentcloud.com/products/cos)는 HTTP/HTTPS 프로토콜을 통해
> 어디서나 모든 양의 데이터를 저장할 수 있는 분산 스토리지 서비스입니다.
> `COS`는 데이터 구조나 형식에 제한이 없습니다. 또한 버킷 크기 제한과 파티션 관리가 없어
> 데이터 전송, 데이터 처리, 데이터 레이크와 같은 거의 모든 사용 사례에 적합합니다. COS는 웹 기반 콘솔,
> 다국어 SDK 및 API, 명령줄 도구, 그래픽 도구를 제공합니다. Amazon S3 API와 잘 작동하여
> 커뮤니티 도구 및 플러그인에 빠르게 액세스할 수 있습니다.

Python SDK 설치:

<CodeGroup>
```bash pip
pip install cos-python-sdk-v5
```

```bash uv
uv add cos-python-sdk-v5
```
</CodeGroup>

#### Tencent COS Directory

## Vector Store

### Tencent VectorDB

>[Tencent Cloud VectorDB](https://www.tencentcloud.com/products/vdb)는 완전 관리형,
> 자체 개발 엔터프라이즈급 분산 데이터베이스 서비스로
>다차원 벡터 데이터의 저장, 검색 및 분석에 전념합니다. 이 데이터베이스는 다양한 인덱스
>유형과 유사도 계산 방법을 지원하며, 단일 인덱스는 10억 개의 벡터, 수백만 QPS 및
>밀리초 쿼리 지연 시간을 지원합니다. `Tencent Cloud Vector Database`는 대규모
>모델에 외부 지식 베이스를 제공하고 대규모 모델의 답변 정확도를 향상시킬 뿐만 아니라
>추천 시스템, NLP 서비스, 컴퓨터 비전 및 지능형 고객 서비스와 같은 AI 분야에 광범위하게 사용될 수 있습니다.

Python SDK 설치:

<CodeGroup>
```bash pip
pip install tcvectordb
```

```bash uv
uv add tcvectordb
```
</CodeGroup>

자세한 내용은 [이 노트북](/oss/integrations/vectorstores/tencentvectordb)을 참조하세요

```python
from langchain_community.vectorstores import TencentVectorDB
```
