>[Slack](https://slack.com/)은 인스턴트 메시징 프로그램입니다.

## Installation and Setup

특별한 설정이 필요하지 않습니다.


## Document loader

[사용 예시](/oss/integrations/document_loaders/slack)를 참조하세요.

```python
from langchain_community.document_loaders import SlackDirectoryLoader
```

## Toolkit

[사용 예시](/oss/integrations/tools/slack)를 참조하세요.

```python
from langchain_community.agent_toolkits import SlackToolkit
```

## Chat loader

[사용 예시](/oss/integrations/chat_loaders/slack)를 참조하세요.

```python
from langchain_community.chat_loaders.slack import SlackChatLoader
```
