LangChain은 [bookend.ai](https://bookend.ai/)에서 제공하는 embedding과의 통합을 구현합니다.


## Installation and Setup


[bookend.ai](https://bookend.ai/) 웹사이트에서 등록하고 `API_KEY`를 받아야 합니다.

## Embedding model

[사용 예제](/oss/integrations/text_embedding/bookend)를 참조하세요.

```python
from langchain_community.embeddings import BookendEmbeddings
```
