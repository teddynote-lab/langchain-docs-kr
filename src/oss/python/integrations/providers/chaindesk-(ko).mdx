>[Chaindesk](https://chaindesk.ai)는 개인 데이터를 Large Language Model과 연결하는 데 도움을 주는 [오픈소스](https://github.com/gmpetrov/databerry) 문서 검색 플랫폼입니다.


## Installation and Setup

Chaindesk에 가입하고, datastore를 생성하고, 일부 데이터를 추가한 후 datastore api endpoint url을 가져와야 합니다.
[API Key](https://docs.chaindesk.ai/api-reference/authentication)가 필요합니다.

## Retriever

[사용 예시](/oss/integrations/retrievers/chaindesk)를 참조하세요.

```python
from langchain.retrievers import ChaindeskRetriever
```
