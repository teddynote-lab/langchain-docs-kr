>`FalkorDB`란 무엇인가요?

>- FalkorDB는 graph database 기술을 전문으로 하는 `오픈소스 데이터베이스 관리 시스템`입니다.
>- FalkorDB를 사용하면 데이터를 node와 edge로 표현하고 저장할 수 있어, 연결된 데이터와 관계를 처리하는 데 이상적입니다.
>- FalkorDB는 독자적인 확장 기능을 갖춘 OpenCypher query language를 지원하여, graph 데이터와 쉽게 상호작용하고 쿼리할 수 있습니다.
>- FalkorDB를 사용하면 프로덕션 수준의 시스템에 적합한 고성능 `graph traversal 및 query`를 수행할 수 있습니다.

>FalkorDB를 시작하려면 [공식 웹사이트](https://docs.falkordb.com/)를 방문하세요.

## Installation and Setup

- `pip install falkordb langchain-falkordb` 명령어로 Python SDK를 설치하세요

## VectorStore

FalkorDB vector index는 semantic search나 example selection을 위한 vectorstore로 사용됩니다.

```python
from langchain_community.vectorstores.falkordb_vector import FalkorDBVector
```

또는

```python
from langchain_falkordb.vectorstore import FalkorDBVector
```

[사용 예제](/oss/integrations/vectorstores/falkordbvector.ipynb)를 참조하세요
