>[Baichuan Inc.](https://www.baichuan-ai.com/)는 AGI 시대의 중국 스타트업으로,
> 인류의 근본적인 요구사항인 효율성, 건강, 행복을 해결하는 데 전념하고 있습니다.


## Installation and Setup

[여기](https://platform.baichuan-ai.com/)에서 등록하고 API key를 받으세요.

## LLMs

[사용 예제](/oss/integrations/llms/baichuan)를 참조하세요.

```python
from langchain_community.llms import BaichuanLLM
```

## Chat models

[사용 예제](/oss/integrations/chat/baichuan)를 참조하세요.

```python
from langchain_community.chat_models import ChatBaichuan
```
