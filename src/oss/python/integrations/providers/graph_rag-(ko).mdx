## 개요

[Graph RAG](https://datastax.github.io/graph-rag/)는 vector에 대한 **비구조화된** 유사도 검색과 메타데이터 속성의 **구조화된** 탐색을 결합한 retriever 인터페이스를 제공합니다. 이를 통해 **기존** vector store에 대한 그래프 기반 검색이 가능합니다.

## 설치 및 설정

<CodeGroup>
```bash pip
pip install langchain-graph-retriever
```

```bash uv
uv add langchain-graph-retriever
```
</CodeGroup>

## Retrievers

```python
from langchain_graph_retriever import GraphRetriever
```
