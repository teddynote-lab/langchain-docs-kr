>[pandas](https://pandas.pydata.org)는 `Python` 프로그래밍 언어 기반으로 구축된 빠르고, 강력하며, 유연하고 사용하기 쉬운 오픈 소스 데이터 분석 및 조작 도구입니다.

## Installation and Setup

`pip`을 사용하여 `pandas` 패키지를 설치하세요:

<CodeGroup>
```bash pip
pip install pandas
```

```bash uv
uv add pandas
```
</CodeGroup>
