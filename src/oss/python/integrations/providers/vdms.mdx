> [VDMS](https://github.com/IntelLabs/vdms/blob/master/README.md)는 그래프로 저장된 visual metadata를 통해 관련 visual 데이터를 검색하고 더 빠른 액세스를 위해 visual 데이터에 대한 머신 친화적인 개선을 가능하게 함으로써 클라우드 규모를 달성하는 것을 목표로 하는 대용량 "visual" 데이터의 효율적인 액세스를 위한 스토리지 솔루션입니다.

## Installation and Setup

### Install Client

<CodeGroup>
```bash pip
pip install langchain-vdms
```

```bash uv
uv add langchain-vdms
```
</CodeGroup>

### Install Database

VDMS를 시작하는 두 가지 방법이 있습니다:


1. docker를 통해 로컬 머신에 VDMS 설치
    ```bash
        docker run -d -p 55555:55555 intellabs/vdms:latest
    ```

2. 로컬 머신에 직접 VDMS 설치. [설치 지침](https://github.com/IntelLabs/vdms/blob/master/INSTALL.md)을 참조하세요.

## VectorStore

이 vectorstore를 import하려면:

```python
from langchain_vdms import VDMS
from langchain_vdms.vectorstores import VDMS
```
VDMS Client connector를 import하려면:

```python
from langchain_vdms.vectorstores import VDMS_Client
```

VDMS wrapper에 대한 자세한 안내는 [이 가이드](/oss/integrations/vectorstores/vdms)를 참조하세요.
