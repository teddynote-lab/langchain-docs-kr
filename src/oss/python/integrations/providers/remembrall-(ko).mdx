>[Remembrall](https://remembrall.dev/)은 언어 모델에게
> 장기 기억, retrieval augmented generation, 그리고 완전한 관찰 가능성을 제공하는 플랫폼입니다.

## Installation and Setup

시작하려면 [Remembrall 플랫폼에서 Github으로 로그인](https://remembrall.dev/login)하고
[설정 페이지에서 API key를 복사](https://remembrall.dev/dashboard/settings)하세요.
