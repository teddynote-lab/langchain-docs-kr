>[Chroma](https://docs.trychroma.com/getting-started)는 embedding을 사용하여 AI 애플리케이션을 구축하기 위한 데이터베이스입니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-chroma
```

```bash uv
uv add langchain-chroma
```
</CodeGroup>


## VectorStore

Chroma vector database를 감싸는 wrapper가 존재하며, 이를 통해 semantic search나 example selection을 위한 vectorstore로 사용할 수 있습니다.

```python
from langchain_chroma import Chroma
```

Chroma wrapper에 대한 자세한 안내는 [이 노트북](/oss/integrations/vectorstores/chroma)을 참조하세요.
