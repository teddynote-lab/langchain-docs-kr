>[Wikipedia](https://wikipedia.org/)는 Wikipedians로 알려진 자원봉사자 커뮤니티가 개방형 협업과 MediaWiki라는 wiki 기반 편집 시스템을 통해 작성하고 유지 관리하는 다국어 무료 온라인 백과사전입니다. `Wikipedia`는 역사상 가장 크고 가장 많이 읽히는 참고 자료입니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install wikipedia
```

```bash uv
uv add wikipedia
```
</CodeGroup>
