> [Infinispan](https://infinispan.org) Infinispan은 Java 객체부터 일반 텍스트까지 모든 유형의 데이터를 저장할 수 있는 key/value 데이터 저장소를 제공하는 오픈 소스 인메모리 데이터 그리드입니다.
> 버전 15부터 Infinispan은 캐시에 대한 vector search를 지원합니다.

## Installation and Setup
Infinispan 서버를 실행하려면 [Get Started](https://infinispan.org/get-started/)를 참조하세요. 인증을 비활성화할 수 있습니다
(현재 지원되지 않음)

from langchain_community.vectorstores import InfinispanVS
