> [Polaris AI DataInsight](https://datainsight.polarisoffice.com/playground)는 다양한 파일 형식에서 문서 요소(텍스트, 이미지, 복잡한 표, 차트 등)를 구조화된 JSON으로 추출하여 RAG 시스템에 쉽게 통합할 수 있도록 하는 문서 파서입니다.

## Installation and Setup

```bash
pip install langchain-polaris-ai-datainsight
```

## Document Loader

[사용 예제](/oss/integrations/document_loaders/polaris_ai_datainsight)를 참조하세요.

```
from langchain_polaris_ai_datainsight import PolarisAIDataInsightLoader
```
