>[MariTalk](https://www.maritaca.ai/en)은 브라질의 요구사항을 충족하도록 훈련된 LLM 기반 챗봇입니다.

## Installation and Setup

MariTalk API key를 받아야 합니다.

또한 `httpx` Python package를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install httpx
```

```bash uv
uv add httpx
```
</CodeGroup>

## Chat models

[사용 예제](/oss/integrations/chat/maritalk)를 참조하세요.

from langchain_community.chat_models import ChatMaritalk
