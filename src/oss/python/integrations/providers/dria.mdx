>[Dria](https://dria.co/)는 개발자들이 공유 embedding lake에 기여하고 활용할 수 있는 
> 공개 RAG 모델 허브입니다.

Dria와 LangChain 통합에 대한 자세한 내용은 
[이 페이지](https://dria.co/docs/integrations/langchain)에서 확인하세요.

## Installation and Setup

python package를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install dria
```

```bash uv
uv add dria
```
</CodeGroup>

Dria에서 API key를 발급받아야 합니다. [Dria](https://dria.co/)에서 가입하여 발급받을 수 있습니다.

## Retrievers
