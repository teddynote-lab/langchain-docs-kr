>[Docusaurus](https://docusaurus.io/)는 즉시 사용 가능한 문서화 기능을 제공하는 정적 사이트 생성기입니다.


## Installation and Setup


<CodeGroup>
```bash pip
pip install -U beautifulsoup4 lxml
```

```bash uv
uv add beautifulsoup4 lxml
```
</CodeGroup>
