> [Forefront AI](https://forefront.ai/)는 오픈소스 텍스트 생성 모델을 
> 파인튜닝하고 추론할 수 있는 플랫폼입니다


## Installation and Setup

[이 페이지](https://accounts.forefront.ai/sign-in?redirect_url=https%3A%2F%2Fforefront.ai%2Fapp%2Fapi-keys)를 방문하여 `ForefrontAI` API key를 받고,
환경 변수(`FOREFRONTAI_API_KEY`)로 설정하세요.

## LLM

[사용 예제](/oss/integrations/llms/forefrontai)를 참조하세요.

```python
from langchain_community.llms import ForefrontAI
```
