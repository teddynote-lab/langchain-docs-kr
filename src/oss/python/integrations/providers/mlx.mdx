>[MLX](https://ml-explore.github.io/mlx/build/html/index.html)는 `Apple` 실리콘에서 효율적이고 유연한 머신러닝을 위해 설계된 `NumPy`와 유사한 배열 프레임워크로,
> `Apple machine learning research`에서 제공합니다.


## Installation and Setup

여러 Python 패키지를 설치합니다:

<CodeGroup>
```bash pip
pip install mlx-lm transformers huggingface_hub
```

```bash uv
uv add mlx-lm transformers huggingface_hub
```
</CodeGroup>


## Chat models


[사용 예제](/oss/integrations/chat/mlx)를 참조하세요.
