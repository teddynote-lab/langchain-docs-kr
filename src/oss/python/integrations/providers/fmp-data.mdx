> [FMP-Data](https://pypi.org/project/fmp-data/)는 Financial Data Prep API에 연결하기 위한 python 패키지입니다. 프로덕션 품질의 데이터에 액세스하는 방법을 간소화합니다.


## Installation and Setup

[이 페이지](https://site.financialmodelingprep.com/pricing-plans?couponCode=mehdi)를 방문하여 `FMP Data` API key를 받고, 환경 변수(`FMP_API_KEY`)로 설정하세요.

그런 다음 [langchain-fmp-data](https://pypi.org/project/langchain-fmp-data/)를 설치하세요.

## Tools

[예제](https://github.com/MehdiZare/langchain-fmp-data/tree/main/docs)를 참조하세요.

```python
from langchain_fmp_data import FMPDataTool, FMPDataToolkit
```
