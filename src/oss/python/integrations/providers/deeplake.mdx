[Deeplake](https://www.deeplake.ai/)는 AI 및 딥러닝 애플리케이션에 최적화된 데이터베이스입니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install langchain-deeplake
```

```bash uv
uv add langchain-deeplake
```
</CodeGroup>
