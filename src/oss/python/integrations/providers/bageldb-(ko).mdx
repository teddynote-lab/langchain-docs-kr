> [BagelDB](https://www.bageldb.ai/) (`AI를 위한 오픈 Vector Database`)는 AI 데이터를 위한 GitHub와 같습니다.
사용자가 vector dataset을 생성하고, 공유하며, 관리할 수 있는 협업 플랫폼입니다. 독립 개발자를 위한 개인 프로젝트,
기업을 위한 내부 협업, 그리고 데이터 DAO를 위한 공개 기여를 지원할 수 있습니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install betabageldb
```

```bash uv
uv add betabageldb
```
</CodeGroup>
