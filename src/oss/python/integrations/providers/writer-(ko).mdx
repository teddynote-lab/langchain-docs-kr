WRITER와 관련된 모든 기능


>이 페이지는 LangChain 내에서 [WRITER](https://writer.com/) 생태계를 사용하는 방법을 다룹니다. 자세한 내용은 Writer [문서](https://dev.writer.com/home/introduction)를 참조하세요.
>[Palmyra](https://writer.com/blog/palmyra/)는 `WRITER`가 개발한 Large Language Model (LLM)입니다.
>
>[Writer API](https://dev.writer.com/api-guides/introduction)는 다양한 기능과 가격대를 가진 여러 Palmyra 하위 모델로 구동됩니다.

## 설치 및 설정

통합 패키지를 설치합니다
<CodeGroup>
```bash pip
pip install langchain-writer
```

```bash uv
uv add langchain-writer
```
</CodeGroup>

WRITER API key를 발급받아 환경 변수(`WRITER_API_KEY`)로 설정합니다
