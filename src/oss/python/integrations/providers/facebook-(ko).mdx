>[Meta Platforms, Inc.](https://www.facebook.com/)는 `Meta`라는 이름으로 사업을 운영하며, 이전에는
> `Facebook, Inc.` 및 `TheFacebook, Inc.`로 명명되었던 미국의 다국적 기술
> 대기업입니다. 이 회사는 `Facebook`, `Instagram`, `Threads`,
> `WhatsApp` 등의 제품과 서비스를 소유하고 운영합니다.

## Embedding models

### LASER

>[LASER](https://github.com/facebookresearch/LASER)는 `Meta AI Research` 팀이 개발한
> Python 라이브러리로,
> [2024년 2월 25일 기준 147개 이상의 언어](https://github.com/facebookresearch/flores/blob/main/flores200/README.md#languages-in-flores-200)에 대한
> 다국어 문장 임베딩을 생성하는 데 사용됩니다.

<CodeGroup>
```bash pip
pip install laser_encoders
```

```bash uv
uv add laser_encoders
```
</CodeGroup>

[사용 예제](/oss/integrations/text_embedding/laser)를 참조하세요.

```python
from langchain_community.embeddings.laser import LaserEmbeddings
```

## Vector stores

### Facebook Faiss

>[Facebook AI Similarity Search (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/)는
> 밀집 벡터의 효율적인 유사도 검색 및 클러스터링을 위한 라이브러리입니다. RAM에 맞지 않을 수 있는 크기를 포함하여
> 모든 크기의 벡터 집합을 검색하는 알고리즘을 포함합니다. 또한 평가 및 파라미터 튜닝을 위한
> 지원 코드도 포함되어 있습니다.

[Faiss 문서](https://faiss.ai/).

`faiss` python 패키지를 설치해야 합니다.

<CodeGroup>
```bash pip
pip install faiss-gpu # For CUDA 7.5+ supported GPU's.
```

```bash uv
uv add faiss-gpu # For CUDA 7.5+ supported GPUs.
```
</CodeGroup>

또는

<CodeGroup>
```bash pip
pip install faiss-cpu # For CPU Installation
```

```bash uv
uv add faiss-cpu # For CPU Installation
```
</CodeGroup>

[사용 예제](/oss/integrations/vectorstores/faiss)를 참조하세요.

```python
from langchain_community.vectorstores import FAISS
```
