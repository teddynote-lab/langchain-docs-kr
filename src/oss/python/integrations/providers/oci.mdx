[Oracle Cloud Infrastructure](https://www.oracle.com/artificial-intelligence/)와 관련된 `LangChain` 통합입니다.

## OCI Generative AI
> Oracle Cloud Infrastructure (OCI) [Generative AI](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm)는 다양한 사용 사례를 다루는 최첨단의 맞춤형 대규모 언어 모델(LLM) 세트를 제공하는 완전 관리형 서비스이며, 단일 API를 통해 사용할 수 있습니다.
> OCI Generative AI 서비스를 사용하면 바로 사용 가능한 사전 학습된 모델에 액세스하거나, 전용 AI 클러스터에서 자체 데이터를 기반으로 미세 조정된 커스텀 모델을 생성하고 호스팅할 수 있습니다.

사용하려면 최신 `oci` python SDK와 langchain_community 패키지가 설치되어 있어야 합니다.

<CodeGroup>
```bash pip
pip install -U oci langchain-community
```

```bash uv
uv add oci langchain-community
```
</CodeGroup>

[chat](/oss/integrations/llms/oci_generative_ai), [complete](/oss/integrations/chat/oci_generative_ai), [embedding](/oss/integrations/text_embedding/oci_generative_ai) 사용 예제를 참조하세요.

```python
from langchain_community.chat_models import ChatOCIGenAI

from langchain_community.llms import OCIGenAI

from langchain_community.embeddings import OCIGenAIEmbeddings
```

## OCI Data Science Model Deployment Endpoint

> [OCI Data Science](https://docs.oracle.com/en-us/iaas/data-science/using/home.htm)는 데이터 과학 팀을 위한 완전 관리형 서버리스 플랫폼입니다. OCI Data Science 플랫폼을 사용하면 머신러닝 모델을 구축, 학습 및 관리한 다음, [OCI Data Science Model Deployment Service](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm)를 사용하여 OCI Model Deployment Endpoint로 배포할 수 있습니다.

사용하려면 최신 `oracle-ads` python SDK가 설치되어 있어야 합니다.

<CodeGroup>
```bash pip
pip install -U oracle-ads
```

```bash uv
uv add oracle-ads
```
</CodeGroup>

[chat](/oss/integrations/chat/oci_data_science) 및 [complete](/oss/integrations/llms/oci_model_deployment_endpoint) 사용 예제를 참조하세요.


```python
from langchain_community.chat_models import ChatOCIModelDeployment

from langchain_community.llms import OCIModelDeploymentLLM
```
