>[Rockset](https://rockset.com/product/)은 대규모로 낮은 지연 시간과 높은 동시성 분석 쿼리를 제공하는 실시간 분석 데이터베이스 서비스입니다. 구조화 및 반구조화 데이터에 대한 Converged Index™를 구축하며 벡터 임베딩을 위한 효율적인 저장소를 제공합니다. 스키마 없는 데이터에서 SQL을 실행할 수 있는 지원 기능은 메타데이터 필터를 사용한 벡터 검색을 실행하는 데 완벽한 선택입니다.

## Installation and Setup

Rockset 계정이 있는지 확인하고 웹 콘솔로 이동하여 API key를 받으세요. 자세한 내용은 [웹사이트](https://rockset.com/docs/rest-api/)에서 확인할 수 있습니다.

<CodeGroup>
```bash pip
pip install rockset
```

```bash uv
uv add rockset
```
</CodeGroup>
