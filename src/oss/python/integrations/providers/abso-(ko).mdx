[Abso](https://abso.ai/#router)는 프롬프트 복잡도에 따라 빠른 모델과 느린 모델 간에 요청을 자동으로 라우팅하는 오픈소스 LLM proxy입니다. 다양한 휴리스틱을 사용하여 적절한 모델을 선택합니다. 매우 빠르고 지연 시간이 짧습니다.

## Installation and setup

```bash
pip install langchain-abso
```
