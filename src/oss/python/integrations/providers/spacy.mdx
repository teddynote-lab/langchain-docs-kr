>[spaCy](https://spacy.io/)는 Python과 Cython 프로그래밍 언어로 작성된 고급 자연어 처리를 위한 오픈소스 소프트웨어 라이브러리입니다.

## Installation and Setup


<CodeGroup>
```bash pip
pip install spacy
```

```bash uv
uv add spacy
```
</CodeGroup>



## Text Splitter

```python
from langchain_text_splitters import SpacyTextSplitter
```
