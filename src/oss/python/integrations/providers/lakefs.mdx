>[lakeFS](https://docs.lakefs.io/)는 데이터 레이크에 대한 확장 가능한 버전 관리를 제공하며,
> Git과 유사한 의미 체계를 사용하여 해당 버전을 생성하고 액세스합니다.

## Installation and Setup

`ENDPOINT`, `LAKEFS_ACCESS_KEY`, `LAKEFS_SECRET_KEY`를 가져옵니다.
설치 지침은 [여기](https://docs.lakefs.io/quickstart/launch.html)에서 확인할 수 있습니다.


## Document Loader

from langchain_community.document_loaders import LakeFSLoader
