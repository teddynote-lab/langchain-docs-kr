>[Context](https://context.ai/)는 LLM 기반 제품 및 기능을 위한 사용자 분석을 제공합니다.

## Installation and Setup

`context-python` Python package를 설치해야 합니다:

<CodeGroup>
```bash pip
pip install context-python
```

```bash uv
uv add context-python
```
</CodeGroup>


## Callbacks

[사용 예제](/oss/integrations/callbacks/context)를 참조하세요.

```python
from langchain.callbacks import ContextCallbackHandler
```
