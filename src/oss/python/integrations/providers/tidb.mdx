> [TiDB Cloud](https://www.pingcap.com/tidb-serverless)는 전용 및 serverless 옵션을 제공하는 포괄적인 Database-as-a-Service (DBaaS) 솔루션입니다. `TiDB Serverless`는 이제 MySQL 생태계에 내장 vector search를 통합하고 있습니다. 이 향상으로 별도의 데이터베이스나 추가 기술 스택 없이도 `TiDB Serverless`를 사용해 AI 애플리케이션을 원활하게 개발할 수 있습니다. 무료 TiDB Serverless 클러스터를 생성하고 https://pingcap.com/ai 에서 vector search 기능을 사용해 보세요.


## 설치 및 설정

TiDB 데이터베이스의 연결 정보를 확보해야 합니다.
[TiDB Cloud](https://tidbcloud.com/)를 방문하여 연결 정보를 확인하세요.

```python
# TiDB에서 데이터를 로드하기 위한 Document Loader import
from langchain_community.document_loaders import TiDBLoader
```

```python
# TiDB VectorStore 연동을 위한 VectorStore import
from langchain_community.vectorstores import TiDBVectorStore
```

자세한 내용은 [여기](/oss/integrations/vectorstores/tidb_vector)를 참고하세요.
