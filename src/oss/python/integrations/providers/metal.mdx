이 페이지는 LangChain에서 [Metal](https://getmetal.io)을 사용하는 방법을 다룹니다.

## Metal이란?

Metal은 프로덕션 환경을 위해 구축된 관리형 검색 및 메모리 플랫폼입니다. `Metal`에 데이터를 쉽게 인덱싱하고 시맨틱 검색 및 검색을 실행할 수 있습니다.

## 빠른 시작

[Metal 계정 생성](https://app.getmetal.io/signup)으로 시작하세요.

그런 다음 `MetalRetriever` class를 쉽게 활용하여 시맨틱 검색, 프롬프트 컨텍스트 등을 위한 데이터 검색을 시작할 수 있습니다. 이 class는 `Metal` 인스턴스와 Metal API에 전달할 매개변수 dictionary를 받습니다.

```python
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
```
