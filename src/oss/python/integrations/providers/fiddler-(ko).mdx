>[Fiddler](https://www.fiddler.ai/)는 엔터프라이즈 규모에서 ML 배포를 모니터링, 설명, 분석 및 개선할 수 있는 통합 플랫폼을 제공합니다.

## Installation and Setup

[Fiddler와 함께](https://demo.fiddler.ai) 모델을 설정하세요:

* Fiddler에 연결하는 데 사용하는 URL
* 조직 ID
* 인증 토큰

Python 패키지를 설치하세요:

<CodeGroup>
```bash pip
pip install fiddler-client
```

```bash uv
uv add fiddler-client
```
</CodeGroup>

from langchain_community.callbacks.fiddler_callback import FiddlerCallbackHandler
