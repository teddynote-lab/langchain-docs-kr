>[Zilliz Cloud](https://zilliz.com/doc/quick_start)는 `LF AI Milvus®`를 위한 완전 관리형 클라우드 서비스입니다.


## Installation and Setup

Python SDK를 설치합니다:
<CodeGroup>
```bash pip
pip install pymilvus
```

```bash uv
uv add pymilvus
```
</CodeGroup>

## Vectorstore

Zilliz index를 감싸는 wrapper를 사용하면 semantic search나 example selection을 위한 vectorstore로 활용할 수 있습니다.

```python
from langchain_community.vectorstores import Milvus
```

Milvus wrapper에 대한 자세한 안내는 [이 노트북](/oss/integrations/vectorstores/zilliz)을 참조하세요
