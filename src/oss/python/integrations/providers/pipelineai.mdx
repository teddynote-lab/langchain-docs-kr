이 페이지는 LangChain 내에서 PipelineAI 생태계를 사용하는 방법을 다룹니다.
설치 및 설정, 그리고 특정 PipelineAI wrapper에 대한 참조, 이렇게 두 부분으로 나뉩니다.

## Installation and Setup

- `pip install pipeline-ai`로 설치합니다
- Pipeline Cloud api key를 발급받아 환경 변수(`PIPELINE_API_KEY`)로 설정합니다

## Wrappers

### LLM

PipelineAI LLM wrapper가 존재하며, 다음과 같이 접근할 수 있습니다

```python
from langchain_community.llms import PipelineAI
```
