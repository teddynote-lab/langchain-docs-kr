>[Aleph Alpha](https://docs.aleph-alpha.com/)는 강력한 AI 시대를 위한 기반 기술을 연구하고 구축하는 사명으로 2019년에 설립되었습니다. 국제적인 과학자, 엔지니어, 혁신가로 구성된 팀은 대규모 언어 및 멀티모달 모델과 같은 혁신적인 AI를 연구, 개발 및 배포하며 유럽에서 가장 빠른 상용 AI 클러스터를 운영하고 있습니다.

>[Luminous 시리즈](https://docs.aleph-alpha.com/docs/introduction/luminous/)는 대규모 언어 모델 제품군입니다.

## Installation and Setup

<CodeGroup>
```bash pip
pip install aleph-alpha-client
```

```bash uv
uv add aleph-alpha-client
```
</CodeGroup>

새로운 토큰을 생성해야 합니다. [지침](https://docs.aleph-alpha.com/docs/account/#create-a-new-token)을 참조하세요.

```python
from getpass import getpass

ALEPH_ALPHA_API_KEY = getpass()
```


## LLM

[사용 예제](/oss/integrations/llms/aleph_alpha)를 참조하세요.

```python
from langchain_community.llms import AlephAlpha
```
