>[Twitter](https://twitter.com/)는 온라인 소셜 미디어 및 소셜 네트워킹 서비스입니다.


## Installation and Setup

<CodeGroup>
```bash pip
pip install tweepy
```

```bash uv
uv add tweepy
```
</CodeGroup>

`Twitter API` token으로 loader를 초기화해야 하며, Twitter `username`을 설정해야 합니다.


## Document Loader

[사용 예시](/oss/integrations/document_loaders/twitter)를 참조하세요.

```python
from langchain_community.document_loaders import TwitterTweetLoader
```
