### FirestoreVectorStore ì´ˆê¸°í™”

`FirestoreVectorStore`ë¥¼ ì‚¬ìš©í•˜ë©´ Firestore ë°ì´í„°ë² ì´ìŠ¤ì— ìƒˆ ë²¡í„°ë¥¼ ì €ì¥í•  ìˆ˜ ìˆìŠµë‹ˆë‹¤. Google Generative AIì˜ embeddingsë¥¼ í¬í•¨í•˜ì—¬ ëª¨ë“  ëª¨ë¸ì˜ embeddingsë¥¼ ì €ì¥í•˜ëŠ” ë° ì‚¬ìš©í•  ìˆ˜ ìˆìŠµë‹ˆë‹¤.

```python
from langchain_google_firestore import FirestoreVectorStore
from langchain_google_vertexai import VertexAIEmbeddings

embedding = VertexAIEmbeddings(
    model_name="textembedding-gecko@latest",
    project=PROJECT_ID,
)

# Sample data
ids = ["apple", "banana", "orange"]
fruits_texts = ['{"name": "apple"}', '{"name": "banana"}', '{"name": "orange"}']

# Create a vector store
vector_store = FirestoreVectorStore(
    collection="fruits",
    embedding=embedding,
)

# Add the fruits to the vector store
vector_store.add_texts(fruits_texts, ids=ids)
```
