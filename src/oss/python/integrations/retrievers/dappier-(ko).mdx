[Dappier](https://dappier.com)는 모든 LLM 또는 Agentic AI를 신뢰할 수 있는 출처의 실시간, 권리가 확보된 독점 데이터에 연결하여 AI를 모든 분야의 전문가로 만들어줍니다. 우리의 특화된 모델에는 실시간 웹 검색, 뉴스, 스포츠, 금융 주식 시장 데이터, 암호화폐 데이터, 그리고 프리미엄 퍼블리셔의 독점 콘텐츠가 포함됩니다. [marketplace.dappier.com](https://marketplace.dappier.com)의 마켓플레이스에서 다양한 데이터 모델을 탐색해보세요.

[Dappier](https://dappier.com)는 LangChain과의 원활한 통합을 위해 최적화된 풍부하고, 프롬프트 준비가 완료된, 문맥적으로 관련성 있는 데이터 문자열을 제공합니다. 대화형 AI, 추천 엔진 또는 지능형 검색을 구축하든, Dappier의 LLM 독립적인 RAG 모델은 자체 검색 파이프라인을 구축하고 관리하는 복잡성 없이 검증되고 최신 데이터에 대한 액세스를 보장합니다.

# DappierRetriever

이 문서는 Dappier [retriever](https://python.langchain.com/docs/concepts/retrievers/)를 시작하는 데 도움이 됩니다. 모든 DappierRetriever 기능 및 구성에 대한 자세한 문서는 [API reference](https://python.langchain.com/en/latest/retrievers/langchain_dappier.retrievers.Dappier.DappierRetriever.html)를 참조하세요.

### Setup

`langchain-dappier`를 설치하고 환경 변수 `DAPPIER_API_KEY`를 설정합니다.

```bash
pip install -U langchain-dappier
export DAPPIER_API_KEY="your-api-key"
```

또한 [Dappier 사이트](https://platform.dappier.com/profile/api-keys)에서 생성할 수 있는 Dappier API 자격 증명을 설정해야 합니다.

[Dappier 마켓플레이스](https://platform.dappier.com/marketplace)로 이동하여 지원되는 데이터 모델을 찾을 수 있습니다.

개별 쿼리에서 자동화된 추적을 원하는 경우, 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API 키를 설정할 수도 있습니다:

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### Installation

이 retriever는 `langchain-dappier` 패키지에 있습니다:

```python
pip install -qU langchain-dappier
```

## Instantiation

- data_model_id: str
    dm_으로 시작하는 데이터 모델 ID입니다.
    사용 가능한 데이터 모델 ID는 다음에서 찾을 수 있습니다:
    [Dappier 마켓플레이스](https://platform.dappier.com/marketplace)
- k: int
    반환할 문서 수입니다.
- ref: Optional[str]
    AI 추천이 표시되는 사이트 도메인입니다.
- num_articles_ref: int
    지정된 ref 도메인의 최소 기사 수입니다.
    나머지는 RAG 모델 내의 다른 사이트에서 가져옵니다.
- search_algorithm: Literal[
    "most_recent",
    "most_recent_semantic",
    "semantic",
    "trending"
]
    기사 검색을 위한 검색 알고리즘입니다.
- api_key: Optional[str]
    Dappier API와 상호 작용하는 데 사용되는 API 키입니다.

```python
from langchain_dappier import DappierRetriever

retriever = DappierRetriever(data_model_id="dm_01jagy9nqaeer9hxx8z1sk1jx6")
```

```python
query = "latest tech news"

retriever.invoke(query)
```
