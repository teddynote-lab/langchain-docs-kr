[KoboldAI](https://github.com/KoboldAI/KoboldAI-Client)는 "여러 로컬 및 원격 AI 모델을 사용하는 AI 지원 글쓰기를 위한 브라우저 기반 프론트엔드"입니다. langchain에서 사용할 수 있는 public 및 local API를 제공합니다.

이 예제는 LangChain에서 해당 API를 사용하는 방법을 다룹니다.

문서는 endpoint 끝에 /api를 추가하여 브라우저에서 확인할 수 있습니다 (예: [127.0.0.1/:5000/api](http://127.0.0.1/:5000/api)).

```python
from langchain_community.llms import KoboldApiLLM
```

아래에 표시된 endpoint를 --api 또는 --public-api로 webui를 시작한 후 출력에 표시된 것으로 교체하세요

선택적으로 temperature 또는 max_length와 같은 parameter를 전달할 수 있습니다

```python
llm = KoboldApiLLM(endpoint="http://192.168.1.144:5000", max_length=80)
```

```python
response = llm.invoke(
    "### Instruction:\nWhat is the first book of the bible?\n### Response:"
)
```
