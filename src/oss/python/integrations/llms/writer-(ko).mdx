[WRITER](https://writer.com/)는 다양한 언어 콘텐츠를 생성하는 플랫폼입니다.

이 예제는 LangChain을 사용하여 `WRITER` [models](https://dev.writer.com/docs/models)와 상호작용하는 방법을 다룹니다.

## Setup

WRITER model에 액세스하려면 WRITER 계정을 생성하고, API key를 받고, `writer-sdk`와 `langchain-community` package를 설치해야 합니다.

### Credentials

[WRITER AI Studio](https://app.writer.com/aistudio/signup?utm_campaign=devrel)로 이동하여 WRITER에 가입하고 API key를 생성하세요. 완료한 후 WRITER_API_KEY environment variable을 설정하세요:

```python
import getpass
import os

if not os.environ.get("WRITER_API_KEY"):
    os.environ["WRITER_API_KEY"] = getpass.getpass("Enter your Writer API key:")
```

## Installation

LangChain WRITER integration은 `langchain-community` package에 있습니다:

```python
pip install -qU langchain-community writer-sdk
```

```output
[notice] A new release of pip is available: 24.2 -> 24.3.1
[notice] To update, run: pip install -U pip
Note: you may need to restart the kernel to use updated packages.
```

이제 writer LLM과 상호작용하기 위해 model object를 초기화할 수 있습니다

```python
from langchain_community.llms import Writer as WriterLLM

llm = WriterLLM(
    temperature=0.7,
    max_tokens=1000,
    # other params...
)
```

## Invocation

```python
response_text = llm.invoke(input="Write a poem")
```

```python
print(response_text)
```

## Streaming

```python
stream_response = llm.stream(input="Tell me a fairytale")
```

```python
for chunk in stream_response:
    print(chunk, end="")
```

## Async

WRITER는 **ainvoke()** 및 **astream()** method를 통한 비동기 호출을 지원합니다
