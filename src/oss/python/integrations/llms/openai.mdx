[OpenAI](https://platform.openai.com/docs/introduction)는 다양한 작업에 적합한 여러 수준의 성능을 가진 다양한 model을 제공합니다.

이 예제는 LangChain을 사용하여 OpenAI [model](https://platform.openai.com/docs/models)과 상호작용하는 방법을 다룹니다.

## Setup

OpenAI model에 액세스하려면 OpenAI 계정을 생성하고, API key를 발급받고, `langchain-openai` integration package를 설치해야 합니다.

### Credentials

[platform.openai.com](https://platform.openai.com)으로 이동하여 OpenAI에 가입하고 API key를 생성하세요. 완료한 후 OPENAI_API_KEY environment variable을 설정하세요:

```python
import getpass
import os

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass.getpass("Enter your OpenAI API key: ")
```

model 호출의 자동 추적을 활성화하려면 [LangSmith](https://docs.smith.langchain.com/) API key를 설정하세요:

```python
os.environ["LANGSMITH_API_KEY"] = getpass.getpass("Enter your LangSmith API key: ")
os.environ["LANGSMITH_TRACING"] = "true"
```

### Installation

LangChain OpenAI integration은 `langchain-openai` package에 있습니다:

```python
pip install -qU langchain-openai
```

organization ID를 지정해야 하는 경우 다음 셀을 사용할 수 있습니다. 그러나 단일 organization에만 속해 있거나 기본 organization을 사용하려는 경우에는 필수가 아닙니다. 기본 organization은 [여기](https://platform.openai.com/account/api-keys)에서 확인할 수 있습니다.

organization을 지정하려면 다음을 사용할 수 있습니다:

```python
OPENAI_ORGANIZATION = getpass()

os.environ["OPENAI_ORGANIZATION"] = OPENAI_ORGANIZATION
```

## Instantiation

이제 model 객체를 인스턴스화하고 chat completion을 생성할 수 있습니다:

```python
from langchain_openai import OpenAI

llm = OpenAI()
```

## Invocation

```python
llm.invoke("Hello how are you?")
```

```output
'\n\nI am an AI and do not have emotions like humans do, so I am always functioning at my optimal level. Thank you for asking! How can I assist you today?'
```

## Chaining

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("How to say {input} in {output_language}:\n")

chain = prompt | llm
chain.invoke(
    {
        "output_language": "German",
        "input": "I love programming.",
    }
)
```

```output
'\nIch liebe Programmieren.'
```

## Using a proxy

명시적인 proxy 뒤에 있는 경우, 통과할 http_client를 지정할 수 있습니다:

```python
pip install httpx

import httpx

openai = OpenAI(
    model_name="gpt-3.5-turbo-instruct",
    http_client=httpx.Client(proxies="http://proxy.yourcompany.com:8080"),
)
```
