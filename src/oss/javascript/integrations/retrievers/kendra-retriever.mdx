## Overview

[Amazon Kendra](https://aws.amazon.com/kendra/)는 Amazon Web Services(AWS)에서 제공하는 지능형 검색 서비스입니다.
고급 자연어 처리(NLP) 및 머신러닝 알고리즘을 활용하여 조직 내 다양한 데이터 소스에서 강력한 검색 기능을 제공합니다.
Kendra는 사용자가 필요한 정보를 빠르고 정확하게 찾을 수 있도록 설계되어 생산성과 의사결정을 개선합니다.

Kendra를 사용하면 문서, FAQ, 지식 베이스, 매뉴얼, 웹사이트 등 다양한 콘텐츠 유형을 검색할 수 있습니다.
여러 언어를 지원하며 복잡한 쿼리, 동의어, 문맥적 의미를 이해하여 매우 관련성 높은 검색 결과를 제공합니다.

이 문서는 Amazon Kendra [`retriever`](/oss/langchain/retrieval) 시작하기를 도와드립니다. 모든 `AWSKendraRetriever` 기능 및 구성에 대한 자세한 문서는 [API reference](https://api.js.langchain.com/classes/langchain_aws.AmazonKendraRetriever.html)를 참조하세요.

## Setup

시작하려면 AWS 계정과 Amazon Kendra 인스턴스가 필요합니다. 자세한 내용은 AWS의 이 [튜토리얼](https://docs.aws.amazon.com/kendra/latest/dg/getting-started.html)을 참조하세요.

개별 쿼리에서 자동화된 추적을 원하는 경우, 아래 주석을 해제하여 [LangSmith](https://docs.smith.langchain.com/) API key를 설정할 수도 있습니다:

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

## Instantiation

이제 retriever를 인스턴스화할 수 있습니다:

```typescript
import { AmazonKendraRetriever } from "@langchain/aws";

const retriever = new AmazonKendraRetriever({
  topK: 10,
  indexId: "YOUR_INDEX_ID",
  region: "us-east-2", // Your region
  clientOptions: {
    credentials: {
      accessKeyId: "YOUR_ACCESS_KEY_ID",
      secretAccessKey: "YOUR_SECRET_ACCESS_KEY",
    },
  },
});
```

## Usage

```typescript
const query = "..."

await retriever.invoke(query);
```
