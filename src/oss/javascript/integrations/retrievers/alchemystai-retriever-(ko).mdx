# Alchemyst AI Retriever

[**Alchemyst AI Retriever**](https://getalchemystai.com)는 생성형 AI 애플리케이션이 관련 컨텍스트와 지식을 검색할 수 있도록 합니다. Alchemyst 플랫폼에서 이러한 정보를 가져옵니다. LLM 및 에이전트 응답을 향상시키기 위해 데이터에 액세스하고, 검색하고, 가져오는 통합 인터페이스를 제공합니다.

## Setup

1. 계정이 없다면 Alchemyst Platform에서 [새 계정에 가입](https://platform.getalchemystai.com/signup)하세요.
2. 로그인 후 [**Alchemyst Platform Settings**](https://platform.getalchemystai.com/settings)로 이동하여 API key를 받으세요.

<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/langchain/install)을 참조하세요.

## Usage

```typescript
import { AlchemystRetriever } from "@alchemystai/langchain-js";
import { RunnableSequence } from "@langchain/core/runnables";
import dotenv from "dotenv";

dotenv.config();

// Instantiate the retriever with your API key and optional config
const retriever = new AlchemystRetriever({
  apiKey: process.env.ALCHEMYST_AI_API_KEY!,
  similarityThreshold: 0.8,
  minimumSimilarityThreshold: 0.5,
  scope: "internal"
});

// Example: Use the retriever in a LangChain pipeline
async function main() {
  // Create a simple pipeline that retrieves documents and outputs their content
  const pipeline = RunnableSequence.from([
    async (input: string) => {
      const docs = await retriever.getRelevantDocuments(input);
      return docs.map(doc => doc.pageContent).join("\n---\n");
    }
  ]);

  const query = "Show me the latest HR policies"; // Put your business/practical query here
  const result = await pipeline.invoke(query);

  console.log("Retrieved Documents:\n", result);
}

main().catch(console.error);
```
