---
title: Voyage AI
---

`VoyageEmbeddings` class는 Voyage AI REST API를 사용하여 주어진 텍스트에 대한 embedding을 생성합니다.

`inputType` parameter를 사용하면 더 나은 embedding 결과를 위해 입력 텍스트의 유형을 지정할 수 있습니다. `query`, `document`로 설정하거나, 정의하지 않을 수 있습니다(이는 `None`과 동일합니다).

- `query`: 검색 또는 retrieval query에 사용합니다. Voyage AI는 query 사용 사례에 맞게 embedding을 최적화하기 위해 prompt를 앞에 추가합니다.
- `document`: 검색 가능하게 만들고자 하는 문서나 콘텐츠에 사용합니다. Voyage AI는 document 사용 사례에 맞게 embedding을 최적화하기 위해 prompt를 앞에 추가합니다.
- `None` (기본값): 입력 텍스트가 추가 prompt 없이 직접 인코딩됩니다.

또한, 이 class는 embedding 프로세스를 추가로 커스터마이징할 수 있는 새로운 parameter들을 지원합니다:

- **truncation**: 입력 텍스트를 model이 허용하는 최대 길이로 자를지 여부입니다.
- **outputDimension**: 출력 embedding의 원하는 차원입니다.
- **outputDtype**: 출력 embedding의 데이터 타입입니다. `"float"` 또는 `"int8"`이 될 수 있습니다.
- **encodingFormat**: 출력 embedding의 형식입니다. `"float"`, `"base64"`, 또는 `"ubinary"`가 될 수 있습니다.

```typescript
import { VoyageEmbeddings } from "@langchain/community/embeddings/voyage";

const embeddings = new VoyageEmbeddings({
  apiKey: "YOUR-API-KEY", // In Node.js defaults to process.env.VOYAGEAI_API_KEY
  inputType: "document", // Optional: specify input type as 'query', 'document', or omit for None / Undefined / Null
  truncation: true, // Optional: enable truncation of input texts
  outputDimension: 768, // Optional: set desired output embedding dimension
  outputDtype: "float", // Optional: set output data type ("float" or "int8")
  encodingFormat: "float", // Optional: set output encoding format ("float", "base64", or "ubinary")
});
```
