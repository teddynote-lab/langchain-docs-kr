이 Embeddings integration은 [TensorFlow.js](https://www.tensorflow.org/js)를 사용하여 브라우저나 Node.js 환경에서 embedding을 완전히 실행합니다. 즉, 데이터가 제3자에게 전송되지 않으며 API key에 가입할 필요가 없습니다. 하지만 다른 integration에 비해 더 많은 메모리와 처리 능력이 필요합니다.

```bash npm
npm install @langchain/community @langchain/core @tensorflow/tfjs-core@3.6.0 @tensorflow/tfjs-converter@3.6.0 @tensorflow-models/universal-sentence-encoder@1.3.3 @tensorflow/tfjs-backend-cpu
```
```typescript
import "@tensorflow/tfjs-backend-cpu";
import { TensorFlowEmbeddings } from "@langchain/community/embeddings/tensorflow";

const embeddings = new TensorFlowEmbeddings();
```

이 예제는 모든 JS 환경에서 작동하는 CPU backend를 사용합니다. 하지만 GPU 및 WebAssembly를 포함하여 TensorFlow.js에서 지원하는 모든 backend를 사용할 수 있으며, 이는 훨씬 더 빠릅니다. Node.js의 경우 `@tensorflow/tfjs-node` package를 사용할 수 있고, 브라우저의 경우 `@tensorflow/tfjs-backend-webgl` package를 사용할 수 있습니다. 자세한 내용은 [TensorFlow.js documentation](https://www.tensorflow.org/js/guide/platform_environment)을 참조하세요.
