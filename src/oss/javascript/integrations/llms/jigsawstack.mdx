---
title: JigsawStack Prompt Engine
---

LangChain.js는 JigsawStack [Prompt Engine](https://docs.jigsawstack.com/api-reference/prompt-engine/run-direct) LLM 호출을 지원합니다.

## Setup

- [계정](https://jigsawstack.com/dashboard)을 설정하세요 (무료로 시작 가능)
- [API key](https://jigsawstack.com/dashboard)를 생성하고 가져오세요

## Credentials

```bash
export JIGSAWSTACK_API_KEY="your-api-key"
```

## Usage

<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/jigsawstack
```
```ts
import { JigsawStackPromptEngine } from "@langchain/jigsawstack";

export const run = async () => {
  const model = new JigsawStackPromptEngine();
  const res = await model.invoke(
    "Tell me about the leaning tower of pisa?\nAnswer:"
  );
  console.log({ res });
};
```
