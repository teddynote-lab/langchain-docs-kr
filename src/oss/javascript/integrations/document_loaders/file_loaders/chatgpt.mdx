title: ChatGPT 파일
---

이 예제는 ChatGPT 데이터 내보내기 폴더에서 conversations.json을 로드하는 방법을 다룹니다. 데이터 내보내기는 다음 경로를 통해 이메일로 받을 수 있습니다: ChatGPT -> (프로필) - Settings -> Export data -> Confirm export -> 이메일 확인.

## 사용법, 모든 로그 추출하기

예제 코드:

```typescript
import { ChatGPTLoader } from "@langchain/community/document_loaders/fs/chatgpt";

const loader = new ChatGPTLoader("./example_data/example_conversations.json");

const docs = await loader.load();

console.log(docs);
```

## 사용법, 단일 로그 추출하기

예제 코드:

```typescript
import { ChatGPTLoader } from "@langchain/community/document_loaders/fs/chatgpt";

const loader = new ChatGPTLoader(
  "./example_data/example_conversations.json",
  1
);

const docs = await loader.load();

console.log(docs);
```
