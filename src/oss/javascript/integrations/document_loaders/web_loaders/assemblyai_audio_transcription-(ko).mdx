---
title: AssemblyAI Audio Transcript
---

[AssemblyAI API](https://www.assemblyai.com/docs/api-reference/transcripts/submit?utm_source=langchainjs)를 사용하여 파일에서 오디오(및 비디오) transcript를 document 객체로 로드하는 방법을 다룹니다.

## 사용법

먼저, 공식 AssemblyAI package를 설치해야 합니다:

```bash npm
npm install @langchain/community @langchain/core assemblyai
```

loader를 사용하려면 [AssemblyAI 계정](https://www.assemblyai.com/dashboard/signup?utm_source=langchainjs)이 필요하며
[대시보드에서 AssemblyAI API key를 받아야 합니다](https://www.assemblyai.com/app/account?utm_source=langchainjs).

그런 다음, API key를 `ASSEMBLYAI_API_KEY` environment variable 또는 `apiKey` options parameter로 설정합니다.

```typescript
import {
  AudioTranscriptLoader,
  // AudioTranscriptParagraphsLoader,
  // AudioTranscriptSentencesLoader
} from "@langchain/community/document_loaders/web/assemblyai";

// You can also use a local file path and the loader will upload it to AssemblyAI for you.
const audioUrl = "https://storage.googleapis.com/aai-docs-samples/espn.m4a";

// Use `AudioTranscriptParagraphsLoader` or `AudioTranscriptSentencesLoader` for splitting the transcript into paragraphs or sentences
const loader = new AudioTranscriptLoader(
  {
    audio: audioUrl,
    // any other parameters as documented here: https://www.assemblyai.com/docs/api-reference/transcripts/submit
  },
  {
    apiKey: "<ASSEMBLYAI_API_KEY>", // or set the `ASSEMBLYAI_API_KEY` env variable
  }
);
const docs = await loader.load();
console.dir(docs, { depth: Infinity });
```

> ** info **
>
> - `AudioTranscriptParagraphsLoader` 또는 `AudioTranscriptSentencesLoader`를 사용하여 transcript를 단락 또는 문장으로 분할할 수 있습니다.
> - `audio` parameter는 URL, 로컬 파일 경로, buffer 또는 stream이 될 수 있습니다.
> - `audio`는 비디오 파일일 수도 있습니다. [FAQ 문서에서 지원되는 파일 형식 목록](https://www.assemblyai.com/docs/concepts/faq?utm_source=langchainjs#:~:text=file%20types%20are%20supported)을 참조하세요.
> - `apiKey` option을 전달하지 않으면, loader는 `ASSEMBLYAI_API_KEY` environment variable을 사용합니다.
> - `audio` 외에 추가 속성을 추가할 수 있습니다. [AssemblyAI API 문서](https://www.assemblyai.com/docs/api-reference/transcripts/submit?utm_source=langchainjs#create-a-transcript)에서 전체 request parameter 목록을 확인하세요.

`AudioSubtitleLoader`를 사용하여 `srt` 또는 `vtt` 자막을 document로 가져올 수도 있습니다.

```typescript
import { AudioSubtitleLoader } from "@langchain/community/document_loaders/web/assemblyai";

// You can also use a local file path and the loader will upload it to AssemblyAI for you.
const audioUrl = "https://storage.googleapis.com/aai-docs-samples/espn.m4a";

const loader = new AudioSubtitleLoader(
  {
    audio: audioUrl,
    // any other parameters as documented here: https://www.assemblyai.com/docs/api-reference/transcripts/submit
  },
  "srt", // srt or vtt
  {
    apiKey: "<ASSEMBLYAI_API_KEY>", // or set the `ASSEMBLYAI_API_KEY` env variable
  }
);

const docs = await loader.load();
console.dir(docs, { depth: Infinity });
```
