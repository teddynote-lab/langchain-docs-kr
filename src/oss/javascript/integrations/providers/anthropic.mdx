Anthropic 모델과 관련된 모든 기능..

[Anthropic](https://www.anthropic.com/)은 AI 안전성 및 연구 회사이며, Claude의 창시자입니다.
이 페이지는 Anthropic 모델과 LangChain 간의 모든 통합을 다룹니다.

## Prompting 모범 사례

Anthropic 모델은 OpenAI 모델과 비교하여 몇 가지 prompting 모범 사례가 있습니다.

**System Message는 첫 번째 메시지여야 합니다**

Anthropic 모델은 모든 system message가 프롬프트의 첫 번째 메시지여야 합니다.

## `ChatAnthropic`

`ChatAnthropic`은 LangChain의 `ChatModel`의 하위 클래스로, `ChatPromptTemplate`과 함께 사용할 때 가장 잘 작동합니다.
다음 코드로 이 wrapper를 import할 수 있습니다:

<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/anthropic @langchain/core
```
```typescript
import { ChatAnthropic } from "@langchain/anthropic";
const model = new ChatAnthropic({});
```

ChatModel을 사용할 때는 프롬프트를 `ChatPromptTemplate`으로 설계하는 것이 좋습니다.
다음은 그 예시입니다:

```typescript
import { ChatPromptTemplate } from "@langchain/classic/prompts";

const prompt = ChatPromptTemplate.fromMessages([
  ["system", "You are a helpful chatbot"],
  ["human", "Tell me a joke about {topic}"],
]);
```

그런 다음 다음과 같이 chain에서 사용할 수 있습니다:

```typescript
const chain = prompt.pipe(model);
await chain.invoke({ topic: "bears" });
```

multimodal input을 포함한 더 많은 예제는 [chat model integration 페이지](/oss/integrations/chat/anthropic/)를 참조하세요.
