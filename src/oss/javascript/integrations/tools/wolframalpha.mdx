title: WolframAlpha Tool
---

WolframAlpha tool은 agent와 chain을 WolframAlpha의 최첨단 계산 지능 엔진에 연결합니다.

## Setup

[WolframAlpha portal](https://developer.wolframalpha.com/)에서 앱을 생성하고 `appid`를 발급받아야 합니다.

## Usage

```typescript
import { WolframAlphaTool } from "@langchain/community/tools/wolframalpha";

const tool = new WolframAlphaTool({
  appid: "YOUR_APP_ID",
});

const res = await tool.invoke("What is 2 * 2?");

console.log(res);
```
