---
title: Google Trends Tool
---

Google Trends Tool은 에이전트가 SerpApi의 Google Trends API를 활용하여 검색 관심도 데이터를 검색하고 분석할 수 있도록 합니다.
이는 트렌딩 주제, 지역별 검색 관심도, 검색어의 과거 인기도를 이해하는 데 유용할 수 있습니다.

API 세부 정보는 [여기](https://serpapi.com/google-trends-api)를 참조하세요.

SerpApi는 쿼리를 캐시하므로 첫 번째 쿼리는 느리지만 이후 동일한 쿼리는 빠르게 처리됩니다.
때때로 관련 쿼리는 작동하지 않을 수 있지만 시간별 관심도는 정상적으로 작동합니다. [여기](https://serpapi.com/playground?engine=google_trends&q=monster&data_type=RELATED_QUERIES)에서 쿼리를 확인할 수 있습니다.

## Setup

이 도구를 사용하려면 SerpApi의 Google Trends API에 대한 액세스를 구성해야 합니다.

[SerpApi](https://serpapi.com/users/sign_in)에서 API key를 받으세요.

그런 다음 API key를 `process.env.SERPAPI_API_KEY`로 설정하거나 `apiKey` constructor argument로 전달하세요.

## Usage

<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/openai @langchain/community @langchain/core
```

```typescript
import { SERPGoogleTrendsTool } from "@langchain/community/tools/google_trends";

export async function run() {
  const tool = new SERPGoogleTrendsTool();

  const res = await tool.invoke("Monster");

  console.log(res);
}
```
