전체 문서는 여기에서 확인하세요: https://nla.zapier.com/start/

**Zapier Natural Language Actions**는 자연어 API 인터페이스를 통해 Zapier 플랫폼의 5,000개 이상의 앱과 20,000개 이상의 액션에 접근할 수 있게 해줍니다.

NLA는 Gmail, Salesforce, Trello, Slack, Asana, HubSpot, Google Sheets, Microsoft Teams 등 수천 개의 앱을 지원합니다: https://zapier.com/apps

Zapier NLA는 모든 기본 API 인증과 자연어 --> 기본 API 호출 --> LLM을 위한 간소화된 출력 반환의 변환을 처리합니다. 핵심 아이디어는 사용자 또는 귀하의 사용자가 oauth와 유사한 설정 창을 통해 일련의 액션을 노출하고, 이를 REST API를 통해 쿼리하고 실행할 수 있다는 것입니다.

NLA는 NLA API 요청 서명을 위해 API Key와 OAuth를 모두 제공합니다.

서버 측(API Key): 빠른 시작, 테스트 및 LangChain이 개발자의 Zapier 계정에 노출된 액션만 사용하는 프로덕션 시나리오(그리고 Zapier.com에서 개발자의 연결된 계정을 사용)

사용자 대면(OAuth): 최종 사용자 대면 애플리케이션을 배포하고 LangChain이 Zapier.com에서 최종 사용자의 노출된 액션과 연결된 계정에 접근해야 하는 프로덕션 시나리오

환경 변수(`ZAPIER_NLA_OAUTH_ACCESS_TOKEN` 또는 `ZAPIER_NLA_API_KEY`)를 통해 NLA 자격 증명을 연결하거나 `ZapierNLAWrapper`의 API 참조에서 params 인수를 참조하세요.

자세한 내용은 [인증 문서](https://docs.zapier.com/platform/build/auth)를 검토하세요.

아래 예제는 Zapier 통합을 Agent로 사용하는 방법을 보여줍니다:

<Tip>
LangChain 패키지 설치에 대한 일반적인 지침은 [이 섹션](/oss/langchain/install)을 참조하세요.
</Tip>

```bash npm
npm install @langchain/openai @langchain/core
```
```typescript
import { OpenAI } from "@langchain/openai";
import { ZapierNLAWrapper } from "@langchain/classic/tools";
import {
  initializeAgentExecutorWithOptions,
  ZapierToolKit,
} from "@langchain/classic/agents";

const model = new OpenAI({ temperature: 0 });
const zapier = new ZapierNLAWrapper();
const toolkit = await ZapierToolKit.fromZapierNLAWrapper(zapier);

const executor = await initializeAgentExecutorWithOptions(
  toolkit.tools,
  model,
  {
    agentType: "zero-shot-react-description",
    verbose: true,
  }
);
console.log("Loaded agent.");

const input = `Summarize the last email I received regarding Silicon Valley Bank. Send the summary to the #test-zapier Slack channel.`;

console.log(`Executing with input "${input}"...`);

const result = await executor.invoke({ input });

console.log(`Got output ${result.output}`);
```
