## Setup

DeepSeek 모델에 액세스하려면 DeepSeek 계정을 생성하고 API 키를 받은 다음 `@langchain/deepseek` integration package를 설치해야 합니다.

[Together AI](/oss/integrations/chat/togetherai) 또는 [Ollama](/oss/integrations/chat/ollama)와 같은 provider를 통해 DeepSeek API에 액세스할 수도 있습니다.

### Credentials

https://deepseek.com/ 에 접속하여 DeepSeek에 가입하고 API 키를 생성하세요. 완료한 후 `DEEPSEEK_API_KEY` environment variable을 설정하세요:

```bash
export DEEPSEEK_API_KEY="your-api-key"
```

모델 호출에 대한 자동 추적을 원하시면 아래 주석을 해제하여 [LangSmith](/langsmith/home) API 키를 설정할 수도 있습니다:

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### Installation

LangChain ChatDeepSeek integration은 `@langchain/deepseek` package에 있습니다:

</CodeGroup>

## Instantiation

이제 model object를 인스턴스화하고 chat completion을 생성할 수 있습니다:


```typescript
import { ChatDeepSeek } from "@langchain/deepseek";

const llm = new ChatDeepSeek({
  model: "deepseek-reasoner",
  temperature: 0,
  // other params...
})
```


```typescript
const aiMsg = await llm.invoke([
  [
    "system",
    "You are a helpful assistant that translates English to French. Translate the user sentence.",
  ],
  ["human", "I love programming."],
])
aiMsg
```
