[LangSmith UI](https://smith.langchain.com)에서 OpenAI API key가 [workspace secret](/langsmith/administration-overview#workspace-secrets)으로 설정되어 있는지 확인하세요.

1. <Icon icon="gear" /> **Settings**로 이동한 다음 **Secrets** 탭으로 이동합니다.
1. **Add secret**을 선택하고 `OPENAI_API_KEY`를 입력한 후 API key를 **Value**로 입력합니다.
1. **Save secret**을 선택합니다.

<Note> LangSmith UI에서 workspace secret을 추가할 때, secret key가 모델 제공자가 예상하는 environment variable 이름과 일치하는지 확인하세요.</Note>
